夜老挂断了电话就来到了夜无忧的房间，夜无忧并没有休息，他的桌子上摆着很多小瓶小罐，他正在那里做研究。
<br/>
<br/>        “无忧，你怎么不休息？”夜老上前，关心的看着夜无忧。
<br/>
<br/>        夜无忧抬头，他生的十分俊美，只是一张脸很白，是那种透着病态的苍白，他有一双混血儿的蓝瞳，瞳仁里冷到毫无温度，他看着夜老，“爷爷，我不困。”
<br/>
<br/>        夜老点头，“无忧，林水瑶的女儿出现了，她叫夏夕绾。”
<br/>
<br/>        夜无忧拿着小瓶的手突然一顿，那双蓝瞳里溢出了炽热的光芒，“爷爷，你是说有着不死之血林水瑶的女儿？”
<br/>
<br/>        “没错，夏夕绾是林水瑶的女儿，她一定也有着不死之血。”
<br/>
<br/>        夜无忧拿出了一个红色的小瓶，递给了夜老，“那真是太好了，我一直在研究对付不死之血的毒药，这瓶毒药刚练成，正好拿那个夏夕绾做个试验。”
<br/>
<br/>        夜老一生痴心医学，夜无忧比夜老更痴迷，不过他痴迷的是用毒。
<br/>
<br/>        夜老看着这个红色小瓶，“无忧，这里面是什么？”
<br/>
<br/>        夜无忧勾唇，缓缓吐出了五个字，“弹指红颜老。”
<br/>
<br/>        弹指红颜老！
<br/>
<br/>        听着这个名字，夜老露出了满意而欣慰的笑容，二十多年前林水瑶身上带来的不但有医书，也有一本毒书。
<br/>
<br/>        但是林水瑶认为医和毒并不能共存，所以想要毁了那本毒书。
<br/>
<br/>        夜老就趁机将那本毒书偷偷藏了起来，这件事也彻底激发了他和林水瑶之间的分歧，导致两个人师徒破裂。
<br/>
<br/>        后来夜老将这本毒书交给了自己的孙子夜无忧，哪怕林水瑶已经死了，他也要向林水瑶证明当年他是对的，毒可以超越医，让人得到梦想的一切！
<br/>
<br/>        ……
<br/>
<br/>        医院里。
<br/>
<br/>        夏夕绾颤了颤纤长的羽捷，然后睁开了眼，她现在在病房里，正挂着点滴，这时陆子羡走了过来，握住了她的小手，“绾绾，你醒了？”
<br/>
<br/>        夏夕绾还觉得全身无力，她差一点就溺死在了游泳池里，刚从鬼门关逃生回来还很虚弱。
<br/>
<br/>        陆子羡将她扶坐了起来，在她背后垫了一个软垫，“绾绾，你没有大碍，休息两天就会恢复了。”
<br/>
<br/>        夏夕绾感激的看着陆子羡，“子羡，谢谢你，如果不是你及时赶到救了我，我可能就真的死了。”
<br/>
<br/>        陆子羡干净的黑眸落在夏夕绾的小脸上，“绾绾，救你的人不是我，而是陆寒霆。”
<br/>
<br/>        什么？
<br/>
<br/>        夏夕绾一僵。
<br/>
<br/>        陆子羡并不想隐瞒，他本来就光明磊落，“绾绾，当时我去陪厉老夫人聊天了，你出事的时候我并不在你的身边，是陆寒霆去而复返，跳下游泳池救了你，听说你呼吸都没了，大家都认为你死了，但是陆寒霆没有放弃你，他一直在给你做心肺复苏和人工呼吸，强制性的将你从鬼门关拉了回来。”
