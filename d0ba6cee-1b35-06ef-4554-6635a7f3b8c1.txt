<br/>　　尤其是这剑阵的阵眼，正是一件天级灵器。

<br/>　　在楚风眠的手中，居然拥有这三件天级灵器。

<br/>　　无数灵剑，空中飞舞起来，组成了一道精密无比的剑阵。
<br/>　　十万灵剑，互相交织，将冷面魔君的十倍力量，在空中猛然相撞。

<br/>　　冷面魔君狰狞的怒吼道。
<br/>　　他现在是真的怒了。
<br/>　　本来为了对付楚风眠，他便是都已经动用了自己隐藏的全力。

<br/>　　却是没想到楚风眠居然是有着如此实力，在他展现出真正实力的时候，还是足以有着跟他抗衡的能力。
<br/>　　如今这恼怒之下，他便是将十倍力量，全部展现出来，想要轰杀楚风眠。

<br/>　　他冷面魔君杀一个人，还从来没有浪费过如此之多的手段。
<br/>　　“今日你就算是躲在这乌龟壳里，我也一样要将你斩杀！”
<br/>　　冷面魔君怒喝一声，在他的背后，无数魔意瞬间爆发出来。
<br/>　　这魔意，逐渐凝聚这，居然是化为了一尊巨大无比的魔神，这一尊魔神，看起来庞大到极点。
<br/>　　周围的空间，随着这魔神的出现，都开始颤抖起来，这魔神降临的气息，令周围的空间，都在颤抖。
<br/>　　“唯一真魔！这是魔道的老祖宗，唯一真魔！”
<br/>　　“这冷面魔君的魔道，居然已经是达到了这一步，将魔道的老祖宗唯一真魔，都给凝聚了出来！”
<br/>　　“这冷面魔君真的是不惜一切代价了，这种压箱底的手段都动用了，这小子是完了，不过这小子是北邙学院的人，他们地魔门的心腹大患，今日这冷面魔君不惜一切代价斩杀了楚风眠，也会得到许许多多的好处。”
<br/>　　无数武者在下方交流着。
<br/>　　看着空中的楚风眠，他们便是都如同看向一个死人。
<br/>　　所有人都觉得，楚风眠是死定了。
<br/>　　冷面魔君将唯一真魔虚影这种手段，都给展现了出来，显然是对楚风眠，起了必杀之心。
<br/>　　这唯一真魔出现的刹那，冷面魔君身上的气质，便是比起刚刚，截然不同。
<br/>　　好像是一尊真正的远古魔头，苏醒的样子，一举一动，都是蕴含魔道真谛。

<br/>　　冷面魔君心神一动，他背后的唯一真魔，便是猛然一跺脚，一股巨大无比的冲击，顿时向着剑阵轰击过去。
<br/>　　擂台都在摇晃，似乎是在这力量之下，摇摇欲坠。
<br/>　　这种力量之下，楚风眠还能活命？
<br/>　　就算是有着剑阵的保护，楚风眠也绝无生机。

<br/>　　就算一直冷静的寒月漓，如今看到这唯一真魔凝聚的一刻，脸色都是变了。
<br/>　　唯一真魔，便是魔道的先祖，传说中缔造魔道的恐怖存在。
<br/>　　凝聚出他的虚影，显然是冷面魔君的魔道，已经是领悟到了真正的魔道真谛。
<br/>　　这唯一真魔虚影出手，比起冷面魔君展现出的十倍力量，都要可怕的多。
<br/>　　冷面魔君面露狰狞的看着楚风眠，大声吼道。
<br/>　　“小子去死吧，死在唯一真魔的手中，你也死而无憾了！”
<br/>　　这唯一真魔的一脚落下，其中蕴含的力量，楚风眠正好面对着，自然是知道的一清二楚。
<br/>　　这力量已经是强到惊人的地步了，若非是生死台阶，碎命之境的突破，不单单是需要力量，更是需要领悟的话。
<br/>　　现在这冷面魔君的力量，只怕是都已经是足以达到了碎命之境的地步了。
<br/>　　现在的他，只是没有超越生死的领悟，但是在力量上，真正的达到碎命之境的范畴了。

<br/>　　楚风眠抬起头，面对这唯一真魔的力量，他的嘴角微微冷笑道。

<br/>　　楚风眠突然开口，他的身躯，猛然一动，直接消失在了原地。
