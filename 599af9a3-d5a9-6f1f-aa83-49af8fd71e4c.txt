夏夕绾整个人的状态十分的糟糕，一边守着林婶，林婶还昏迷不醒，另一边魂不舍设。
<br/>
<br/>        那一日陆寒霆摔门而去后就再也没有回来过，这都两天过去了，老夫人告诉她，说他又出差去了。
<br/>
<br/>        夏夕绾知道他没有出差，只不过找了一个借口不让老人家担心，另一方面也给了她体面。
<br/>
<br/>        这样的状态一直持续到下午，夏小蝶的电话打来了。
<br/>
<br/>        夏夕绾按键接通，那端夏小蝶春风得意的嗓音迅速传递了过来，“夏夕绾，晚上一起来1949酒吧玩啊，你敢不敢来？”
<br/>
<br/>        夏夕绾不想让自己再颓废下去了，她不会忘了这一次自己回来的初衷，她还有很多事情要做。
<br/>
<br/>        “好啊，晚上我会准时赴约的。”
<br/>
<br/>        ……
<br/>
<br/>        1949酒吧。
<br/>
<br/>        夏夕绾进入了包厢，夏小蝶已经来了，还有孔真儿。
<br/>
<br/>        夏小蝶迫不及待的竖起了自己的手，“夏夕绾，你看我的无名指上戴了什么？”
<br/>
<br/>        夏小蝶手上戴了一个大钻戒。
<br/>
<br/>        夏夕绾还没有说话呢，孔真儿已经“哇”一声，一唱一和了，“小蝶，你这个钻戒是苏少送的么，你生日苏少送了一个钻石项链，现在又送了你一个大钻戒，苏少真的好宠爱你啊。”
<br/>
<br/>        夏小蝶无比得意的看着夏夕绾，想从她的脸上看出一丝羡慕之色，“是啊，这是苏希哥哥送给我的钻戒，关键是，过两天我跟苏希哥哥就要举办一个盛大的订婚party了，到时苏希哥哥还会向我求婚。”
<br/>
<br/>        “小蝶，我真是太羡慕你了，嫁到苏家当少奶奶这可是海城名媛们梦寐以求的事情。”
<br/>
<br/>        夏小蝶看着夏夕绾，只可惜夏夕绾没有流露一点的羡慕，只是淡淡的听着，她感觉自己的拳头都砸在了棉花上。
<br/>
<br/>        “夏夕绾，你表面装得这么淡定，其实心里早羡慕我了吧，苏希哥哥是我的，他爱的也是我，你抢不走的！”
<br/>
<br/>        夏夕绾看向夏小蝶，淡淡的莞尔，“放心吧，我不会抢你的苏希哥哥的，而且在你们的订婚party上，我还会送你们一份神秘大礼！”
<br/>
<br/>        什么神秘大礼？
<br/>
<br/>        夏小蝶觉得夏夕绾这个土包子根本就拿不出钱来送什么神秘大礼。
<br/>
<br/>        这时“叮”一声，夏夕绾的手机响了，来短信了。
<br/>
<br/>        苏希发来的。
<br/>
<br/>        夏夕绾点开一看过来1949酒吧，给你看一样好东西。
<br/>
<br/>        真巧啊，她现在就在1949酒吧。
<br/>
<br/>        夏夕绾看了一眼对面的夏小蝶和孔真儿，然后回了一条我现在就在1949酒吧，你到包厢里找我。
<br/>
<br/>        将手机收在了包里，夏夕绾道，“夏小蝶，你出来一下，我有话跟你说。”
<br/>
<br/>        “有什么话不能在这里说？”虽然夏小蝶嘴里嘀咕着，但是她好奇心太强，还是跟着夏夕绾出去了。
<br/>
<br/>        包厢里就剩下了孔真儿。
<br/>
<br/>        ……
<br/>
<br/>        孔真儿一个人觉得十分的无聊，夏小蝶点了不少好酒，她偷偷的喝了两杯。
<br/>
<br/>        这酒是好喝，但是酒精度数太高了，容易醉。
<br/>
<br/>        孔真儿的小脸被醉意烧红的时候包厢门开了，颀长俊逸的苏希走了进来。
