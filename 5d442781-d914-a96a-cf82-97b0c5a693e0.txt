<br/>        欧美风的别墅，里面的装潢低调而奢华，是他一贯的喜好和品味。

<br/>        夏夕绾蹬蹬的跑了过去，先拆开了一个盒子，盒子里是一个盆景。
<br/>
<br/>        夏夕绾瞳仁一缩，这就是她前段时间跟双双说过的…传说里的曼陀沙罗。
<br/>
<br/>        这个传闻生长在生死两界的曼陀罗有着黑色的根茎，它的花朵是红色的，红到近乎妖冶。
<br/>
<br/>        夏夕绾在医书里见过曼陀罗花，但远远没有现在亲眼看到来的震惊，她更震惊的是，双双真的弄到了曼陀罗。
<br/>
<br/>        这个曼陀罗全世界估计都弄不到两株，上一次双双拍着胸膛打着包票说给她弄到手，当时她还狐疑了，没想到双双不是开玩笑的。
<br/>
<br/>        盒子里有一张字条，是双双写的绾绾，生日快乐呀，ps：终于在你生日这天将曼陀罗给培育出来了，小心，曼陀罗真的剧毒无比，我不小心中了一分毒，但勿念，我已种了一棵天山雪莲吃下解毒了，慎用之。
<br/>
<br/>        夏夕绾看着“培育”这两个字，又看了看种了天山雪莲来解毒，她的小脑袋简直就炸开了，天哪，双双究竟是什么人，曼陀罗是她培育出来的，她可以随便种出天山雪莲。
<br/>
<br/>        夏夕绾突然想到了双双最喜欢种花种草了，她有很多盆盆罐罐，难道那些土里都是些稀世药材么？
<br/>
<br/>        夏夕绾现在好想找到双双问一问，她将曼陀罗小心翼翼的收了起来，陆先生的病情刻不容缓，现在有了曼陀罗，她明天就要试毒炼药。
<br/>
<br/>        夏夕绾拆开了另一个盒子，这是叶翎送的。
<br/>
<br/>        翎翎会送什么生日礼物呢？
<br/>
<br/>        盒子打开，里面是一箱的…避孕-套。
<br/>
<br/>        叶翎也留了字条绾绾，生日快乐呀，ps：我也不知道陆先生用多大的号，但目测陆先生是财大器粗，最大号总没错了。

<br/>        “…”
