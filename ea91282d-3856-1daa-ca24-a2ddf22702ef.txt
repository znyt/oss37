　　不得不说，皇权对于古代人的影响是十分巨大的，这种影响几乎已经深入到了百姓的骨头里面，以至于李承乾只是简简单单的说了一句把称心当成朋友，就已经可以让称心直接跪伏于地，俯首叩拜。
<br/>　　不过好在李承乾这家伙到底也是受过现代教育的影响，对称心的这种行为并没有觉得理所当然，见他拜服于地，立刻上前将他拉了起来：“起来，你我之间不必如此，而且说实话，你能下决心脱离这个圈子，我其实还是很替你高兴的。”
<br/>　　“陛下，臣有负圣恩，罪在不赦。”挣扎了一下，但最终还是顺势站了起来，看着李承乾痛陈道：“只是陛下，臣是在情报科里出来的，知道那里是个什么样子，臣真的不想让小钰再走上臣的老路，只想让她平安喜乐的度过一生。”
<br/>　　“这一点朕可以答应你，你的那个小女徒弟将来的路由你来安排，不会有人横加干涉。而且就像我刚刚说的，如果你真的想离开这个圈子，我也可以放你离开！不过，机会只有一次，你要慎重选择！”李承乾先是以皇帝的身份向称心做了保证，接着又以私人的身份对他提出建议。
<br/>　　称心当然明白李承乾是什么意思，他也很清楚，皇帝对他的照顾可一不可再，如果接二连三给他选择的机会，那么便会影响他的权威，对于其他人也会显的不公平。
<br/>　　但是现在他已经改变了自己的初衷，考虑到自己一个人带着一个小女娃浪迹江湖着实有些困难，再加上现在有了李承乾的保证，他也不再担心那个小丫头。
<br/>　　当然，还有更重要的原因他一直没有说，那是他心里一个永远的秘密，他决定把这个秘密带进棺材里面，永远也不会告诉任何人。
<br/>　　……
<br/>　　西市，西域香料店迎来了一位年轻的客人，此人衣着考究，手里拿着一把折扇，身后带着两个护卫，进了店里之后先是四下打量了一眼，然后才对店里的伙计说道：“伙计，你这店里可有陈年的八角？”
<br/>　　伙计明显愣了一下，放下手里的活计抬头看了一眼那年轻人，继而笑着说道：“这位公子，小店有三年前的存货，只是不知公子需要多少？”
<br/>　　“三百二十五斤六两七钱半，不知道你这里可有？”那年轻人把手里的扇子摇了摇，又随手合上，盯着那伙计问道。
<br/>　　伙计并没有任何犹豫，迎应答道：“公子来的不巧，小店里只剩下两百二十三斤三两五钱，公子可要全部买下？”言罢，看着那年轻人比了个请入后宅的手势，只不过那只伸出来的右手中指与拇指却是扣在一起的。
<br/>　　暗号已经完全对上了，年轻人回头看了一看身后的两个护卫，不着痕迹的打了一眼色，然后跟着那个伙计走起了后宅。
<br/>　　……
<br/>　　店铺的后宅是一个很大的院子，院子的四周搭着无数的棚子，在那些棚子的下面，则是一袋袋装的满满的香料。
<br/>　　而就在院子的当中，有一颗很粗的老树，树的下面正坐着一个大食人，长着一脸的胡子，看上去就像年轻版的圣诞老人。
<br/>　　此时他正在摆弄着一壶茶，一壶功夫茶，看他熟练的动作显然不是第一次。
<br/>　　“东家，您要等的人来了！”伙计带着年轻人一路进了后宅，来到了那个大食人的身边，俯下身子低声说道。
<br/>　　那大食人闻言，右手微微顿了顿，随后便将手里的小茶壶放到了一边，抬起头看向那年轻人：“孙公子果然守时，没有让在下一番白等。”
<br/>　　不过那年轻人脸上却露出一抹古怪的笑意：“托尼克先生似乎认错人了，在下不姓孙，而是姓王。”
<br/>　　大食人显然是故意的，被年轻人指出错误之后，只是轻轻一笑：“哎呀，你看看，这人老了精力就是不行，竟然一时记差了公子的名讳，失礼，失礼了！”
<br/>　　“托尼克先生未免太过小心了吧。”年轻人自动自觉的坐到了那大食人托尼克的面前，主动拿起桌上的茶壶，翻起一只杯子，给自己倒了一杯茶，一边喝一边说道：“在下既然能来到这里，见到先生，难道先生还不相信我？”
<br/>　　“小心无大错，这里毕竟是大唐皇帝的眼皮子底下，若是万一出了什么事情，那是要出人命的。”托尼克耸了耸肩膀说道。
<br/>　　他是大食这次来大唐的副使，身上肩负着一些不为人知的使命。
<br/>　　要知道，现在大唐与大食之前可不是一片祥和，海上的战争那可是打的如火如荼，他这个副使怎么也不可能像其他人那样来大唐享福。
<br/>　　而且托尼克的身份是副使，对于这次出使来说显的可有可无，这正好方便了他在其中搞一些事情，从来不会担心被某些人注意到。
<br/>　　当然，就算是这样他还是要十分小心，如果能不被发现还是不要被发现的好，虽然眼下的大食并不一定真的惧怕大唐，但多一事总是不如少一事。
<br/>　　王姓年轻人对托尼克的解释显然不是那么满意，如果不是因为这次的事情十分重要，他真的很想调头就走。
<br/>　　所以在略一沉默过后，王姓青年一口抽干了杯中的茶水，面色变的阴郁了许多：“托尼克先生既然如果不相信在下，那在下无话可说，就此告辞好了。”
<br/>　　“王公子且慢。”托尼克见年轻人似乎真的有要走的意思，连忙摆了摆手，然后又再次给他倒了杯茶：“王公子，如果你知道大唐情报科无孔不入的能力，估计就会相信我这么做并不是没有原因的。”
<br/>　　“情报科又如何，难道你以为他们真的能把手伸进我们几大世家中来？”青年撇撇嘴，有些不屑的说道：“托尼克先生未免太过高看李承乾和他的情报科了。”
<br/>　　“在下还是那句话，小心无大错。”托尼克叹了口气，然后缓缓开口，岔开话题说道：“怎么样，王公子你是否有了决定，这次的合作在下可是期待已久了，如果你能答应在下的条件，日后我们一定会成为最好的合作伙伴。”
