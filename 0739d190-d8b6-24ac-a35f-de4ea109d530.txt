　　十招高级武技。
<br/>　　十个超级天才全力打出。
<br/>　　十道威势无比的攻击同时朝萧逸打来。
<br/>　　萧逸手中无双剑一震，头顶上空猛地出现了一个五百丈的真气漩涡。
<br/>　　“覆海斩。”萧逸的剑，瞬间变了。
<br/>　　剑身，仿佛成了一个无边无际的大海之源。
<br/>　　剑身震荡，一道道剑音扩散。
<br/>　　如同澎湃潮涌，又如海啸席卷。
<br/>　　那一道道无形的剑音波纹，仿佛是一个个滔天巨浪。
<br/>　　所过之处，寸草不生，一切吞没。
<br/>　　叶铭的木龙降世，瞬间化为一堆木屑。
<br/>　　玉如龙的万兽吼，瞬间消散。
<br/>　　苗千火的千道火焰，瞬间熄灭。
<br/>　　…
<br/>　　一切一切，在这些剑音波纹前，是那么的无力，连一秒都抵挡不住，支撑不住。
<br/>　　原本威势无比，肆虐全场的十道攻击。
<br/>　　竟在一瞬间消散，好像硬生生被吞没了一般。
<br/>　　这是一种怎样的场景，无法言喻。
<br/>　　只知，数千人都在瞪大着眼睛，脸色久久无法平复。
<br/>　　十堂首席，叶铭等九人，尽皆吐血击飞。
<br/>　　他们原本自信的脸色，早已变成了惊骇。
<br/>　　他们的眼珠中，仿佛映衬着一个滔天巨浪，在朝他们涌来。
<br/>　　“噗…噗…噗…”
<br/>　　九人直接被震下比武台，大口大口地吞着鲜血。
<br/>　　他们的惊骇脸色，始终挥之不去。
<br/>　　那是一种发自内的害怕和恐慌。
<br/>　　那是仿佛一个人在无边大海中，随波漂浮，永远无法到岸，却又担心下一刻会被巨浪吞噬，彻底葬身大海的无尽恐慌。
<br/>　　这种感觉，能让人绝望。
<br/>　　覆海斩，是范围性的音波攻击。
<br/>　　给予的，绝不是单纯的肉体伤害这么简单。
<br/>　　地阶巅峰武技，仅次于天阶武技的存在，岂是轻与的。
<br/>　　当然，萧逸留手了，并没有用尽全力。
<br/>　　这九人，要不了多久就会恢复。
<br/>　　而这时，萧逸并没有停下攻击。
<br/>　　因为，还有一人，没有离开比武台，正是顾长空。
<br/>　　叶铭等九人，刚才都是率先攻击，距离萧逸最近。
<br/>　　覆海斩，最先将这九人震出比武台。
<br/>　　只有顾长空，本来就是后面才攻击，也靠得后。
<br/>　　加之使用了冥空剑，七煞斩的威力早已达到了破玄境层次。
<br/>　　覆海斩轻松破掉七煞斩后，他得以有反应的时间，立刻御空飞行至高空。
<br/>　　他倒不蠢，反应过来覆海斩的音波攻击是范围性，且只能扩散到一定的距离。
<br/>　　他立刻飞到高空，可以立刻脱离这个范围。
<br/>　　同时，他这样做，并不算落下比武台，并不算输。
<br/>　　萧逸御空而上，直追而去。
<br/>　　顾长空这种小人，心思歹毒，屡次想置自己于死地，今日不除，他日必定后患无穷。
<br/>　　虽然10分钟已过，但只要双方不认输，比武还是可以继续下去。
<br/>　　另一边，长老席上，一众长老看着高空中那五百丈的巨大漩涡，吓了一跳。
<br/>　　“十一，怎么回事，萧逸怎么可能一次性调动这么多真气？”大长老连忙问道。
<br/>　　易老似是早就知道，笑道，“他的气泉，足有五百丈。”
<br/>　　“以他现在洞玄七重的修为，估计是使用了什么禁招，强行将修为提升到洞玄九重，气泉堪堪填满的地步。”
<br/>　　“气泉五百丈？”一众长老，倒吸一口凉气。
<br/>　　“那岂不是说，萧逸比那些传说中的绝世天才还要厉害？”
<br/>　　“不对。”大长老忽然问道，“就是他是五百丈气泉，但也不可能直接将所有真气释放出来…”
<br/>　　“笨蛋。”易老笑骂道，“萧逸连五百丈气泉都拥有，难道就没有道体？”
<br/>　　“我猜测，他的道体，必然是比先天道体更强大的存在。”
<br/>　　“足可让他将体内真气全部释放出来。”
<br/>　　“原来如此。”大长老点点头，“原本，我还在遗憾萧逸就算是未来的剑主，也会因控火兽武魂而显得名不符其实。”
<br/>　　“现在看来。”大长老笑了，“五百丈气泉、强大的道体，啧啧，我有感觉，我们裂天剑派有史以来最强的一位剑主，就要诞生了。”
<br/>　　“最强的剑主。”其余长老，也眼睛一亮，看向高空中萧逸的目光，完全变了。
<br/>　　再不是看一个普通弟子，而是真正的值得他们无比重视的存在。
<br/>　　……
<br/>　　另一边，萧逸追上高空后，本想直接击毙顾长空。
<br/>　　没想到，顾长空硬是借着冥空剑内的庞大力量，将自己的飞行速度提升到极致。
<br/>　　导致萧逸没能一下子追上。
<br/>　　要是萧逸现在能使用紫炎火翼，顾长空早已殒命。
<br/>　　“哈哈哈哈。”顾长空肆意大笑起来，“萧逸，没有上品灵器，我看你拿什么与我斗。”
<br/>　　“当然，你也可以现在立刻认输。”
<br/>　　“反正，10分钟已过。”
<br/>　　“你认输也没什么影响，就是自认不如我罢了。”
<br/>　　他在用激将法，他其实并不想这么快结束战斗。
<br/>　　他还是想杀掉萧逸。
<br/>　　只不过，这正合了萧逸的意。
<br/>　　“哼。”萧逸冷哼一声，他知道顾长空心中所想。
<br/>　　但他暂时没有拆穿。
<br/>　　同时，下方的长老们也仍旧在观战，而并非是出手停下战斗。
<br/>　　他们自信自己的实力，能应对一切意外的发生，起码不会出现人命。
<br/>　　萧逸也知道这一点，所以，他还在等。
<br/>　　萧逸，永远不会做没有把握的事。
<br/>　　杀人，特别是必杀之人，不能急。
<br/>　　要么不杀，一出手便必须是一击必杀。
<br/>　　高空中，萧逸仍旧追着顾长空。
<br/>　　顾长空，仍旧得意洋洋地上升，拉开和萧逸的距离。
<br/>　　他相信，萧逸追不上他，却不知道，萧逸一直在把控着距离。
<br/>　　而且，这个距离越来越短。
<br/>　　十数秒后，当顾长空已经上升到一定高度。
<br/>　　萧逸才冷冷一笑，体内三千三百三十三丈的气泉，已填满七成，达到二千多丈的恐怖真气，一瞬间在小世界调动起来。
<br/>　　他的速度，也陡然间激增，快得所有人都反应不过来。
<br/>　　“覆海斩。”萧逸大喝一声。
<br/>　　“什么？好快的速度。”顾长空瞳孔一缩，瞬间大惊。
<br/>　　剑音波纹，以一种恐怖的速度向他涌去。
<br/>　　他的眼珠中，已经映衬出巨大波浪的画面。
<br/>　　下方，长老席上，一众长老顿时色变。
<br/>　　他们明显感觉到萧逸的杀意，也知道萧逸想做什么。
<br/>　　“萧逸，不要冲动。”大长老大喝一声。
<br/>　　“小子，你敢？”剑堂长老怒发冲冠。
<br/>　　易老，则皱起了眉头。
<br/>　　但，他们都没有动作。
<br/>　　因为，顾长空和萧逸上升的高度，已经很高。
<br/>　　他们就算现在出手，也来不及了。
<br/>　　高空中，萧逸冷漠一笑，“顾长空，这次没人能救你。”
<br/>　　“不…不要…”
<br/>　　顾长空无比恐慌着，他甚至已经忘了要举剑抵挡。
<br/>　　他的求饶声，仿佛不是对萧逸说，而是在对他眼中的‘滔天巨浪’说。
<br/>　　一种绝望的感觉，涌上心头，让他无力。
<br/>　　“死吧。”萧逸的声音，无比冰冷。
<br/>　　但…
<br/>　　下一秒，剑音波纹刚要打到顾长空身上时。
<br/>　　不远处，一道身影，疾速飞来。
<br/>　　飞行的速度，极快，绝非洞玄境武者能达到。
<br/>　　来人是破玄境武者。
<br/>　　‘轰’的一声，来人仿佛一头不长眼睛的疯马。
<br/>　　直直地撞上了剑音波纹，而后将覆海斩的剑音完全撞碎。
<br/>　　“好强的实力，起码是破玄五重武者。”萧逸大吃一惊。
<br/>　　当然，剑音波纹破碎。
<br/>　　来人似乎也不好受。
<br/>　　硬生生地停下了疾速飞行的身影，而后口中溢出一丝血丝，惊骇地看着面前的萧逸。
<br/>　　“我擦，裂天剑派什么时候出了这么个妖孽，年纪轻轻，竟有如此实力。”来人惊骇说道。
<br/>　　这时，萧逸也看清了来人的样子，顿时一惊。
<br/>　　来人，竟是数月前有过一面之缘的血刀卫统领，聂如山，破玄五重武者。
<br/>　　“聂如山？”下方的长老们，也认出了聂如山。
<br/>　　嗖嗖嗖，十一道身影，瞬间来到高空中。
<br/>　　剑堂长老来到顾长空面前，查看他有没有事。
<br/>　　易老，则将萧逸挡在了身后。
<br/>　　“聂如山，你来裂天剑派做什么？”大长老问道。
<br/>　　“哦，差点忘了正事了。”聂如山拍了拍脑袋。
<br/>　　“参见众长老。”聂如山行了一礼，而后脸色大变，道，“诸位长老，东荒十八城，兽潮袭来。”
<br/>　　大长老摆摆手，道，“东荒那边，每几年都有兽潮，你来我裂天剑派做什么。”
<br/>　　“不是，这次的情况非常紧急。”聂如山着急道，“这次的兽潮，不知为何，出现了许多破玄境妖兽。”
<br/>　　“哦？有多少？”大长老问道。
<br/>　　“一堆。”聂如山脸色骇然地吐出了两个字。
