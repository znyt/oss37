<br/>众人疑惑的互相对视了一眼，这名貌似曾经强大的老者竟然失忆了，难道是走火入魔的后遗症不成。<br/>　　<br/>　　“老人家，您不要着急，慢慢想，也许你这是走火入魔导致的，如果真的想不起来就不要想了。”李振邦轻声安慰着。<br/>　　<br/>　　“我是谁？我为什么来这里？我到底怎么了？”老者盘膝坐在那里，眼神呆滞，喃喃自语道。<br/>　　<br/>　　“振邦，他不会成傻子吧？刚才还好好的，怎么一问名字就变成这个样子了？我看咱们还是走吧！”张大友看了看已经沉醉在自己世界中的老者，拽了拽李振邦的袖子，轻声说道。<br/>　　<br/>　　李振邦看了一眼平台上的老者，皱了皱眉头。一名强大的光系强者，为什么会出现在这里？为什么会变成现在这个样子？为什么精神力感知到他并不是人只是一团能量？可是刚才自己和他接触的时候，明明感觉到他就是一个有血有肉的人啊！<br/>　　<br/>　　正在这时，李振邦的脑海中响起一个声音，成功拯救光系强者，开启特殊任务——找回强者的记忆！<br/>　　<br/>　　李振邦有些愕然的看向同伴们，他们也一脸难以置信的看向李振邦，显然他们也接收到了这么一条没头没脑的信息。<br/>　　<br/>　　找回记忆？怎么找回？是要击杀什么魔兽还是要找到什么承载着老者记忆的东西？什么提示都没有，这怎么完成？还有是谁提示他们的？难道这就是中年老师说进来以后就知道的意思吗？<br/>　　<br/>　　到现在为止这算是第一个接到的任务，而且按照提示是一个特殊任务，什么是特殊任务？李振邦脑海中突然闪现出前世玩大型网络游戏时的一个词语——支线任务。<br/>　　<br/>　　难道这是自己触发的一个支线任务？那主线任务是什么？完成主线任务应该才算是完成考核，可是到现在为止根本就不知道主线任务是啥！<br/>　　<br/>　　想明白了这一点，李振邦豁然开朗，难怪自己精神力感觉到老者是一团能量体，接触到却感觉到他是有血有肉的真人。<br/>　　<br/>　　老者根本就是这个世界的一部分，或者说老者根本就不是人，只是他们任务的一部分，是一个这个世界的NPC罢了。这个世界的魔兽既然都是能量构成的实体，那这个老者自然也不会例外了。<br/>　　<br/>　　李振邦将自己想到的情况尽量用众人能听懂的话解释过之后，张大友点了点头问道：“振邦，那我们下一步该怎么办？”<br/>　　<br/>　　“我也没办法了，只能走一步算一步了，估计老白是指望不上了。”<br/>　　<br/>　　“老白？”众人疑惑的看向李振邦，李振邦耸了耸肩膀，用下巴指了指依然在盘膝坐在平台上自言自语的白胡子老头儿。<br/>　　<br/>　　众人不禁莞尔一笑，老白似乎还是挺适合这老头儿的。<br/>　　<br/>　　“我们是继续往下走还是退回去？”李振邦将众人集中在一起，开始商讨下一步计划了。<br/>　　<br/>　　“还是继续走下去吧！如果退回去，鬼知道还会遇到什么任务，而且那头顶上的那群蝙蝠该怎么解决？我可不相信我们的运气永远那么好。”张大友撇了撇嘴，第一个出声发言。<br/>　　<br/>　　想到头顶上倒垂的那些蝙蝠，众人不禁打了个冷颤，众人暂时还没想到解决他们的好办法，看样子只有继续走下去这一条路了。<br/>　　<br/>　　李振邦从空间手镯中拿出食物、肉干和水袋分给大家，大家吃完饭简单的休整了一下，继续开始征程了。<br/>　　<br/>　　期间李振邦有些不死心的又跑到老白的身边，希望能从他嘴中得到一些有用的消息，可是老白依然自言自语，根本不理会李振邦。<br/>　　<br/>　　李振邦问他记忆怎么找回来的时候，他的眼神会变得迷茫，然后捂着头，痛苦的“嗷嗷”的叫着，李振邦只好无奈的放弃了。<br/>　　<br/>　　在休整的时候，李振邦尝试着使用了两次召唤术，希望能召唤出这个位面的魔兽，来给自己做一些参考。<br/>　　<br/>　　不过很可惜，两次召唤出来的都是野兽，一头野猪，一头野牛。虽然不是魔兽，但是还勉强算是有点用处。野猪和野牛多少也算是有一些战斗力，用来趟路还是不错的，而且野牛脊背宽阔，力气和耐力都很好，成为了林婉儿的代步工具。<br/>　　<br/>　　前进的路上战斗虽然不断，但是并没有遇到什么太大的危险，偶尔遇到两个厉害点的魔兽，也在李振邦的指挥下有惊无险的解决了。<br/>　　<br/>　　“振邦哥，你有没有觉得有些不对劲儿？”沙含山突然开口问道。<br/>　　<br/>　　“嗯，是有些不对劲儿的感觉，但是说不好为什么。等一下，我查看一下。”李振邦说完，将精神力扩展到极致，周围的一草一木连内部结构都清晰的呈现在他的脑海中。<br/>　　<br/>　　“保持警惕，我们似乎遇到麻烦了。”李振邦脸色大变，直接大叫了出来。<br/>　　<br/>　　众人虽然不知道李振邦说的麻烦是什么，但是出于对李振邦的信任还是第一时间做好战斗准备。<br/>　　<br/>　　“怎么了，振邦？”张大友疑惑的问道。<br/>　　<br/>　　“我们似乎进入幻阵之中了！”李振邦有些不确定的说道。<br/>　　<br/>　　他精神力探查到周围环境并没有什么异常，如果不是探查到小草内部环境是静止的，不符合常识，他也不会发现异常。<br/>　　<br/>　　众人听到李振邦说幻阵，脸色都是一变。<br/>　　<br/>　　李振邦第一时间联系上了多多，不过多多的回复是他也没有办法，因为这个不是幻阵，而是幻境。幻境对它来说没有什么影响，但是它也没有能力破除。<br/>　　<br/>　　幻阵是用各种稀有材料构成的法阵，只要破除了阵眼，幻阵自然也就破了，而且幻阵主要的作用是迷惑和消耗。<br/>　　<br/>　　可是幻境是直接作用于精神上的，没有任何破除的办法，除非将释放幻境的人杀死或者精神力远远高于释放幻境的人，否则没有任何办法。<br/>　　<br/>　　幻境因为是直接作用于精神力上的，所以在幻境中受到伤害，大脑就会认为是真实的。假如在幻境中被人砍掉一条胳膊，那大脑就会认为这条胳膊没有了，就算最后幻境被破除了，但是想要恢复这条胳膊的行动能力也是不容易的。<br/>　　<br/>　　李振邦并没有告诉大家这是幻境，否则众人恐怕连挣扎的勇气都没有了。<br/><br/><br/>
