　　第277章
<br/>　　云中龙的根据地在灰羊原，山寨外围以木栅栏围着，就这样赫赫立在一片平原上。
<br/>　　这几日，程大雷想方设法打探云龙中的信息，而云中龙何尝不在打听关于程大雷的情报。
<br/>　　云龙中手下有四员大将，周差海，方闻雷，乔梅浪，田倜，上次抢劫高飞豹的就是老三乔梅浪。
<br/>　　那是一次失败的行动，不仅折了弟兄，还被人将马抢走，所得到的不过十车盐而已。而云中龙就是本地的私盐贩子，他不缺这些东西。
<br/>　　所以云中龙最近一直在打听这伙人的来历，能够在五十人的包围中脱身，杀人夺马干净利落，丝毫不拖泥带水。如果只是一个人有这样的身手，并不令人意外，可十几个人都有这样的身手，就有些令人不寒而栗。
<br/>　　关键是拥有这样好的身手，他们为何要贩私盐，直接抢劫不是来钱更快么。
<br/>　　查来查去，云中龙没有查到任何消息。毕竟，程大雷还没有在这一带露面，只绑了西门春长一次，也没有闹出多大动静。
<br/>　　这伙人就像鬼魅一般，突然冒出来，又突然消失。
<br/>　　“龙爷，您别上火，估计是一伙过路的，知道咱们这儿不好惹，就去别的地方了。“夜来，云中龙一伙人聚在一起喝酒。
<br/>　　“哼，别让我再遇到他，不然非让他知道我的厉害。”乔梅浪冷哼一声。
<br/>　　乔梅浪最近心情很憋屈，在山寨有些抬不起头来，所以这几天他也是磨刀霍霍，就想着报仇雪恨，扬眉吐气。
<br/>　　“三哥，他们肯定已经走了。“
<br/>　　“嘿，让他们走了狗屎运，落在我手中，我非得把他们千刀万剐。”
<br/>　　一众人心中都有些不屑，上次败得那么惨，往后如果遇不到，不知是谁走了狗屎运。当然，大家表面上还得捧着聊。
<br/>　　“一定的，一定的。人有失手，马有失蹄，三哥你也别太放在心上。如果下次还能遇上，三哥肯定让他们知道知道厉害。”
<br/>　　“嘿，怕他们已经不敢在这一带露面了。”
<br/>　　这个时候，张肥和高飞豹已经来到灰羊原寨门前。此时夜风一吹，张肥的酒劲已经醒了大半。
<br/>　　我是谁，我在哪儿，我在这儿干嘛？
<br/>　　“三爷，你是不是害怕了？”高飞豹。
<br/>　　“怕！嘿，怕这个字我都不会写。”张肥将蛇矛扛在肩头：“里面什么情况？”
<br/>　　三军未动，情报先行，耳濡目染，大家也从程大雷身上学到对情报的重视。这几天，高飞豹也摸了摸云中龙的底儿，不然不会找到灰羊原。
<br/>　　“里面就是几百喽啰，能打的没几个，咱们兄弟杀进去，给他来个一网打尽。”
<br/>　　“几百人……还一网打尽，这个靠谱么？”
<br/>　　此时节，已经有门口的小喽啰发现了二人，站在寨门前喝道：“你们是什么人？不想活了么？”
<br/>　　高飞豹眼底一道寒芒闪过，抡着大刀冲了过去，口中还高喝着。
<br/>　　“三爷，事到临头，已无退路，冲杀进去，今天就是咱们兄弟扬名露脸的时候！”
<br/>　　咔嚓！
<br/>　　大刀砍下，那小喽啰已经身首异处，一颗好大头颅咕噜噜翻滚。
<br/>　　张肥一咬牙，口中哇呀呀暴叫。
<br/>　　“你三爷爷来了！”
<br/>　　二人骑马闯破寨门，云中龙平日作威作福，也从未想过某天会有人来攻打他们，更没有想到两个人就敢来打他们。所以，此刻山寨的大门洞开着，二莽汉冲进去后，见人就杀，是人就砍。
<br/>　　“大胡子在哪里，大胡子在哪里！”
<br/>　　高飞豹哇呀呀暴叫。
<br/>　　山寨中的人，短时间内竟没有反应过来。
<br/>　　“龙爷，龙爷……”
<br/>　　有喽啰闯进大厅，云中龙板起脸：“慌慌张张做什么？”
<br/>　　“有人杀进来了……要找大胡子。”
<br/>　　“对方有多少人呐？”
<br/>　　“好像只有两个。”
<br/>　　“胡说八道，只有两个人也敢来打我们。”
<br/>　　一行五人抓起兵器冲出屋子。只见院子内，两个大汉一边杀人一边放火，其中有个大汉口中还不停喊着：“大胡子在哪里，大胡子在哪里？”
<br/>　　乔梅浪眼睛一亮，抓起铁棒冲了过去。
<br/>　　“你家乔爷爷在此！”
<br/>　　二人一相见，顿时针尖对麦芒，胸中怒火翻腾，相互冲杀过来。
<br/>　　高飞豹借着马力，大刀挥砍过去，来势汹汹，乔梅浪用铁棍相格。
<br/>　　高飞豹早已不是一年前的高飞豹，他现在也是顶级山贼，花刀架门甩开，一柄刀玩得又快又稳。
<br/>　　只是三个回合，便将乔梅浪手腕斩断，乔梅浪发出一声撕心裂肺的尖叫。
<br/>　　高飞豹用刀背砸对方，一边砸一边大叫。
<br/>　　“你是谁老子，你是谁老子！”
<br/>　　咔！
<br/>　　一刀将乔梅浪劈为两截，高飞豹大叫：“我才是你老子！”
<br/>　　高飞豹轻取乔梅浪，而此时，张肥独斗四人，还显得略有余力。
<br/>　　“谁是云中龙？”张肥边打边叫。
<br/>　　云中龙没料到这二莽汉如此了得，听到张肥的话，下意识答音。紧跟着，他就觉得天旋地转，被张肥从马上拽下来，提在手中。
<br/>　　张肥将云中龙搁在马鞍桥上，口中大喝：“二爷，走啦。”
<br/>　　张肥是猛，但不像高飞豹这样愣，所谓擒贼先擒王，今天能将云中龙绑了行动已经算成功。真靠两人就把人家杀光，委实有些扯淡。
<br/>　　二人拨马往外闯，山寨门口密密麻麻都是人。周差海大喝：“拦住他们，拦住他们！”
<br/>　　这些小喽啰，手中拿着兵器，却也是面露惧色，膝盖不停大哆嗦。
<br/>　　“给俺……”张肥轮开丈八蛇矛：“散开！”
<br/>　　众人恍惚看到一头猛虎下山，面对周围的羊群发出一声咆哮。
<br/>　　用手中长矛开路，硬生生闯开一条血路，二人踏着尸体前进。
<br/>　　正此时节，山寨大门外呼律律一阵长嘶，程大雷带着几十人冲杀进来。
<br/>　　他手抡着斧头一马当先，将喽啰大队冲散，高飞豹见到这一幕，心中一喜。
<br/>　　“大当家，你怎么来了？”
<br/>　　程大雷脸黑得像锅底一般，一斧劈开周围，口中喝道：“别废话，先撤。”
<br/>　　张肥和高飞豹融入大队，几十人骑着马扬长而去。
<br/>　　
<br/>(本章完)
