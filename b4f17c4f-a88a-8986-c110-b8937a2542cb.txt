　　橙子见叶少阳犹豫，上前抓着他胳膊摇晃起来，“你都好久没带我一起啦，以前就不说了，这次是阴司的行动啊，我们一定要参加，小狼是不是。”
<br/>　　萧逸云只好跟着点头。
<br/>　　“行，那你们先回去，行动之前我叫你们，我先回去商量下，徐公，你跟我们一起吗？”
<br/>　　“我跟你们走。”
<br/>　　一行人走出宅子，互相道别，叶少阳试着破开虚空，结果作法失败，徐文长告诉他，阴司如今封锁了空间，想到鬼域来，只能通过涅槃法杖，于是大伙又回去。
<br/>　　烽火台附近，聚集了很多从人间来的法师，不过慕寒等人已经不见踪影了，叶少阳一行人回到人间，简单商量了一下，叶少阳便让瓜瓜去空界请林三生，任何行动都少不了他来出谋划策。
<br/>　　“那我也先离开少许，我去见见我那个徒弟。”徐文长有点不好意思地说道，从窗外飞出去。
<br/>　　叶少阳感觉有点饿，让瓜瓜在酒店等着，自己跟四宝一起出去吃东西，找到一家肯德基，四宝点餐回来，发现叶少阳坐在椅子上发呆，半天不出声，笑道：“我知道你在想什么，鬼兵围城，是吧。”
<br/>　　叶少阳不置可否，在他脑海中，总是挥之不去地浮现着酆都城外那些黑压压的鬼兵，简直触目惊心。
<br/>　　“虽然一直都知道太阴山实力很强，能跟阴司叫板，但真正看到这一幕，还是有点不敢相信……”叶少阳叹了口气，“原来阴司也不是无敌的。”
<br/>　　“这世上就没是无敌，我师祖说，就算是轩辕上帝、酆都大帝、无极鬼王，本质上也是人，知道为什么吗，跟实力无关，他们就算真能洞察三界，什么都知道，但他们的头脑，并不比人类聪明，他们也会犯错，而且在不断犯错。”
<br/>　　“你师祖是谁？”
<br/>　　“鸠摩罗什，他说我是他隔代弟子，在轩辕山上，他跟我说了挺多，也指点了我一些佛法，都是人间学不到的，最纯正的佛法，比藏传还纯正。”四宝得意地眨了眨眼睛。
<br/>　　叶少阳用手肘撑着脑袋，趴在桌子上看他，“你这德行，怎么看着一点也不担心？”
<br/>　　“担心什么，就算天塌了，也有高个子的顶着呢。”
<br/>　　“谁是高个子？”
<br/>　　“你呀，你就比我高。”
<br/>　　叶少阳白了他一眼，“跟你说正经的，你说，这算不算天劫开始了？”
<br/>　　“算吧，这要是不算，还得是什么才算。”
<br/>　　“可是之前大家都说，鬼王会先通过转世鬼童、将自己投影到人间，先把人间弄乱，怎么反倒正面刚上阴司了？”
<br/>　　四宝想了想，道：“两种可能，第一，计划赶不上变化啊，假设冷玉是鬼童，那就是她做了什么，打乱了鬼王的计划，让他没法投影，第二，都以为他要从人家开打，人间反而选择了阴司，暗渡陈仓，一下子就冲到酆都城了，这叫兵不厌诈。”
<br/>　　叶少阳听他说的，觉得有点道理。
<br/>　　四宝突然凑到他耳边，说道：“山羊，我问你个事，你是不是真要铲平太阴山？”
<br/>　　“这……有这个志向。”
<br/>　　“那我给你出个主意，太阴山眼下不是把军队都派出来跟阴司正面开战了吗，他们拖住了阴司的几路大军，但自己其实也被拖住了，如果这时候我们去抄他老家……这事儿没准真能成功！”
<br/>　　叶少阳听的怦然心动。“别扯淡！”
<br/>　　“什么扯淡，这叫直捣黄龙！”
<br/>　　“太阴山就算派了大军出来，在老巢也不会一点防备没有的，我们去等于送死。”
<br/>　　四宝拿起一个炸鸡腿，贪婪地啃着，一边说道：“你要是这么想，那你这辈子都没机会了，我给你分析一下第一，我也相信有驻防，但相对平时，已经算是兵力最少的时候了，你难道想等阴司打赢了，再去抄人家老家？真到那时候，人家才是闭门不出，一心防御，那才叫一点机会也没有了。再说，人家信心满满地倾巢而出打着一仗，谁胜谁负还不一定呢。”
<br/>　　叶少阳发呆地看了他一会，突然伸手在他额头上弹了一下。
<br/>　　四宝怒道：“搞什么玩意！”
<br/>　　“我看你是不是被军师附体了，这么有条理的分析，不像你的风格啊。”
<br/>　　“靠，你这就是看不起我，你就说吧，我说得有没有道理？”
<br/>　　叶少阳道：“满满的都是道理，但是……你能想到的，无极鬼王也能想到，我猜他一定做好了准备，去了就是送死。”
<br/>　　四宝把啃干净的鸡腿扔掉，擦了擦手，严肃地说道：“鬼王未必做好了准备，他应该不会相信，有人居然胆大到去抄他的老家！”
<br/>　　叶少阳心中一动，沉吟起来，“你也说是未必……”
<br/>　　“是啊，但这已经是最好的战机了，赌一把，你如果想平平安安地抄了人家老家，那十辈子都没希望，能有机会给你赌一把，就是最好的机会了！”
<br/>　　叶少阳缓缓点头，道：“你说的有道理。”
<br/>　　“所以？”
<br/>　　“可是我们压根就不知道太阴山在哪里，怎么进去。想赌都没机会。”
<br/>　　“找啊，线索总是有的。”
<br/>　　叶少阳皱起眉头，“你怎么这么激动，你以前没这么狂热啊。”
<br/>　　“最后一战。”四宝摊了摊手，“如果你决定要攻打太阴山，我怎么也得陪着，早打早安心，是死是活，也算有个交代，不然心里一直有个疙瘩，怪不舒服的。当然，你如果压根就不想打这一架，那当我没说。”
<br/>　　说完，四宝盯着叶少阳，朝他努了努嘴。
<br/>　　叶少阳端起皮蛋瘦肉粥，一口喝完，抹了抹嘴，说了一个字：
<br/>　　“打！”
<br/>　　“打！”
<br/>　　轩辕山北斗观，清长风也大喊了一声，一拳砸碎了桌子，站起来走了几步，望着大殿里被砸碎的轩辕大帝的神像，那些被打翻的桌案、香炉，脸皮一阵抽搐，门外，一个弟子跪在地上，惊慌失措地望着他。
<br/>　　“打！去点好人马，我们去人间，找道风算账！”
<br/>　　弟子领命离去。
