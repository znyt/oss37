　　第651章
<br/>　　野猪山地势险要，易守难攻，所以方大虎才可以在此地坐大。但他太过大意，便疏忽了防范，连瞭望的探子都没放置。所以才令百里胜神兵天降般摸到此地。
<br/>　　双方一照面，几乎是一面倒的局势。百里胜的部下一拥而入，逢人就杀，是人就砍，下手十分狠辣。不过短短时间，便有数人被杀，那方大虎本身武艺不弱，应在混江龙之上。但可惜这次他碰见的是卢俊义，只是一个照面就丧命在卢俊义的大枪之下。
<br/>　　混江龙混在人群之中，他那里想得到，自己刚想办法见到方大虎，方大虎的山寨便有了灭门之灾。他倒是想要归附百里胜，但看百里胜的样子，明显不需要俘虏。
<br/>　　他已经在悄悄撤退，乱战之中也没人注意到他。一脱离人群，他便撒丫子向外狂奔，便如野地的兔子，一溜烟不见了踪影。
<br/>　　一口气跑出山林，混江龙扶着大树呼哧呼哧喘气。自己可当真是倒霉呐，没能如愿投靠方大虎，反而差点将性命与他陪葬。
<br/>　　可如今天大地大，自己该往何处去，至于他本身，做孤魂野鬼的日子实在太久了。
<br/>　　休息一番后，继续向东走，前面找到村镇，也能有口水喝。
<br/>　　刚走不就，就见前面一个大汉坐在那里长吁短叹。混江龙一激灵，这人不是旁个，正是不停追杀自己的单雄信。
<br/>　　单雄信也有些郁闷，出来够久了，也受了一些苦，他早就想回山寨。可杀不了混江龙，有什么面目回蛤蟆城。这么久一点线索都没有，上次好不容易碰到，还被他侥幸逃了。
<br/>　　心头正憋闷之际，下意识回头看了一眼，正好与混江龙的目光撞在一起。
<br/>　　“啊！”
<br/>　　“啊！”
<br/>　　两个人几乎同时发出一声惊呼，不过一个是惊喜，另外一个是惊吓。
<br/>　　单雄信提起兵器就追：“狗贼，哪里逃！”
<br/>　　……
<br/>　　凉州，朔方城郊外。
<br/>　　农田连成一片，河渠纵横，刚播种不久的麦田已泛起新绿，绿意盎然一片。程大雷信步走在农田中，看到眼前一幕，长长舒了口气。
<br/>　　系统奖励的技能看来是生效了，照这个劲头下去，本来到秋天才能收获的庄稼，在这个夏天就能收获，庄稼的生长时间大概缩短了一半。良田千顷，这些庄稼收获后，朔方城就算是达到自给自足，另外还能有些结余，补充蛤蟆城的消耗。
<br/>　　在这个商业落后的时代，程大雷也就只能种种田了。但种田是个力气活，耕田播种除草这些出力的事情自不必提，除此之外还要担心野兽破坏。现在正是庄稼的生长的关键时刻，每天田里都有人守着，防范着野猪野兔什么的。
<br/>　　当然，这些事情不必程大雷吩咐，和珅早已安排好了。从接手朔方城后，和珅便如打了鸡血一般，夜以继日的废寝往日，程大雷很担心他某日过劳死了。
<br/>　　内政与军事同样重要，有时候甚至更加麻烦。庄稼从种到收，皆需要专人负责，等到收获时，还要担心有人来抢粮烧粮。天干物燥，只需一把火就可以让一年辛苦颗粒无收。
<br/>　　除了负责种田，和珅现在还在做两件事，一是将破败的朔方城修修补补，使它像个样子；二是想办法扩充朔方城的人口，目前朔方城不过两万余人，还远远达不到程大雷的需求。
<br/>　　有了人口才能有人气，有了人气一座城池才能慢慢活过来。农业、商业、军事互相补充，小城池成大城市，如此自己的势力才能慢慢壮大起来。
<br/>　　程大雷从郊外返回朔方城，最近他忙着朔方城的事，诸多事物便在朔方城处理。杨国忠现在已经接手蛤蟆城的事物，他虽然年纪轻轻，但果然不愧是做奸臣的料，一切事物做起来井井有条，已经能胜任蛤蟆城的工作。
<br/>　　阿喜将喜字队收集到的情报送到这里，程大雷靠此了解帝国目前的情形。
<br/>　　总而言之，帝国仍旧很乱。东海反了刘顶天，帝国其他地方也不安分，多有人举兵造反，只是刘顶天搞得比较大而已。
<br/>　　东海那边是怎样情况，程大雷也并不了解，毕竟两地相隔太远，一来一去都需要时间。程大雷现在只盼着刘发财回来时，能顺便将东海的情报带回来。
<br/>　　他与黑衣老尼结伴去寻华氏一族，如今已经有些时间，却是不知事情进展如何。
<br/>　　除了东海之外，其他各地也发生了诸多事情，无非就是你打我我打你，互相之间打来打去，也没有太大意思。
<br/>　　真正吸引程大雷注意的，是京州的事。阿喜刚刚送来的消息：尉迟离死了。
<br/>　　尉迟离也算是军中老将，不过死得委实有些窝囊。早晨出恭的时候，一口气没喘上来，倒在了马桶中，浑身粪尿臭不可闻。
<br/>　　尉迟离一死，怕就没人能辖制相府的势力扩张，如此看来，长安也要乱了。
<br/>　　长安是个烂摊子，程大雷早早便看出这一点，诸多世家盘踞在这里。各家各户都有自己的利益，他们几乎把持了京州的商业、农业、军事、政治。以前，他们势力的覆盖范围是帝国十三州，只不过如今天下大乱，他们也只能控制京州一地。
<br/>　　李乐天想要掌握京州，从而掌握天下，其他都可以不管。但十五万禁卫军必须牢牢握在自己手里，只不过，禁卫军中有了太多世家子弟，李乐天也不是想掌握就能掌握的。
<br/>　　“阿喜。”
<br/>　　程大雷心中思索一阵，将手中的情报合上。
<br/>　　“大当家，您吩咐……”
<br/>　　“传讯给长安的兄弟，给尉迟离送一幅挽联，毕竟是个人物，可惜这样死了。”程大雷道：“另外，也告诉相府的人，帝国还有我们这家势力，莫让他们太猖狂。”
<br/>　　“是，我立刻就着手办。”阿喜答应一声：“大当家，挽联上写什么……”
<br/>　　“这个？”程大雷顿了顿：“去问和总管吧。”
<br/>　　程大雷本身文墨功夫不行，也没有特意训练过这方面的修养。于是也只能去问和珅。
<br/>(本章完)
