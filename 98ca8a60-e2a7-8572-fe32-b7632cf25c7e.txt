夏妍妍穿着一身优雅的连衣裙来到了大厅里，她在大厅里看到了很多有威望的医学大师和教授，这算是她第一次参加这种场合，所以心里十分的激动和兴奋。
<br/>
<br/>        这时钱教授走了过来，“妍妍，你来了？”
<br/>
<br/>        夏妍妍迅速上前挽住了钱教授的手臂，“导师，我来了，今天来了好多教授啊，我以前在电视采访上看过他们，还拜读过他们的医学论文。”
<br/>
<br/>        钱教授道，“妍妍，这些年你一直在念书，又刚从圣黎院留学归来，手上没有医学界的人脉，这一次我将你带到这个聚会上就是想多介绍一些人给你认识，最重要的是，你要拜师了。”
<br/>
<br/>        夏妍妍心里咯噔一跳，是的，她要拜师了，每一个医学大师都有自己的医学团队，他们从事着最先进的临床医学研究，还有着广泛的关系和人脉，她需要加入其中。
<br/>
<br/>        钱教授四周看了一下，“你们研究院的李文清院士还没有回来么？”
<br/>
<br/>        夏妍妍摇头，“听说李院长还要几天才能回来。”
<br/>
<br/>        钱教授有点可惜了，“现在医学界得高望重的院士屈指可数，李文清绝对是其中一个，而且他作为枢密研究院的院长，和帝都那里的关系匪浅，这里所有人都比不上李文清院士，如果能拜他为师，他肯定能带着你进入帝都，那可是医学界的中心。”
<br/>
<br/>        夏妍妍心里怦怦直跳，露出了渴望，“导师，我就是想拜李院长为师，所以我才进入枢密的，现在李院长还没有回来，我把握不大，毕竟这些年李院长一个关门弟子都没有，他眼光太高，至今还没有看得上眼的。”
<br/>
<br/>        钱教授拍了拍夏妍妍的小手，“不用担心，我跟李院长有点交情，等他回来，我会帮你游说的。”
<br/>
<br/>        “真的么？导师，你对我太好了！”夏妍妍露出了甜美的微笑。
<br/>
<br/>        这时陆茵茵出场了，今晚的陆茵茵一身黑色长裙，整个人优雅干练又美丽，她拿着高脚杯穿梭名利场，俨然是主角的风范。
<br/>
<br/>        夏妍妍一眼就看到了陆茵茵，她很崇拜这位校长的，她希望自己以后也能成为校长这样的人。
<br/>
<br/>        “导师，我们去见见校长吧。”
<br/>
<br/>        “好。”
<br/>
<br/>        钱教授将夏妍妍带到了陆茵茵的面前，“校长，这是我收的学生夏妍妍，我记得妍妍也是通过校长你的手进来的。”
<br/>
<br/>        陆茵茵的目光落在了夏妍妍的身上，她想起来了，前几年陆寒霆曾经亲自打过电话给她，推荐一个人进圣黎院。
<br/>
<br/>        她和陆寒霆的关系一直就很生疏，陆寒霆能为了一个女孩儿打电话过来，她当时还很诧异，她自然就同意了，这个女孩儿就是面前的夏妍妍。
<br/>
<br/>        不过当时她问陆寒霆跟这个女孩儿是什么关系，陆寒霆只说为了还恩情。
<br/>
<br/>        夏妍妍在圣黎院这种天才汇聚的地方并不拔尖，所以陆茵茵的目光也没有在夏妍妍的身上停住过，慢慢就淡忘了。
<br/>
<br/>        见陆茵茵看着自己，夏妍妍迅速露出了完美的微笑，“校长，你好，我一直就很仰慕你，你是我的偶像。”
