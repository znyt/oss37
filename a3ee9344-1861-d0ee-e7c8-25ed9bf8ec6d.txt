　　如果自己在出枪的同时，快速向前推进十米，那么自己就能提高百分之十的概率击杀他们中的一个，而且还有李毅峰和慕容飞这两个狙击高手，不说可以把他们两个全都留下，但是留下一个应该不存在问题。
<br/>　　“明白！”
<br/>　　“收到！”
<br/>　　李毅峰和慕容飞分别作出回答。
<br/>　　“快速反应小组，没有我的命令，不要擅自动手！”
<br/>　　罗昊又通过无线电向周围负责监控的快速反应部队，做出要他们待命的要求。
<br/>　　只是这次罗昊没有在第一时间就得到K国快速反应部队的回复，直到罗昊又重复了一次命令后，耳麦中才传来快速反应部队监控小组组长的回答。
<br/>　　罗昊没有直接动手，主要还是有所顾虑。智天使这家伙狡猾无比，他和吴云既然敢这么明目张胆的出现在现身，必然是有着准备的。不说，SHADOW手里掌握的那些能在短时间内就让网络陷入瘫痪的黑客病毒，罗昊相信智天使一定还有着其他准备。
<br/>　　站在窗户边的智天使拿出一个遥控器一样的东西，对着站在街道对面的罗昊晃了晃，然后伸出拳头做出一个爆炸开花的手势，告诉罗昊，这附近有炸药不要轻举妄动，不然他就会引爆炸药。
<br/>　　“不要轻举妄动，他手里有引爆器！”
<br/>　　罗昊通过耳麦向周围人做出吩咐；可是，罗昊话音刚落，他的耳边就传来一声清脆的枪声，紧接着，他就看到一颗子弹在夜色下划出一道粉红色的弹道轨迹飞向站在窗户边的智天使。
<br/>　　罗昊眼睛猛地瞪大，他无力阻止那颗子弹，只能眼睁睁的看着那颗子弹飞向写字楼，罗昊不知道在此之后会发生什么事情，但是他可以肯定一件事，那就是这颗子弹肯定打不中智天使，因为他看到了智天使在子弹飞向他的时候，就做出了一个规避动作。
<br/>　　哗啦！
<br/>　　子弹击碎了那块玻璃墙，飞入到写字楼内，碎裂的玻璃洒落下来，落在地面上砸得粉碎。
<br/>　　清脆的枪声，在这条平时本应该很热闹但是现在却没有行人和车辆通过的马路上空回荡着，是那么的刺耳，刺痛着罗昊的耳膜！
<br/>　　指挥中心距离目标写字楼并不算远，所以在片刻之后，指挥中心内的人也是听到了从目标写字楼方向传来的枪声。
<br/>　　赵天宇怔了怔，立刻扔掉自己手中的铅笔，快步走到窗户边，拉开窗帘往外看去，并且通过耳麦向罗昊询问现场情况；可是赵天宇接连询问了两遍，回答他的除了沉默还是沉默。
<br/>　　罗昊听到了从耳麦中传来的赵天宇的询问，但是他没有回答，他只是怔怔看着那堵被子弹击碎的玻璃墙，片刻之后，他对着耳麦愤怒的吼道：“谁开的枪！？是谁开的枪！？我说了待命，待命，你们都是聋子吗！？”
<br/>　　“报告！”
<br/>　　K国快速反应部队监视小组的狙击手在罗昊吼完后，按住耳麦回答道：“是我开的枪。他在我的狙击视野中，我有把握打中他！”
<br/>　　“他的手里有炸弹遥控器，你脑袋里装得是大便吗！？”罗昊怒骂道。
<br/>　　罗昊的怒骂也是激起了监视小组狙击手的火气，语气不悦的说道：“我的教官说过，作为一个狙击手要有自我判断的能力，机会可能一瞬即逝，所以必须要把握住机会，做到一击必杀！”
<br/>　　“那么你现在一击必杀了吗！？没有！”罗昊现在恨不得冲上前去狠狠地抽那个自作主张的狙击手两个耳光，“你的教官，会因为你今天晚上的愚蠢，而后悔有你这么一个比驴还要蠢的蠢货学生！因为你的愚蠢，今天晚上会有很多无辜的人死于非命，而你就是夺走他们生命的凶手！”
<br/>　　“对不起长官，我并不认同你的话！”狙击手依旧态度强硬的说道：“我认为，他刚才那个动作只是虚张声势！”
<br/>　　罗昊绝对要比这个愚蠢的狙击手了解智天使，这家伙绝对不会做虚张声势这么无聊的事情。
<br/>　　智天使会虚张声势吗？显然不会！写字楼内，散落着一些玻璃碎片的地面上，那个之前被智天使握在手里的遥控器此时被仍在地上，遥控器上，一个红色小灯以很高的频率闪烁着，这快速闪烁的红色小灯就像是在召唤着死神的降临……
<br/>　　距离目标写字楼差不多一千米的地方，这里是多条地铁线路的转换站，所以这里的人流量也要比其他地铁站要多，加上此时正值下班高峰，进出地铁站的人就更多了，电梯上站满了人，有些人因为忍受不了电梯的拥挤，情愿选择走楼梯为的就是能赶早一班地铁，尽快回家去。
<br/>　　只是此时拥挤在地铁站的人都不知道，死神正缓缓降临在这个地铁站。
<br/>　　一条地铁线路的铁轨上，一枚经过了改装，增加了遥控起爆装置的反坦克地雷安静已经进入了倒计时，只是在地铁站等候着地铁进站的乘客或是埋头看着自己手里的手机，或是正在打着电话，或是看着手里刚从地铁站外买的晚报，谁也没有注意到反坦克地雷上飞快跳动的红色倒计时……
<br/>　　这时地铁呼啸着冲进站，地铁减速跟铁轨发出巨大的噪音，站台上，等候着地铁的乘客这个时候停下自己手中正在做的事情，等着地铁开门。
<br/>　　而就在地铁门打开的瞬间，两条铁轨之间那颗经过改装的反坦克地雷上的计时器也是归零了。
<br/>　　轰！
<br/>　　震耳欲聋的爆炸声响起，一团火球从地铁下方冲起，冲破了地铁的阻拦，直接冲出了地面。马路被炸开一个巨大的缺口，火光冲天而起，一辆正巧经过的汽车也是被掀飞到了半空中。
<br/>　　罗昊能清楚的看到那团火光冲上天空，能清楚的听到从远处传来的爆炸声，能清楚的感觉到自己脚下的地面，因为爆炸而产生的抖动……
<br/>　　罗昊呆呆的看着爆炸传来的方向，没有回过神来；几秒钟前，态度还异常强硬的狙击手，此时却是吓得脸色苍白，比纸还要白……
