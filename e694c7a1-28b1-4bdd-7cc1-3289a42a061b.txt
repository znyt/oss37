　　第427章
<br/>　　“快些，再快些！”
<br/>　　远处响起了男人暴躁的催促声，甚至还没有看见人，声音已经震得耳膜作响。能将声音传这么远，声震如雷者，除了张三爷找不到第二人。
<br/>　　程大雷猜得果然不错，远远看着张三爷的人马已经到了。不仅仅是他，秦蛮、关鱼、高飞豹四支部队几乎同时到达。
<br/>　　四人率领的都是步兵，行军速度和赵子龙的轻骑兵不能比。但四人路上都较着劲儿，谁也不想落后其他人。
<br/>　　张肥是如此心思，其他人心思也是相同。大家心里想的都是：快些吧，再晚就要被赵子龙把人杀光了。
<br/>　　此刻赶到一看，赵子龙还在外围，并没有攻进敌军大营。
<br/>　　张肥哈哈大笑，道：“子龙兄弟果然仗义，没有自己一个人吃独食。”
<br/>　　赵子龙不是不想吃独食，而是……吃不下去。
<br/>　　在张肥四人没赶到之前，己方和敌方的兵力比达到可怕的十比一。如果真的拉开战线，赵子龙利用轻骑兵的超高机动性，一点一点把敌人蚕食，也未必做不到。可要在一天之内，将三万凉州兵击溃，确实有些强人所难。
<br/>　　面对张肥的褒奖，赵子龙虚伪的应承着。心里想的却是，自己是不是太保守了。就该找个法子，直捣黄龙，在他们到来之前，解决战斗。
<br/>　　不仅仅是张肥，秦蛮三人脸上也露出满意的表情。大家心急如焚的赶路，怕的不就是赵子龙吃独食么。如今凉州兵就在眼前，日夜操练，足足磨刀三年，为的不就是今日亮刀么。
<br/>　　当然，诸人心仪的战场还是长安，那里才是群雄逐鹿，英雄会武。可无奈和大当家不许出兵，大家也只能用眼前这支凉州兵泄泄杀意。
<br/>　　但陆洛一张脸已经是铁青无比，他看着敌人的兵力越来越多，已几乎和己方持平。而且这些人你一言我一语，脸上流淌着兴奋的眼神，似乎把自己当成一盘豆芽菜，谁过来谁都能吃一口。
<br/>　　“陆大人，我们向来和蛤蟆城井水不犯河水，这中间是不是有什么误会，不若向他们求和吧？”手下人向陆洛道。
<br/>　　“井水不犯河水！”陆洛冷哼一声，恨不得一剑将这人脑袋砍掉。
<br/>　　这货该不会是个瞎子聋子吧，但凡有眼睛有耳朵的，谁不知道程大雷是宋伯康的眼中钉肉中刺。在自家后花园，宋伯康怎么愿意有一股势力茁壮成长。之所以两家保持着『和平相处』，并不是因为宋伯康不想收拾他，而是……收拾不了他。
<br/>　　如今留下五万人守凉州城，防的是谁，防的不就是程大雷么。
<br/>　　黄昏日落，荒原上镀一层金黄。陆洛立在哪里，脸上阴晴不定，目光不停闪烁。
<br/>　　若有机会，除掉程大雷，必然是大功一件。而眼前，未必不是个机会。
<br/>　　己方以逸待劳，并没有消耗太多体力。而敌人长途奔袭，必定筋疲力尽。而在兵力上，己方甚至占有一定优势。
<br/>　　人生能有几回搏，搏得过今天，自己就斩掉这恶贼，天下扬名。
<br/>　　陆洛虽然是个文人，但并不缺乏勇气，千古文人侠客梦，今天陆洛就要告诉程大雷，甚至告诉整个凉州：你以为本人是个书生，本人其实是个武夫。
<br/>　　“全军集结！”陆洛持剑下令：“杀！”
<br/>　　而恰好，程大雷也没有和陆洛交谈，说些今天误会一场，你好我好大家好之类的话。甚至根本不需要程大雷下令，关鱼五人已发动进攻。
<br/>　　程大雷今天没有指挥权，仔细说，算是个裁判，也判断一下五军的战斗力强弱。
<br/>　　而五支队伍，算五支单独的作战单位，单独都有一套攻防系统，互相之间并不配合。
<br/>　　迎着落日的余晖，双方发起对冲。
<br/>　　依旧是赵子龙的轻骑兵队伍率先出动，毕竟，四条腿要比两条腿跑得更快些。三千如狼似虎的悍将，杀入敌阵，将凉州兵冲开一条血路。
<br/>　　然后是双方步兵的正面交锋，这是真正的血战，在这种情况下，秦蛮的蛮字军发挥出极大作用。
<br/>　　真正收到奇效的是梅花阵，十人一朵梅花，凝聚成一团团攻不破的防御，而在乱军丛中，梅花阵时而合拢，时而散开，不停收割着性命。
<br/>　　步兵正面冲锋，最怕的不是体力耗尽，最怕不是敌人武艺出众，最怕的是找不到敌人在哪里。现在蛮字军已梅花阵的形式合为一体，倒是不惧这一点。
<br/>　　当然，这少不了秦蛮的指挥，迈入绝世后，秦蛮的武艺和统率能力都提升一大等级。手中一杆长枪，无人能敌，另外还有余暇，指挥梅花阵的攻击。
<br/>　　蛮字军大概有六千人左右，组成梅花阵后，撒入了乱军中六百朵梅花，像六百朵不破的堡垒。
<br/>　　张肥、关鱼、高飞豹一看，呀，这小子有压箱底的东西啊，为什么我们不知道。
<br/>　　张肥三人都是战场上的暴徒，此刻在秦蛮的刺激下，人人争先，杀敌无算。凉州兵觉得，自己不是在和人战斗，而是和虎狼搏击。
<br/>　　于此同时，山谷内传来一阵喊杀声。陆洛心中一慌，忍不住回头看去，这个时候，高飞虎从山谷中杀了出来。
<br/>　　高飞虎都憋屈了很久，千里投奔程大雷，最后却搞得需要程大雷来救，终究是有些丢人的。此刻见程大雷已经在山谷外发起总攻，他们也从山谷内部展开突围。
<br/>　　陆洛立刻感觉到腹背受敌是一种怎样的滋味，前后左右都是敌人。正面是梅花阵扛住冲击，张肥、高飞豹、关鱼三人又是杀人如麻，而赵子龙的轻骑兵一刻也没有休息，在队伍中来回冲杀。
<br/>　　很快，程大雷和高飞虎的两支部队便合到一起，士气又一次飙升到一个恐怖的程度。
<br/>　　“大人，撤吧！”手下高呼道。
<br/>　　陆洛一张脸黑得极其难看，已经不用他下令，手下士兵已经开始逃跑。在整个帝国，凉州兵都是有数的强军，但帝国军打不过就逃的习惯也被他们完美继承。
<br/>　　兵败如山倒，一旦士兵开始溃逃，就绝不是陆洛能够阻止的。
<br/>　　他站在乱军之中，忽然有种无力回天的感觉。他以为他是个武夫，其实他只是个书生。
<br/>(本章完)
