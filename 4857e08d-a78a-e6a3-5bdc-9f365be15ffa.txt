<br/>　　旋即，屋内就响起一阵悦耳的娇吟声，半小时后，屋内那令人羞耻的声音方才渐渐消散。
<br/>　　床榻上，小医仙柔美的脸颊绯红如霞，美眸痴痴的趴在他的怀中。

<br/>　　可是，怀中的佳人现在能看能摸，唯独就是不上突破最后一道防线。
<br/>　　无奈，只好俯下身子，在她晶莹剔透的耳边低声呢喃了几句，也不知道说了些什么，小医仙脸上的红霞直达耳根，看上去煞是可爱。
<br/>　　但最后拗不过他，纤纤玉手有些颤抖的握住慕清霄最关键的部位，娇羞无限，随后干起了没羞没臊的事情。
<br/>　　竖日清晨，天刚蒙蒙亮，慕清霄就习惯性的醒来，望了一眼怀中身无寸缕的娇躯，苦涩一笑。
<br/>　　离开的十天内，小医仙的实力已经从一星斗者，攀升到了九星斗者，一瓶伴生紫晶源的效果非常可观。
<br/>　　当然，她的资质也是不可否认的，拥有厄难毒体这种体质，资质本就不可能差。
<br/>　　储存空间中还有两瓶，依旧能够助她提升不少修为。
<br/>　　要知道，两人离开青山镇不过十天左右，小医仙的修为就攀上到如此程度，简直就是在坐火箭。
<br/>　　半小时后，小医仙也醒了过来，见到慕清霄脸上的坏笑后，却不敢看他，脑海中不禁浮现荒唐的一幕。
<br/>　　最终，羞涩化作温柔，往她怀中挤了挤，只要能陪在自己身边，不管做什么，她都愿意。

<br/>　　前往了一趟商店，与冰皇告别后，慕清霄就搂着小医仙，周围空间一阵扭曲，两人的身影就消失的无影无踪。
