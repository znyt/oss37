<br/>　　“地下室！”
<br/>　　周柏突然想到了什么，对罗昊说道：“被关押在这里的时候，我曾听西伦科特提起过地下室，如果西伦科特就藏在这围墙以内，并且这两栋别墅中有地下室的话，那么西伦科特现在一定在地下室中。”
<br/>　　“找！”罗昊斩钉截铁的说道：“就算是挖地三尺也要把西伦科特这个混蛋给找出来。”
<br/>　　就在这个时候，别墅外突然传来汽车引擎声，是那辆之前离开的改装SUV回来了！
<br/>　　如果按照正常的巡逻时间，这辆改装SUV还不到返回的时间，想来应该是察觉到了发生在山顶别墅中的情况，所以赶回来进行支援的，只是他们的支援太迟了。
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　一名影子部队队员站在改装SUV中，操控着M134转管机枪，对别墅的方向进行扫射，对别墅中的特种部队队员进行火力压制，而改装SUV中其他四名影子部队队员则是跳下SUV，朝着别墅靠近过来。
<br/>　　“狐狸，去二楼给我把那个该死的机枪手解决了！”罗昊对着李毅峰吼道。
<br/>　　李毅峰抱着M2010狙击步枪转身冲上楼梯。
<br/>　　没过多久，别墅外外装SUV上的那挺M134转管机枪就哑火了。
<br/>　　罗昊对着身边的特种部队队员低声吼道：“干掉他们！”
<br/>　　三名被罗昊救出来的“雷鸣”特种部队队员跃出别墅，在身体接触地面的瞬间，做出一个翻滚的动作，半蹲着身体用手中的自动步枪打出点射。
<br/>　　哒哒哒！哒哒哒哒哒！哒哒哒哒！
<br/>　　而别墅中其他特种部队队员也是跟着冲出别墅，缺少了M134转管机枪的火力压制，这些特种部队队员就像是冲出闸门的猛虎，憋了一肚子的火在这一刻全都发泄了出来。
<br/>　　尽管这四名影子部队队员身上穿了防弹衣，可是就算是防护等级四级的防弹衣也不可能抵挡住十几支自动步枪，外加两支使用特制子弹的狙击步枪的火力。
<br/>　　眨眼间，四名从从赶回来支援的影子部队队员就被打成了马蜂窝，他们就连拉开身上手雷同归于尽的机会都没有！
<br/>　　为了防止还会有人进入这个院子，罗昊留下了一般人在地面构建起防御，其他人则是分散到两栋别墅中仔细寻找起可能藏在这两栋别墅中的地下室。
<br/>　　而在此时，地下室中的西伦科特打开了冷藏柜上的密码，将冷藏柜打开，一股股白色的寒气从冷藏柜中涌出来，冷藏柜中放着三个直径有五公分，高有三十公分，套了金属外壳的玻璃瓶，玻璃瓶中装着紫红色的液体，赫然就是P病毒！
<br/>　　西伦科特手中掌握的这三瓶P病毒是“暗月”最新升级后的P病毒，除了可以通过空气传播外，里面的病毒如果沾染到人的皮肤，也会让人感染到病毒，而且这种升级后的P病毒传染力更加强大，“暗月”内部给这种全新的P病毒代号是“毁灭-3”型融合性致命病毒。
<br/>　　发生在地面上的战斗，西伦科特自然不可能不知道，但是他现在已经顾不上外面的情况，他知道，他杀了联合行动特种部队队员，联合行动司令部不可能无动于衷，更何况，他本身就是美国军方通缉令上的人，所以这次联合行动司令部必然是来势汹汹，不干掉自己绝对不罢休！所以，西伦科特此时想的就是要怎么样安全的从这里逃出去。
<br/>　　布克尔特建造这两栋别墅是为了跟当地政府高官会面，交易，合作的，这种地方自然是要做好充足准备，防备国际刑警和联合军突击行动的。所以在这两栋别墅中都是有着地下室，而西伦科特现在所在这栋别墅地下室则是用着一条暗道可以逃出去，而且在地下室中还准备了一辆防弹GLK。
<br/>　　西伦科特戴上手套，把三个玻璃瓶小心的拿出冷藏柜，装入液氮低温瓶中，盖上盖子塞入背包中，这三瓶“毁灭-3”型病毒是西伦科特手中的底牌，也是最后的底牌！
