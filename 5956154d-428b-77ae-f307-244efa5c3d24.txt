
<br/>　　<br/>　　    半个小时后，中铁某局的领导还有施工队的人全都来到了出事的这段盘山道上，几乎集体脸都绿了，一天死一个，两天两条命，这种节奏太让人措手不及了，这已经算是严重事故了。<br/>　　<br/>　　    “怎么回事啊，小贺？”局领导不满的问道，死两个人是责任事故，当领导的估计要背锅了。<br/>　　<br/>　　    贺然咽了咽唾沫，还没有从徐伟的坠崖回过神来呢，连续两天，第一天他眼睁睁的看着刘洋被一块石头给砸死了，仅仅相隔了一天，又眼睁睁的看着徐伟开车坠崖了，最关键的是，这两次同事的死亡都和贺然仅差一步之遥了，前天那块石头是擦着他耳边飞过去的，哪怕稍微只要偏两公分，那死的就是贺然了，今天同样如此，如果是他开车的话，坠崖的也是他了。<br/>　　<br/>　　    所以，贺然等于是连续两次和死神擦肩而过了。<br/>　　<br/>　　    “我，我本来是和徐伟开车准备去山下买，买纸钱的，然后打算回来晚上烧给刘洋的，但，谁知道车开到这就趴窝熄火了，老徐让我下去推车，他打算别一下，看能不能点着火，可没想到我推着推着，车子被发动了之后，一头就朝着悬崖下面冲了过去……”<br/>　　<br/>　　    其实，贺然不知道的是，这是贺三刀给他的那道求来的护身符起作用了，使他连续免糟了两劫，但王惊蛰也说了，这个坎他是躲不过去的，等到贺然身上的那道护身符的作用被磨没了，那横祸就该降临到他的身上了，刘洋死了，徐伟死了，下一个就是他了。<br/>　　<br/>　　    这就跟以前有部美剧讲述的差不多，那部电影叫死神来了，说的是男主角偶然间觉察到了意外，和他一起的人都死了，他意识到这都不是巧合，所以就想方设法的避免自己出现意外，但没想到最后大结局，这男主角仍然没有能逃脱得了宿命的轮回，被死神盯上的人一个都没跑的了。<br/>　　<br/>　　    说下护身符这种玩意，其实并不是一直都具备保命护身作用的，笼统的，用通俗易懂的话来解释就是，护身符里含着能量，分为多少和强弱，这个能量如果很多很多还特别强，并且是由得道高人开光或者撰写的话，可能会保你一条命，一次两次也没准，但每保一次护身符里的能量就会消耗一些，等到能量没了，这护身符就啥作用都没有了，其效果等同于一张普通的黄纸了。<br/>　　<br/>　　    此时，贺然就是连续两次躲过了命劫，而与此同时那身上的护身符也马上就要报废了，距离明天还有不到二十四小时。<br/>　　<br/>　　    没过多久警方也来了，排查现场，进行推论，贺然也被暂时控制了起来了，最后警方和消防的人全都下到了悬崖底部，先是准备救援看看徐伟有没生还的可能，但人从这个高度摔下去，几乎当场就死了，然后他的尸体就被拖了上来。<br/>　　<br/>　　    贺然怔怔的看着徐伟的尸体，脑袋里一片空白，在单位里面刘洋跟徐伟，既是他同事又是朋友，三人已经相处多年了。<br/>　　<br/>　　    世上最大的痛苦之一，莫过于是生死两茫茫，天人两隔了。<br/>　　<br/>　　    警方简单的检查了下那辆皮卡车，初步认定是汽车的转向机出了问题，导致方向被锁死了，至于徐伟最后为啥没有踩住刹车，则是没有一个明确的定论了，只能认为他当时太过慌张，手忙脚乱下甚至都有可能踩上了油门也说不准。<br/>　　<br/>　　    “这是从车里拿出来的物品，你看看哪些是你的，哪些是死者的……”皮卡肯定是吊不上来了，警方也只能将车里的物品都搜罗上来，交给贺然辨认。<br/>　　<br/>　　    贺然看着一堆东西，麻木的指出了几个，当他看见那个针线盒的时候就愣了下，没想到警方把这玩意儿也给带了上来。<br/>　　<br/>　　    现场勘查完之后，贺然也不能在下山了，就回到了工程指挥部驻地，他把自己关在了宿舍的房间里，宿舍就三张床，他一张，剩下的是徐伟和刘洋的，此时就只剩下了他自己，另外两个已经踏上了黄泉路。<br/>　　<br/>　　    贺然痛苦的抓着头发，喘息了几口气后，红着眼圈倒在了床上，一直躺倒晚上都是浑浑噩噩的，两人接二连三的死去给他造成了不小的打击。<br/>　　<br/>　　    这天晚上，一直到后半夜的时候，贺然才迷迷糊糊的睡着了，睡梦之间他又做了个梦，梦见了那个搭车的女人，对方的身影有点淡，说话的时候语气仍然是邦邦硬的，始终都在重复着一句话。<br/>　　<br/>　　    “你答应我的事怎么还没去做……”<br/>　　<br/>　　    隔天早上，清晨时分贺然醒来，躺在床上皱着眉头狐疑的想着昨晚做的那个梦，然后拿出那个针线盒又摆弄了几下，之前他和刘洋徐伟三人就是随手一拿简单看了两眼，根本就没注意看，此时他仔仔细细的端详了片刻，发现这针线盒挺古怪的。<br/>　　<br/>　　    盒子本身似乎有很严重的被火烧过的痕迹，上面很多焦糊的地方，里面的针还有被燎黑的迹象，特别是几个缝衣线，用手一撮其中有的变成了灰。<br/>　　<br/>　　    这玩意儿瞅着挺怪的，可是贺然看了半天也没多想，就又放在了一旁，这时候他旁边的电话就响了，拿起来一看是他爸打过来的。<br/>　　<br/>　　    “喂，爸”电话接通后，贺然情绪失落，语气低沉的问了一句。<br/>　　<br/>　　    贺三刀今天一早上起床的时候，就始终心神不宁的，吃饭吃了几口就难以下咽了，这种莫名的烦躁让他特别的不安，然后就想到了王惊蛰的到来，他木然间意识到，自己儿子的大祸似乎就在最近几天了，于是赶紧给贺然打了个电话过来，电话一通，听见贺然的声音，他才稍稍的放下了心。<br/>　　<br/>　　    “儿子啊，你这几天工作的怎么样，顺心不的？”贺三刀问候了一声，然后叹了口气，说道：“你说你非得跑那么远，在京城干点什么不好呢，我还得惦记你，这眼看就快要入冬了，你们那边也该下月封路了吧，不行你就早点回来”<br/>　　<br/>　　    “嗯，我知道了，爸……”<br/>　　<br/>　　    听见贺然情绪不高，贺三刀就问道：“你怎么了，说话有气无力的呢？”<br/>　　<br/>　　    贺然眼圈“唰”的一下就红了，他抿着嘴唇哽咽着说道：“爸，工地出事了，刘洋和徐伟都死了，就是曾经去过咱家的那两个同事”<br/>　　<br/>　　    电话那头，贺三刀一听当即就愣住了，他呼吸略微急促起来：“你，不是，那你呢，你有没有事啊？”<br/>　　<br/>　　    “我还好……”
