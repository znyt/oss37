　　“楚风眠！是你！你怎么会这么变的这么强横！”
<br/>　　被打破身躯，这万龙皇才终于是看清楚了楚风眠的身形，不可思议的怒声吼道。
<br/>　　他上一次见到楚风眠的时候，楚风眠还不过只能够是跟他平起平坐。
<br/>　　他本以为他得到奇遇，下一次交手就可以杀了楚风眠，一雪前耻，可没想到再次遇见，他就连跟楚风眠交手的资格都没有了。
<br/>　　“也算是让你当一个明白鬼，知道死在了谁的手里，现在你也该死了。”
<br/>　　楚风眠眼神冰冷无比，他也是故意让这万龙皇知晓是楚风眠杀了他的。
<br/>　　不然以现在楚风眠的实力，突然出手，这万龙皇都不会任何的反应过来，就会被楚风眠所诛杀。
<br/>　　楚风眠现在的实力，连半步天人境都能杀，更何况这一个小小的万龙皇。
<br/>　　楚风眠开口的一刻，手指一用力。
<br/>　　万龙皇的身躯在星空中猛然爆炸，鲜血狂飙，还没有来得及飞走，便是被一股力量无情的拉扯，进入这吞天秘术之中去。
<br/>　　这万龙皇的一身精血，也被吞天秘术所吞噬，再也没有重新复活的可能。
<br/>　　万龙皇陨落。
<br/>　　青梦在一旁看到这一幕，都是百感交集，之前万龙皇，可是万龙堂的堂主。
<br/>　　身为八堂之主的他，在圣龙秘境内地位超然，再进一步，也就可以晋升圣龙秘境的高层之一了，地位高高在上。
<br/>　　可自从他选择跟楚风眠为敌的一刻，就注定了他今日的死。
<br/>　　万龙皇，在现在的楚风眠眼中只是一个小人物，杀了万龙皇之后，楚风眠的注意力依然还在那天星河之上。
<br/>　　他要先收取了天星河，在进圣龙秘境之中，去找其他人来算账。
<br/>　　短短的时间内，天星河的本源，已经是被楚风眠收取大半了。
<br/>　　整个天星河，都在剧烈的抖动起来，试图反抗楚风眠的收取。
<br/>　　整个圣龙秘境内，都是在不断的震动，宛如末日降临一般。
<br/>　　“怎么回事？到底是什么人在攻打天星河？”
<br/>　　又是数道身影，从圣龙秘境之中飞了出来，许许多多在圣龙秘境内部潜修的老古董，如今也因为天星河的震动，纷纷苏醒了过来。
<br/>　　在这些人物之中，许多都是楚风眠的熟人。
<br/>　　悟剑堂堂主，还有悟剑堂上一任的堂主，罗剑古帝，还有万龙帝等，都赫然在列。
<br/>　　当然在这些人物之中，许多人的实力，甚至是比起他们还要强横，只是楚风眠没有见过。
<br/>　　这些人，应该都是圣龙秘境神秘的高层，平日里都不曾从圣龙秘境内现身的。
<br/>　　在圣龙秘境之中，一旦是踏入九劫古帝，便是就会晋升成为高层，潜心闭关修炼，冲击半步天人境，没有重要的事，根本不会现身。
<br/>　　这一次楚风眠收取天星河的动静，实在是太大，将他们这些闭关不出的老古董，如今都给惊醒了过来，纷纷出关。
<br/>　　“万龙皇的精血，万龙皇死了？”
<br/>　　这万龙帝身形飞了出来，一下子就闻到了周围淡淡的血腥味，便是脸色大变的说道。
<br/>　　“不错，万龙皇想要瓜分圣龙宗，罪无可赦，我便是杀了他。”
<br/>　　楚风眠的身形，缓缓的现身，来到了众人面前，不过他仍然留有吞天秘术，在天星河之中，吞噬这天星河的本源。
<br/>　　青梦也是现身出来，站在了楚风眠的身边，她看向这一群人，眼神也是蕴含无边杀意。
<br/>　　对付圣龙宗，可不是一个小小的执法堂，万龙堂等等，可以做到的。
<br/>　　圣龙宗从圣龙秘境之中被连连针对，其中必然是蕴含了圣龙秘境高层的意思，这些高层，都是圣龙宗的仇敌。
<br/>　　“楚风眠，青梦！你们两个人是打算以下犯上，背叛圣龙秘境吗？”
<br/>　　从人群之中，那万龙帝身形一步踏出，看向楚风眠跟青梦，怒声喝道。
<br/>　　“万龙皇已经晋升为九劫古帝，乃是我们圣龙秘境的高层，你们居然敢杀他？大逆不道！哪怕是万龙皇犯了错，也不是你们可以杀的！也要经过我们的审判才行！”
<br/>　　“你们两个私自杀了万龙皇！残杀同门，罪大恶极！现在还不速速跪下认错！不然你们圣龙宗，就等待着灭顶之灾吧！”
<br/>　　这万龙帝的话，绰绰逼人，他的身边站着的可都是圣龙秘境的高层，哪怕是察觉到了楚风眠跟青梦实力已经是今非昔比，也依然无比嚣张。
<br/>　　“楚风眠，青梦，你们今日居然敢攻打天星河，简直是欺师灭祖！龙千绝那老家伙，也保不住你们！”
<br/>　　那悟剑堂上一任堂主，罗剑古帝也是一步踏出，大声的吼道。
<br/>　　“速速跪下臣服，接受我们的审判！这是你们唯一的生路，不然就凭你们的所作所为，明日圣龙宗就要被毁灭！你们二人就是毁灭圣龙宗的罪魁祸首！”
<br/>　　圣龙秘境高层之中，想要毁灭圣龙宗的人物，不在少数，他们都不是出身自圣龙宗之中，自然不想要看着圣龙宗一直超然地位下去。
<br/>　　所以从数千年前，针对圣龙宗的行为，就不断的进行下去，压迫圣龙宗。
<br/>　　一直是等到云尊崛起，圣龙宗的地位才有所改善，可是随着云尊的消失，圣龙秘境对圣龙宗的压迫，又在继续。
<br/>　　今日楚风眠杀了万龙皇，正是给了他们一个极好的借口，正好可以趁着这个机会，灭了圣龙宗。
<br/>　　“圣龙宗的人，都是一些妖邪之辈，依我来看，在就应该灭亡了，不然不知道要诞生出多少的妖邪魔头。”
<br/>　　在高层之中，一尊老者缓缓的开口道，这老者看起来仙风道骨，倒是有着几分读书人的气质。
<br/>　　不过此人开口，声音却是无比尖锐，跟外表极为不同。
<br/>　　这老者说完，目光便是打量了一眼楚风眠跟青梦，眼神之中的贪婪之色，浮现而出，毫不掩饰。
<br/>　　“而且这些圣龙宗的妖邪，也该被关押起来，作为奴隶，为我们所用，这圣龙宗从圣龙秘境之中，不知道吸血过多少，现在正好是让他们好好的付出一下，回报圣龙秘境。”
