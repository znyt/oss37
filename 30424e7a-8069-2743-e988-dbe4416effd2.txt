<br/>				<br/>　　    杜凡见自顾走进去的男子，便也关上门跟了进去，道：“我家主子出门了，阁下可以过段时间再来找她。”<br/>　　<br/>　　    前面的易修染一听身后传来的这话，脚步一顿，回头瞥了杜凡一眼：“出门了？她不是刚回来吗？怎么又出门了？”<br/>　　<br/>　　    “因有些事情要处理，所以出去了。”杜凡说着，目光落在易修染的身上。这易修染他倒是从段夜那里听说过这人，应该是与主子认识的不错。<br/>　　<br/>　　    “她是去哪了？什么时候回来？”他找了个地方坐下，询问着。<br/>　　<br/>　　    “约莫一月左右归，至于去哪，这则无可奉告。”<br/>　　<br/>　　    “给我为杯茶。”易修染看向杜凡说着。<br/>　　<br/>　　    见此，杜凡便让人端上了茶水来，看着他在那里坐着，时而皱着眉头，时而又摇了摇头，心下有些奇怪，便问：“易公子找我家主子有事？”<br/>　　<br/>　　    “我是正好路过这里，顺路进来看看她的，能有什么事情？”易修染声音散懒的说着，一手端着茶杯，一手用茶盖轻轻的刮着，也不知在想着什么。<br/>　　<br/>　　    “对了，那个段夜呢？我听说最近她大哥惹出了不少事情，这烂摊子还没收拾呢！她也放得下心出门？”易修染边喝着茶边问着。<br/>　　<br/>　　    “易公子劳心了，我家主子把事情都交待好才出门的。”杜凡礼貌性的笑着。<br/>　　<br/>　　    “既然她不在，那我便留下等她吧！一个月要过去应该也是挺快的。”<br/>　　<br/>　　    听到这话，杜凡怔了一下：“易公子要在这里住下？”<br/>　　<br/>　　    “不行？你不是说她顶多出去一个月左右吗？难道还要我回去？”易修染瞥了他一眼，坐着不动了。<br/>　　<br/>　　    见此，杜凡笑了笑：“这倒不是，只不过，最近我们天丹楼的麻烦事比较多，我担心易公子若是住在这里我们会招呼不到。”<br/>　　<br/>　　    “无妨，我不用你们招呼。”<br/>　　<br/>　　    闻言，杜凡神色微动，看了下一副架势压根没想离开的易修染一眼，便道：“易公子请稍坐一会，我去去就来。”说着，行了一礼后便先离开。<br/>　　<br/>　　    来到后面，八名凤卫队长询问着：“那人是谁啊？”<br/>　　<br/>　　    “易修染。”杜凡看了他们一眼：“还记得当时段夜提起过的那个人吗？就是他。”<br/>　　<br/>　　    听到这话，几人相视了一眼：“段夜口中那个实力不在他之下的易修染？”<br/>　　<br/>　　    杜凡点了下头：“不错，他说是来找主子的，不过主子刚离开你们也是知道的，现在他说想在这里住下等主子回来，这事我想着找你们还有冷华商量一下。”<br/>　　<br/>　　    “这人没问题吧？他的来历我们也没查过，这样让他留下会不会不妥？”其中一人问着。<br/>　　<br/>　　    “这也是我担心的。”杜凡说着。<br/>　　<br/>　　    冷华手里拿着东西走过来，听见他们的话，便询问着：“担心什么？”<br/>　　<br/>　　    见是他，几人便将那易修染一事跟他说了一下。听到他们几人的话后，冷华往外走出几步，朝那前面角落茶桌的人看去，微顿了一下，沉思着，道：“要不这样吧！让他去凤府住。”<br/>　　<br/>
