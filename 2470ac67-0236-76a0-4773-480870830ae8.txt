　　李二的状态有些不对，早早便回去休息了，只等老头子一走，下面那些各大家族代言人立刻就恢复了本色，七嘴八舌的开始发问。
<br/>　　性子急的更是直接开口：“陛下，这无线电可是了不得的东西，只是不知道这东西是否对外销售？“
<br/>　　“对外销售？”李承乾眨眨眼睛，想了想说道：“这倒不是不可以，不过这东西过于精密，就算是朕卖出去，能用的人也不会很多，而且坏了的话也不像以前那些机器那么容易修理，如果从成本上考虑，朕觉得你们还不如自己养点信鸽来的实在。”
<br/>　　听了李承乾的话，下面一群人互相对视了一眼，若有所思的点了点头。
<br/>　　本来嘛，这刚刚研究出来的东西本身就十分昂贵，如果只是为了传递信息，根本没有必要花上百十万贯买么一个东西回去，因为这根本就不划算。
<br/>　　这就好像我们后世的卫星电话，那东西信号的确是不错，只要不是在地下，基本任何地方都可以无障碍通话。
<br/>　　可问题是那东西不仅仅是电话本身贵，服务费也贵的可以，除了那些一分钟几十万上下的大佬，普通人用那个东西就是浪费，因为普通人根本用不上那无障碍通话的功能。
<br/>　　李承乾看着众人似乎已经从刚刚的盲从中回过神，笑了笑继续说道：“不过你们可以放心，无线电台这东西经过试验之后很快就会量产，在未来的三到五年之内每一个州府都会安置上一台，到时候如果有什么消息到府衙去发信息也是一样的，收费也绝对比自己买一台要划算。”
<br/>　　听到李承乾如此说，下面众人齐齐露出笑容，纷纷称赞陛下圣明，马屁拍的‘啪啪’直响。
<br/>　　李承乾见众人都没有什么异议，便散了会议，左右也没有什么大的事情，叫众人过来也就是分享一下成功的喜悦，说完了就算了。
<br/>　　……
<br/>　　时间转眼就过去半个月，身处大食的薛仁贵也终于等到了来自长安的军列。
<br/>　　心急知道结果的老薛很着不淡定，火车刚刚停下，车长刚刚下车，他便已经出现在其面前：“车上都装了什么物资，十万副手套有没有运来？”
<br/>　　“有，有啊！就在后面第十二车皮里面。”车长被薛仁贵吓了一跳，连连点头说道。
<br/>　　可本以为薛仁贵在知道手套运达消息会很高兴，可车长却意外的发现这位将军大人似乎有些失魂落魄，两只眼睛失神的看着远方，口中喃喃：“竟然是真的，竟然是真的！”
<br/>　　什么是真的？车长并不知道，想问又有些不知道如何开口，最后只能把所有的话闷在肚子里，权当这大佬在发癔症好了。
<br/>　　可是他并不知道，薛仁贵在离开车站之后，直接便去了无线电试验场，路上胡乱找了一些藤蔓让亲卫拿着。
<br/>　　等到了试验场正门，老薛也不顾不气寒冷，三下五除二将上衣脱了个精光，又让亲卫把那些藤蔓绕到自己身上，这才大步走了进去。
<br/>　　试验场里的那些研究员事实上早就已经忘了半个月前的事情，一个两个见了老薛负荆请罪的样子，顿时目瞪口呆，那白大褂更是吓的差点没跳起来。
<br/>　　要知道，半跪在自己面前的可是统领二十万大军的大将军级人物，而他却只是一个小小的项目负责人，论品级可能最多也就是个六品上，与薛仁贵正三品的实职那差距完全可以用天地之差来形容。
<br/>　　“这位先生，半月之前仁贵无礼在先，今日特来向先生请罪！”薛仁贵见白大褂手足无措的样子心中虽然好笑，但脸上却没有露出来，只低着头沉声道歉。
<br/>　　“薛将军，使不得，万万使不得啊，您这真是折杀吾等了。”白大褂反反复复就是这么一句话，竟也不知道找衣服给老薛披上，说他读书读傻了倒也一点都不为过。
<br/>　　好在一边有人机灵，飞快的跑了过来，在他耳边小声嘀咕了几句，这才让白大褂反应过来，脱了自己的衣服给薛仁贵披上之后，这才将老薛拉了起来。
<br/>　　不过这一番折腾倒也不是没有效果，科学院的书呆子们经过这次的事情明显对外面守着他们的那些大头兵有了一些好感，见面之下也知道点头打声招呼。
<br/>　　薛仁贵对此也是喜闻乐见，与那白大褂打着哈哈聊起了关于无线电的事情：“我说老赵，你们科学院可真是太厉害了，无线电这么神奇的东西都被你们研究出来了。”
<br/>　　老赵自然就是白大褂，刚刚一番折薛仁贵已经知道了他的名字，熟悉了之后便以此来称呼他。
<br/>　　不过让薛仁贵有些意外的是，这次老赵竟然没有再吹牛、、逼，反道是苦笑着摇摇头：“薛将军过奖了，其实这其中最关键的东西都是陛下提出来的，我们科学院只是做了一些基础工作而已，要说厉害，陛下才是真的厉害！”
<br/>　　“哦，陛下？陛下还懂无线电？”薛仁贵没想到老赵竟然会有这样的回答，愣了一下才问道。
<br/>　　“可不是么，陛下懂的东西远比我们要多的多，从最开始的蒸汽机到现在的无线电，哪个不是陛下首先提出来，科学院再去研究的。而且不仅仅是这样，就连那火药，炸药，还有枪械也都是陛下首先提出来的，这个你们军方作为使用者，应该知道的比我们多。“老赵的脸上闪过无限的敬仰之色，看在薛仁贵眼中，甚至觉得那是一种狂热的崇拜。
<br/>　　不过话说回来，抛开李承乾皇帝的身份，薛仁贵了觉得他就像是一位先知，这个世界上好像没有什么事情是他不知道的。
<br/>　　想想自己当初躲在山里，吃都吃不饱，如果不是当今皇帝陛下把自己从山里挖出来，估计现在早就饿死了。
<br/>　　可如果抛开那些有的没的再想想，当初李承乾到底是怎么把自己找出来的呢？难道他真的有预知的能力，知道若干年后自己一定会成为大将军，所以才会不遗余力的找自己？
