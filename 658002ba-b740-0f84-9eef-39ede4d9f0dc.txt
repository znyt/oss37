　　“你刚才不还说，担心少阳在那边无法立足！”谢雨晴道。
<br/>　　老郭耸了耸肩，“我就是担心一下，他是我师弟，我作为家长，担心一下怎么了，不过理性的说，我们根本不用为他担心。”
<br/>　　“什么意思？”
<br/>　　“十二年蝉七姑，修罗鬼母，上古邪神，化蛇，这些哪个不是不世出的妖邪，不都被少阳给办了，我就不信民国有什么东西比修罗鬼母还厉害？就算是女魃，不也没能杀死他？”
<br/>　　雪琪眼神闪了闪，提起修罗鬼母，她又想起了叶小烁，心里一阵伤感。同时修罗鬼母那强悍的实力，也是让她心有余悸。那真的是……无比惨烈的一次战斗。
<br/>　　谢雨晴想了一下说道：“那不一样，这些毕竟是邪物，我怕他一个外来者，跟当时的那些道士和尚什么的产生矛盾，他就一个人，对方可是抱团的。”
<br/>　　“也未必是一个人，宝爷和吴嘉伟，还有军师师徒，凤兮，如果他们在一起，那就没任何问题，有这几个人陪伴少阳，就算是整个法术界，也绝不能把他们怎么样。”
<br/>　　他们之所以只担心叶少阳一个，并非是因为跟叶少阳最亲，抛开爱情这方面，四宝对于老郭等人来说，情感上也是一样的，都是可以为了对方放弃一切的这种铁杆兄弟，大家只担心叶少阳，因为这一切都是针对叶少阳的，四宝他们只是误闯进去，除了担心他们回不来，倒不会有什么别的危险。
<br/>　　周静茹沉吟道：“从这封信里看，少阳好像没跟他们一起，不然一定会写出来的。”
<br/>　　老郭想了一下说道：“现在不在一起，不代表以后不会遇上。不过就算少阳是一个人，我也相信他能够应付当下的局面。”
<br/>　　“哎呀，要是我跟师父在一起就好了！穿越啊，民国啊，想想都好刺激！”张小蕊爱冒险的本性不改。
<br/>　　老郭把叶少阳写信用的符纸拿起来研究了一下，说道：“没毛病，少阳在那暂时还挺好的。”
<br/>　　周静茹立刻问：“你怎么知道？”
<br/>　　“他以前的灵符，都是我给提供的，这几张灵符虽然过了九十年，但一看就不是我的东西。这说明他在那边找到了补给法药的地方。还有，他既然有工夫给我们写信，那说明情况肯定还不错，至少没遇到什么危险。”
<br/>　　大伙觉得他分析的有道理，也就更加放心了。
<br/>　　讨论起他怎么会遇到“大肉包子”家的老板，想到用这种方式来给他们发信，大家都感到匪夷所思，不过想来应该也是个偶然，从叶少阳字里行间，看得出他对这种传信的方式也不是很有信心，将来也不会付出代价再用这种方式来传信。
<br/>　　“除了确定了少阳的安全，我们还有一个重要的发现。”老郭说道，“之前道风推测的没错，少阳在的那个时代，跟我们这个时代是联通的，或者说，是一种顺延的关系，所以他在那个世界做出的改变，才会影响到我们这个世界。”
<br/>　　大伙思考起来，对这种事提出了很多科学推论和假设，但因为实在超出了他们的认知，最后干脆不想了。最后老郭召唤扁头从空界回来，让他去通知捉鬼联盟的成员们，告诉他们叶少阳来信这件事，对他们来说，这已经算是一个天大的喜讯了。至少大家都不用再胡思乱想，去揣摩叶少阳的安危。
<br/>　　“我们现在需要做的，就是等待少阳归来。”老郭说道，“我相信他一定能办到。”
<br/>　　大家纷纷点头。
<br/>　　谢雨晴说道：“我只是不理解，为什么徐福要把他带到民国，1922年，肯定不是随机选的时间吧？”
<br/>　　“那是当然，一定是有什么目的。”老郭沉吟起来。
<br/>　　叶小萌这时候说道：“我觉得肯定是1922年发生了什么大事，徐福希望他去解决一些事情。”
<br/>　　她的这句话，听在大伙的耳朵里，如同灵光乍现。
<br/>　　周静茹道：“直奉战争……”
<br/>　　老郭立刻摆手：“不可能，徐福不可能让他去参与这些世俗的事情，而且少阳是法师，战争的事跟他没关系，他也左右不了，肯定是跟法术界有关的事，或者是影响阴阳两界的大事……”
<br/>　　雪琪道：“法术界各大门派不是都有典籍的吗，会将过去发生过的重大事件都记录上去，可以去找一找。”
<br/>　　老郭道：“那个并不完整，因为后来宗教被打击，毁了很多东西，不过确实值得去找一找，虽然我们知道了没用，可也比不知道要强。”
<br/>　　雪琪道：“如果能找到的话，别的再说。”
<br/>　　“好，我明天就去茅山找苏钦章，让他把典籍给我看。另外……雪琪你最好能去下风之谷，找道风，把这封信带个他看，让他们几个也放心，同时一起想办法。”
<br/>　　雪琪道：“我没去过风之谷，不过我可以去找瓜瓜他们，让他们带我一起去。”
<br/>　　老郭道：“我今晚先走阴，去一趟阴司，上次去，萧郎君告诉我橙子正在聚魂，还没有复生，这次不知道怎么样了，我去探望一下，正好也把事情告诉萧郎君，让他想办法也查一下阴司的档案，看看1922年在鬼域或别的空间，发生过什么大事没有。”
<br/>　　等了不久，瓜瓜他们跟着扁头从空界回来了，十分激动地阅读叶少阳的信，大家又激动了一场，之后老郭说出了他们的结论，该走的就走，分头去办了。
<br/>　　剩下老郭等人，晚上一起吃饭，又讨论了一番，之后各自离去，等待消息。
<br/>　　老郭给苏钦章打了电话，说了叶少阳的事——之前星宿海一战，叶少阳为了不将茅山卷入其中，没有通知苏钦章知道，不过事后老郭还是将情况告诉了苏钦章，毕竟叶少阳失踪了，得跟苏钦章打个招呼，让他照应茅山上下的事情。
<br/>　　“掌教没事，那就太好了！”苏钦章听了老郭的讲述，也是抑制不住的兴奋。
<br/>　　老郭接着讲了自己要去茅山的事，苏钦章听了，自然也是一口答应。“山门里还有很多密封的卷宗，你可以过来查看，我帮你一起找。”
