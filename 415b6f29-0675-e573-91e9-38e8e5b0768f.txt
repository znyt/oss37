　　笼罩烟云遁光，苏重毫不犹豫，一头扎进蝙蝠群。
<br/>　　所过之处，所有蝙蝠尽数化作血雾肉泥。
<br/>　　“哈哈，好！”苏重心中畅快，猛然催动烟云遁光。
<br/>　　烟云遁法可不单是飞遁法门，它同时融合《流光遁发》、《腾云驾雾》、《气御阴阳》三种秘法。以剑气符为基，融入道种成就剑阵。
<br/>　　飞遁迅捷，防御强悍，更重要的是，他还有强大攻击力。一枚枚尖刺上，带着犀利无比的切割攻击。
<br/>　　他猛然发力，狠狠冲进蝙蝠群，一路冲出一条血肉之路。
<br/>　　齐昊等人目瞪口呆，曾书书大张着嘴，使劲戳身旁的张小凡。
<br/>　　“这是你师弟，他平时就这样的吗？”这手段也太血腥了些。
<br/>　　张小凡讷讷，他能说什么。他也震惊啊。平日里淡然温和的苏重，没想到一旦动起手来，竟如此霸道暴戾。
<br/>　　轰！
<br/>　　周围天光猛然一亮，众人心头一喜，终于冲出来啦！
<br/>　　几人丝毫不敢放松，继续御剑飞行，把身后蝙蝠远远甩掉，这才找了一处偏僻地方落下遁光。
<br/>　　几人一个个气喘吁吁脸色发白，刚才那一通飞，可把众人累的不轻。
<br/>　　“几位抓紧时间休息吧，我来给你们护法。”苏重站在不远处，一脸温和道。
<br/>　　相比其他人，此刻苏重气定神闲，身上衣服干净如新。和他们几个狼狈模样天差地别。
<br/>　　“师弟，你可真是让人刮目相看。”齐昊双手捧着六合镜递给苏重。心里着实不舍，这东西虽然消耗巨大，但他效果好啊！绝对是保命利器。
<br/>　　苏重摆摆手：“齐师兄先留着吧，我有法子自保。还需你照顾小凡他们。等回到青云，你再还给我好了。”
<br/>　　有烟云遁法在手，苏重可攻可守还可以逃跑，用不到六和镜。
<br/>　　“多谢师弟。”齐昊道谢。然后几人赶紧盘坐调息，刚才一阵猛冲，消耗巨大。
<br/>　　见众人闭目修炼，苏重深吸一口气，闻着山间腐烂与清新混杂草木气息，脸上露出一个愉悦笑容。
<br/>　　到如今他终于验证心中猜测：“《庚金剑气》果然是魔门法诀。”
<br/>　　他越是靠近滴血洞，修炼速度竟越来越快。只是赶路途中，他就抽空凝聚一百多枚剑气。
<br/>　　到了这里，几乎呼吸之间，凶戾之气就被道种吸收，帮助苏重凝聚剑气。
<br/>　　“只要时间充足，我的剑气符很快就能恢复一千大关！”而随着时间推移，他的剑气符将会越来越多。
<br/>　　苏重也没想到，这种穷山恶水之地竟是他的修炼宝地！
<br/>　　正琢磨着如何提高修为，心头突然升起一种被人窥视之感。
<br/>　　苏重感知向来敏锐，几乎毫不犹豫，一道剑气直接飞射视线方向。
<br/>　　砰！
<br/>　　一声闷响，剑气炸开，周围树木瞬间倒伏，露出藏在密林之内两道身影。
<br/>　　两人一男一女，男的英俊女的秀媚，气质出众着实不凡。
<br/>　　“好狠辣的小子，师兄，这必定是魔门余孽。”女子一开口就扣了一顶大帽子。显然嫉恨苏重刚才突袭。
<br/>　　苏重淡然瞥了对方一眼，右手举起，五根手指顶端顿时出现五道剑气。他也不说话，就这么遥遥锁定对方。
<br/>　　那男子脸色顿时难看无比。看了眼苏重身后几人，尤其在陆雪琪身上扫视一眼，眼中带着丝丝异样。最后才看着苏重拱手道：“在下焚香谷李洵，这是我师妹燕虹，几位可是青云门下。”
<br/>　　“师弟不得无礼，这是焚香谷高足。”齐昊连忙起身。他可不敢让自家师弟乱来。
<br/>　　就刚才那直接冲入蝙蝠群的狠辣摸样，他真怕这位师弟一言不合就打起来。关键是苏重剑气犀利，真要有个损伤，可就不好交代了。
<br/>　　正说着话，天空飞来两道黄色遁光，显出身形却是两个光头和尚。一个魁梧雄壮好似怒目金刚，一个却清秀和善，满面慈悲。
<br/>　　“在下天音寺法相，这是我师弟法善，诸位是青云门下？”法相看着已经站起来的齐昊道。
<br/>　　几人相互见礼，然后各自说出来意，竟都是在探查滴血洞异常。天音寺的和尚已经和焚香谷众人见过面。几天前也注意到蝙蝠群，不过巧妙躲了过去。
<br/>　　苏重他们倒霉，一头扎进蝙蝠堆里，搞得各个狼狈不堪一身血污。苏重一身整洁，反而更加有种鹤立鸡群的意思。
<br/>　　听着几人寒暄，商讨探查计划，苏重有些不耐烦。他能感觉到周围戾气浓度，越深入滴血洞就越高。
<br/>　　“与其在这里和这些人扯皮，不如钻进深山里修炼！”天书第一卷就在眼前，他需要提高实力保证万无一失。
<br/>　　“你们在这里休整吧，我先去了。”不等众人反应，苏重砰的一声飞上天空，划过一道优美弧线，眨眼间消失不见。
<br/>　　独自一人，苏重立刻没了约束，遁光催发极致，天空中只留下一道细细白线。
<br/>　　砰！一阵尘土飞扬，苏重收敛烟云遁光，打量眼前漆黑洞窟。
<br/>　　洞口高十多米，内部黑暗无光，幽深不可测。隐隐有腥臭气息传出。
<br/>　　“这里大概就是万蝠古窟了。”对照脑海中记忆的地图，苏重确定自己找对了地方。
<br/>　　放开感知，浓郁凶戾气息盘桓洞内。看着怪石嶙峋黑暗诡异的黑洞，苏重满脸感慨：“真是个宝地啊！”
<br/>　　别人会觉得这里是穷山恶水，可这种险地，最适合苏重修炼。别人避之不及，他却高兴不已。
<br/>　　“洞内四通八达，找魔门余孽不容易，可如果我藏在里面，他们找我也不容易。”他决定找个地方先修炼一波再说。
<br/>　　……
<br/>　　望着消失在天边的白色遁光，李洵、法相二人瞳孔一缩，对视一眼都看到彼此眼中的惊讶。苏重的遁光太快了！
<br/>　　“刚才这位，不知是青云哪一脉的高徒？”法相好奇问道。
<br/>　　“他是大竹峰的田师叔的关门弟子李尔旦，天赋卓绝，是此次七脉会武的魁首。”齐昊无奈。他根本就没办法指挥苏重。
<br/>　　一路走来，齐昊已经看出来了，这位年纪最小的师弟，有着自己的行事准则。他想干涉根本不可能。
<br/>　　“自己一人就敢闯进去，不知是真有本事还是年少无知。”燕虹冷笑道。
<br/>　　几个青云弟子脸色都不好看，眼看就要争吵起来，法相赶紧出生打圆场：“李师弟艺高人大胆，我们可不能如此鲁莽，几位青云师弟师妹显然消耗不小，贸然冲进去，恐怕会出危险。”
<br/>　　齐昊心里担忧，即便苏重是擅自行动，可真要出了问题，也是他看护不理的责任。
<br/>　　但身后几个师弟师妹确实消耗不小，自己刚才法力也没有完全恢复，他有些犹豫不决。
<br/>　　李洵扫了眼宛若冰莲花一般的陆雪琪，注意到其苍白脸色，不由开口道：“魔窟内凶险万分，谁都不知道具体情况。我看还是休息一晚，养精蓄锐，明日我们再探查也不迟。”
<br/>　　至于苏重，他巴不得出事才好，刚才苏重的剑气，可差一点儿就砸在他脸上了！
