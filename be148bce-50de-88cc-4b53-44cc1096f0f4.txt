　　“哈哈哈，小子，不用白费力气了！这祭坛乃是我地魔门的无上老祖炼化的，谁也无法打破！”
<br/>　　暗古魔尊大笑道。
<br/>　　“你还是乖乖受死吧！”
<br/>　　“什么杀神虚影？统统给我破！”
<br/>　　说着，暗古魔尊的一掌之力便是更加强大，直接压迫下来，将那远古杀神，都给硬生生的压迫而散。
<br/>　　“在坚固的大阵又如何？我倒是要看看你，能不能抗下极道仙兵的力量！”
<br/>　　在楚风眠的手心中，一尊古老残破的瓦罐，便是出现。
<br/>　　一瞬之间，闪耀无比的光芒，便是从这古老的瓦罐之上，爆发而出，充斥这整个洞窟之中。
<br/>　　随之出现的，便是浑厚到极点的力量，这力量的浑厚，超越一切。
<br/>　　“人皇之道！教化众生！”
<br/>　　人皇罐的力量，在楚风眠的催动之下，全然爆发而出了。
<br/>　　在这黑暗的洞窟之中，人皇罐的光芒，便是如同炙热的太阳一般，漫天魔意，在这力量之下，尽数毁灭。
<br/>　　那暗古魔尊的一掌，都在这力量之下，灰飞烟灭。
<br/>　　“这是什么力量？超越了天级灵器的力量，极道仙兵？这等传说中的东西，怎么会在这小子的手中？”
<br/>　　暗古魔尊看着这一幕，眼睛都要瞪大的掉了出来。
<br/>　　这可是极道仙兵，传说中的东西，如今居然出现了，还在楚风眠的手中，这让他如何相信。
<br/>　　但是如今他再不相信，这人皇罐的力量，他也是足以清晰感觉的到的。
<br/>　　在这一股力量下，他若是现在不抵抗，只有一死，暗古魔尊连连后退，妄想抵挡人皇罐的力量。
<br/>　　“想逃？在人皇罐的力量之下，无人能逃！”
<br/>　　楚风眠冷哼一声，催动这其中的力量，顿时无数宏大灵力，便是将这洞窟，彻底封锁，令暗古魔尊无处可逃。
<br/>　　不过现在楚风眠的第一对手，可不是暗古魔尊，他将大部分的力量，都给用在了对付那祭坛的上面。
<br/>　　只要毁了这祭坛，没有万古不败真魔体的暗古魔尊，在楚风眠的面前，便是蝼蚁都不如。
<br/>　　“人皇之矛！给我凝！”
<br/>　　楚风眠大喝一声，人皇罐上爆发出的无尽宏大灵力，在楚风眠的手中，猛然凝聚，化为了一根古朴长矛。
<br/>　　这便是传说中的人皇之矛！继承这远古人皇的宏大力量，如今人皇罐的碎片多了一部分，楚风眠便是足以在其中，凝聚出真正的人皇之矛来。
<br/>　　虽然只是凝聚出的人皇之矛，却也拥有超越天级灵器的威力。
<br/>　　这人皇之矛，被楚风眠握在手中，看起来如今楚风眠，便是继承了远古人皇的力量一般。
<br/>　　“刺！”
<br/>　　楚风眠手持人皇之矛，猛然冲着那祭坛刺了过去。
<br/>　　这祭坛觉察到危险，再一次的凝聚出了无数阵法来，妄图抵挡。
<br/>　　但是这小小阵法，哪里可以抵挡得住人皇之矛的锋利。
<br/>　　“轰隆！”
<br/>　　楚风眠手中的人皇之矛，一下刺入，其中一股爆发力，便是从楚风眠的手中震荡起来。
<br/>　　这人皇之矛的力量，实在是太过强大，令楚风眠一时间似乎都有些无法掌控。
<br/>　　人皇之矛，一下子刺破祭坛上的大阵，直接便是刺入到了祭坛之中去。
<br/>　　“咔嚓！”一声。
<br/>　　在这祭坛之上，便是一股灵力，猛然刺穿了过去，直接出现无数裂痕，随后猛然破碎。
<br/>　　在这祭坛之中，突然一道光芒，随着祭坛的破碎，居然是直接飞了出来。
<br/>　　楚风眠一眼望去，穿破无数光芒，便是看出，这其中，居然是一枚头颅大小的巨大内丹。
<br/>　　“龙丹？”
<br/>　　楚风眠的眼神，瞬间一亮，如此庞大的祭坛，其中定然是有着一件东西来支持其中的力量。
<br/>　　没想到这在祭坛之中的，居然是一枚龙丹。
<br/>　　这龙丹，已经是通体黑色，显然已经是受到了魔意的影响，但是在楚风眠看来，依然是一件无上至宝。
<br/>　　“好东西！这龙丹虽然受到魔意影响，但是其中力量蕴含却是极多，以我的战龙血脉，可以将其中的魔意，全部化解！”
<br/>　　“这一枚龙丹，足以助我突破，令我踏入神力境的成功，又大进一步。”
<br/>　　楚风眠狂笑一声，直接便是将这龙丹抓了过来，一下子镇压在了空戒之中。
<br/>　　“小子，你敢毁了祭坛！我们地魔门的强者不会放过你的！会永生永世的追杀你的！你就等着我们地魔门的追杀吧！”
<br/>　　暗古魔尊疯狂大吼道，但是他的声音中，却是蕴含颤抖。
<br/>　　他的手心一动，一道玉符，突然打出，似乎是想要通知地魔门的强者来。
<br/>　　“想叫人前来？”
<br/>　　“你以为你能将这玉符打出去？”
<br/>　　楚风眠冷哼一声，大手一挥，这玉符猛然被楚风眠封锁，被楚风眠手一捏，直接捏碎。
<br/>　　“暗古魔尊，就算是地魔门的人追杀我又如何？今日你的死亡，便是注定的！没有魔意，我看你的万古不败真魔体还能有什么用！”
<br/>　　楚风眠冷哼一声，猛然一剑，便是冲着暗古魔尊斩了过去。
<br/>　　“嘶！”
<br/>　　这一剑，快到极致，猛然刺穿了暗古魔尊的胸口。
<br/>　　同时这一剑中的剑锋，也从暗古魔尊的胸口中爆发而出，猛然之间，他的胸口都直接破开了一个大洞，五脏六腑，全部毁灭。

<br/>　　暗古魔尊疯狂怒喝这，他的身躯，突然层层炸裂开来。
<br/>　　“真魔解体！”
<br/>　　一股股魔意，从暗古魔尊的身上涌现而出，将他的身躯，都给直接魔化，其中的灵力，更是变的疯狂。
<br/>　　“想要舍弃身躯？魂魄遁逃？”
<br/>　　楚风眠看着暗古魔尊的举动，自然清楚他到底想要做是什么。
<br/>　　这真魔解体，也是曾经九魔城的秘术之一，足以舍弃身躯，将力量全部集中到魂魄之中，魂魄遁逃。
<br/>　　只要是随意在夺舍一具身躯，这暗古魔尊，便是只需要休息一段世间，就可以重新复活。
<br/>　　但是如今在楚风眠的面前，这暗古魔尊想要逃，简直天真无比。
