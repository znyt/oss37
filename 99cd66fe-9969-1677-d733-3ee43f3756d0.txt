　　气丹已成，但萧逸并没有停止修炼。
<br/>　　“接下来，该炼化主脉了。”萧逸自语一声，再次开始了修炼。
<br/>　　凡境武者突破后天境，之所以要凝聚气丹。其中一个原因是气丹比丹田更加坚固、强大。
<br/>　　同时，气丹根据自己的武魂属性而成，能更好地配合武魂发挥出更强战力。
<br/>　　而第二个原因，便是为了炼化主脉。
<br/>　　人的全身，有无数经脉，在这数之不尽，或大或小的经脉中，又有九条连接气丹的最为重要的经脉，称之为主脉。
<br/>　　只可惜，九条主脉，生而堵塞，武者根本无法利用。
<br/>　　后天境武者的修炼，便是要将九条主脉重新炼化，使其通畅，为己所用。
<br/>　　每炼化一条主脉，对武者的增幅与好处都是巨大的，且每一条，等同于后天境武者的一重修为。
<br/>　　当九条主脉完全炼化时，便是后天九重修为，强大至极。
<br/>　　而炼化主脉，需要快速调动体内真气，且以极强的力度冲击堵塞的主脉。
<br/>　　由于主脉堵塞的程度非常大，一切灵气和真气都难以通过。所以，这所需的速度和力度，都远不是丹田所能够给予的，只有气丹能够做到。
<br/>　　这也是为什么凡境武者突破后天境时，必须先凝聚气丹。
<br/>　　而这一过程，又称之为凝丹化脉。
<br/>　　稍作一番准备后，萧逸立马开始了炼化第一条主脉。
<br/>　　在冰鸾剑武魂的帮助下，天地灵气澎湃地涌入体内。
<br/>　　一部分灵气用于包裹主脉，温养主脉，免得冲击时主脉受损。
<br/>　　剩下的部分则全部用于冲击。
<br/>　　气丹调动灵气的速度和力度，比丹田强了五倍以上。
<br/>　　而冰火气大，则强了十倍都不止。
<br/>　　萧逸明显可以感觉到，涌入体内的天地灵气转化为真气后，正以一种可怕的速度在快速调动着。
<br/>　　在自己的控制下，这股真气如洪水猛兽般，冲向了如干涸土地一般的主脉中。
<br/>　　洪水之下，干涸化为了润泽，杂质被一冲而散。
<br/>　　不知过去了多久，当这所有的杂质完全被冲散时，原本堵塞的主脉，变得畅通无阻，脉络坚韧。
<br/>　　萧逸微微睁开了眼睛，发现自己身上，竟出现了一堆粘稠的污秽。
<br/>　　这些污秽粘稠而腥臭，让人一阵恶心。
<br/>　　萧逸皱了皱眉头，但没有太过在意。
<br/>　　他知道，这些就是堵塞主脉的杂质，主脉被炼化畅通后，杂质通过自己全身的毛孔，排到体外。
<br/>　　“第一条主脉已炼化，如今我已是后天一重武者。”萧逸自语了一声，心中对自己的成就感到满意。
<br/>　　只有踏入后天境，才算真正踏入武道修炼的门槛。
<br/>　　后天境，已经算非常不错的战力了。这代表着，自己在这个世界的自保之力，又多了一分。
<br/>　　“喝。”萧逸已经迫不及待想试试自己的实力，猛地打出一拳。
<br/>　　那全力轰出的一拳，竟产生一道破空之声，前方的一张实木椅子瞬间化为齑粉。
<br/>　　“好强。”萧逸倒吸一口凉气。
<br/>　　这还是只是凌空轰出的威力，若真切打在敌人身上，其威力必然更强。
<br/>　　萧逸盘膝坐下，继续修炼。他刚刚突破后天一重，必须稳固一下修为。
<br/>　　此时，天地灵气入腹，转化为真气后，不再是储存于丹田，而是储存于主脉。
<br/>　　凡境武者，真气储存于丹田，要调动时，真气分散到无数经脉，最后才传递到武者的四肢。
<br/>　　后天武者，真气储存于主脉，要调动时，如臂指使。
<br/>　　如果将真气比作水，那么丹田就是水泵，而主脉就是一条通畅的水管。
<br/>　　很显然，有水管之后，喷射出的水会更具爆发力，速度也更快。
<br/>　　这也是后天境武者远强于凡境武者的原因。
<br/>　　半个小时后，已经稳固修为的萧逸站了起来，结束了修炼。
<br/>　　第一条主脉由于气丹刚成，所以很容易炼化。而剩下的八条主脉，将会一条比一条难炼化，需要的灵气和时间都极多。
<br/>　　毫无疑问，后天境的修为提升，将会比凡境困难得多。
<br/>　　资质愚钝者，可能数年时间乃至更久都无法炼化一条主脉。
<br/>　　萧逸也不急，武道修炼，必须循序渐进，不得操之过急。
<br/>　　萧逸记得，前世曾经有一个武学前辈说过，“人体，本身就是一个有待发掘的宝藏。这个宝藏，无穷无尽，无法挖掘完。如果有人能挖掘完，那么这就不是人，而是神。”
<br/>　　这个世界的武道修炼，本身就是一个完整的体系。
<br/>　　凡境炼体，后天境炼脉，乃至于之后更高深的境界，都是在将武者的身体变得更为强大，从而拥有莫大的力量。
<br/>　　那些动辄开山裂石，翻江倒海的强者；那些可以飞天遁地，乃至传说中遨游太虚的武道至尊。
<br/>　　“这个世界，越来越精彩了。”萧逸嘴角咧过一道期待与自信的弧度。
<br/>　　如今的他，如果再次对上萧若狂，就算不使用升龙或者任何一式形意五绝，都能轻松碾压萧若狂。
<br/>　　同为后天一重，萧逸的冰火气丹，能够让主脉发挥更强的爆发力。
<br/>　　就算只是单纯的普通攻击，也足以在后天一重中再无敌手，乃至越级战斗。
<br/>　　当然，这也是因为双生武魂的存在，萧逸才凝聚出了强大的冰火气丹。
<br/>　　推开房门，萧逸看着那和煦的阳光，暗暗猜想着自己到底修炼了多久。按他的估计，从凝聚气丹开始到现在，应该有两天时间了吧。
<br/>　　这时，不远处传来依依的声音。
<br/>　　“少爷，您终于出关了。”
<br/>　　萧逸见依依神色着急，不禁问道，“怎么，发生什么事了吗？”
<br/>　　依依回答道，“三长老找您。”
<br/>　　“哦？有什么事吗？”
<br/>　　依依摇了摇头，道，“三长老没说，不过应该是关于慕容家的。两天前，慕容家的人来找您，不过您闭关了，萧家的人大闹了一番后就离开了。”
<br/>　　“找我，还大闹了一番？”萧逸皱起了眉头。
<br/>　　“具体怎么回事我也不清楚。”依依回答道。
<br/>　　“我去找三长老问清楚吧。”萧逸道。
<br/>　　“咦，少爷你身上怎么臭臭的？”依依忽然皱眉捏住了自己的鼻子。
<br/>　　萧逸注意到自己身上粘稠的污秽，笑了笑，道，“怎么，嫌弃我脏了？”
<br/>　　依依连忙道，“不是，只是觉得少爷您这样会不舒服，我马上去给您准备热水清洗一番。”
<br/>　　“嗯。”萧逸点点头。
<br/>　　依依转身去准备热水，但萧逸明显能听到依依在喃喃自语，“没想到少爷两天不洗澡就脏成这样了，看来我要更好地照顾少爷了.....”
<br/>　　萧逸顿时满头黑线。
<br/>　　.....
<br/>　　近两天，紫云城内流传着一则谣言。萧家少家主萧逸，因被慕容家千金慕容娇儿退婚，恼羞成怒，竟不择手段设计玷污了娇儿小姐。
<br/>　　谣言一出，顿时满城哗然。
<br/>　　而作为当事人的萧逸，居然没有出来作任何解释。而另一个当事人慕容娇儿，同样没有否认。
<br/>　　特别是在慕容家带人大闹萧家一番后，这个谣言，几乎被坐实。
<br/>　　三长老府邸中，萧逸从三长老口中得知此事，也吃了一惊。
<br/>　　“此事你怎么看？”出言询问的是大长老。
<br/>　　让萧逸惊讶的是，大长老和二长老竟也在此。印象中，这两位老人都是大忙人，一般极少能见到。
<br/>　　“什么怎么看？”萧逸反问一声。
<br/>　　大长老脸色淡然，道，“你可知两天前是谁带人来萧家闹事？乃是慕容家的大公子，慕容娇儿的亲哥哥，慕容千军亲自来找你算账。”
<br/>　　大长老忽然脸色肃然，道，“慕容千军说了，他只有一个条件，将你交出去，由他们慕容家发落。如果我们不交的话，你有种就别走出萧家一步，否则，他会杀了你。”
<br/>　　“呵。”萧逸不在意地笑笑，“想杀我的人很多，只不过，从来没有人成功过。”
