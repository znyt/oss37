　　当夜邪与叶梵等人醒来后，发现自己等人发生了翻天覆地的变化，大多数人从朝气蓬勃的青年变成了暮气的老爷爷，老奶奶，有两人变成了中年大叔与四十岁婶婶，两人模样不变，正是夜邪与庞博，一人变成了小正太，是叶梵。
<br/>　　几人又是一阵惊恐与绝望，按原本的命运轨迹，之后几人在一个名为薇薇的女孩的引导下，拜入修仙门派，正式踏入修行之路！
<br/>　　与原定轨迹不同的，只是多了个夜邪而已。
<br/>　　而在众人都不知道的情况下，一个陌生人，踏入荒古禁地最为神秘的深渊！
<br/>　　混沌道君本来是在青铜古棺中的小仙界中参悟此界天地规则的，但此刻他却是出来了。因为此界的特殊性，世界不够完美，而导致此界的天地规则十分容易参悟，仅仅只是短短数日的时间，他便领悟了数千种大道，这种天大的便宜事，他做梦都没有想到。但这的确是事实。
<br/>　　只是短短几日的功夫，他便将小仙界中的规则参悟得七七八八了，一身实力在不断上涨，这简直不可思议，要知道，到了他这个境界，每提升一点实力，都是千难万难的。数千上万年的刻苦修炼，还抵不上这个世界的短短几天的参悟，这是该讽刺还是该庆幸？
<br/>　　当然，这只是他刚来到这个世界的适应阶段，再加上他修为高等种种原因，才有了这等修炼速度。这就如同一块海绵从干燥的空气中到了水中，会拼命的吸收水份，等它吸收到一定程度后，它便会饱和，不再吸收。混沌道君此刻就如同刚入水的海绵，在疯狂的吸收着新世界的道则，等过段时间，便会恢复正常速度。
<br/>　　但，就算如此，他也很满意了，毕竟，小仙界中的规则，蕴含着仙古、乱古等时期的道，最重要的是，里面有部分荒天帝的大道，毕竟这个小仙界便是荒天帝布置的。
<br/>　　能够有幸领悟到荒天帝的大道，本身就是一种幸运了，他还有什么不满足的？
<br/>　　此刻，混沌道君便是想要见识一下荒古禁地中的这位古今才情第一，精才绝艳的狠人大帝！
<br/>　　狠人大帝，一生傲世古今，古今最强最神秘的大帝之一，活的时间也算得上是最长的几人之一，乃是遮天界中仅有的几个值得混沌道君看得上的人物。
<br/>　　遮天界中，从不缺少天骄。古往今来，更是不知出现过多少大帝至尊。
<br/>　　天道无始亦无终，万法尽头见虚空，
<br/>　　一株青莲惊万古，九圣凌霄破苍穹。
<br/>　　太阴太阳孰弱强，混化太极便称王；
<br/>　　斗战压塌九天界，不败不灭不死皇。
<br/>　　斩我明道唯乱古，太皇龙剑敌万夫，
<br/>　　阿弥陀佛慧慈苦，道胎天证西皇出。
<br/>　　飞升羽化长空耀，九黎王图显神通。
<br/>　　狠人横扫三千界，轮回尽头见老疯。
<br/>　　白衣神王惊艳久，盖世九幽傲骨风。
<br/>　　红颜青丝怜紫月，凡尘一叶遮天中。
<br/>　　遮天界中才算得上是天骄如雨，哪一个成道的大帝，不是踏无尽天骄尸骸上位的。能够在历史传说中留下浓厚的一笔，又有哪一个不是真正的天才中的天才。而这其中，能够被混沌道君看得上的，只有那么几个。
<br/>　　狠人大帝算一个，无始大帝也算一个，帝尊算一个，虚空大帝算一个，青帝算一个，未来的叶天帝叶也算一个，至于其他的，虽然也很惊艳，道君也没有小看他们，但只是相对来说，这几人道君看得上而已。
<br/>　　其中，混沌道君最佩服的，便是……这位狠人大帝！
<br/>　　作为史上最惊才绝艳的大帝，古来最有才情的女子！狠人大帝傲视古今，她与天争、与地争、与己争，并非天赋异禀，但却靠一己之力，斩尽诸王，独立九天上，神灵都不能挡其路，以一介不如凡体的体质最终走到绝巅，成为古今最为强大的人物之一。
<br/>　　体质对她来说算不得什么，什么血脉来了，她皆可杀之！
<br/>　　古来有几人敢号称天帝！她就是其中之一，南岭天帝名动天下！
<br/>　　狠人大帝一生与天争高，有惊艳万古的才情，可以杀九天外的神灵，屹立九天之上，让众神战栗，然而却不为长生，只为在那红尘中等“你”归来。
<br/>　　狠人大帝，一位强势的女帝，古来号称绝艳第一。
<br/>　　她可以斩星辰、摘日月，杀遍人间，世上独尊，为古来最不可琢磨的女子，一个狠字震动万古。
<br/>　　她看起来绝代风华，丰姿绝世，可是一旦凌厉起来，这天地都将被毁灭，古来无人可挡。
<br/>　　狠人不同于其他大帝，最初号称吞天大帝，整片大地都因她而战栗，当年，吞天大帝血洗天下，后来纵有不世天功于人族，也并未被传诵，被人为的磨灭掉了功绩。
<br/>　　狠人大帝逆天而行，站在了人族所能走到的最绝巅上，傲视古皇，睥睨后帝，惊艳万古。
<br/>　　狠人是谁？万古绝艳，比神话时代中的人还要强，力压诸神。
<br/>　　她的一生充满了传奇，还有许多事情不为人知，辉煌到极致。
<br/>　　未来黑暗****最后的危急时刻，荒古禁地中的狠人大帝觉醒，打遍天下无敌手，强势击杀极尽升华的弃天至尊，逼得石皇不得不果断回不死山，平息黑暗****。叶梵平神墟时狠人大帝屹立在荒古禁区上方悬着的青铜仙殿之上，傲古凌今，随时准备祭出一击，威慑三大禁区，最终三大禁区无人出世，神墟成为过去。
<br/>　　狠人大帝在这么一个长生物质匮乏的世界中，更是逆活了数世，每一世都是逆天人物！
<br/>　　第一世，为吞天大帝，晚年创《不灭天功》，蜕去魔胎，如蠕化蝶一样，从老体中破茧而出，蜕出一个神胎，斩断过去因果，化生混沌体。
<br/>　　第二世，演化出混沌体，打破天地限制，将旧体炼成吞天魔罐。
<br/>　　第三世，以不死神药续命，再获一世寿元，神胎恢复青春年华，可并未衍生新的神胎。
<br/>　　第四世，在万龙巢留下半截尸身。
<br/>　　第五世，不知道如何活出第五世的荒主，也正是如今的荒古禁地之主。
<br/>　　未来在荒古禁地中修炼更是不知道蜕变了几世，还是一直不变，这一切都未知。
<br/>　　狠人一生，更是创出无数神秘强大的秘术仙功。吞遍天下造化的《吞天魔功》，肉身永恒不灭的《不灭天功》，度尽时世间诸王的《度神诀》，还有专门应付九秘的九种神秘秘术《飞仙诀》、《一念花开，君临天下》、《唯我独尊圣术》、《斩我明道诀》等，堪称恐怖！
<br/>　　狠人的才情，由此可见！
<br/>　　此刻，混沌道君便来到了这令人闻风丧胆的荒古禁地深渊，见一见这位傲绝古今的狠人大帝！
