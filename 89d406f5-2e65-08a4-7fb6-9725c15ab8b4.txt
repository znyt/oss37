　　第623章
<br/>　　程大雷三人从官兵的包围圈中冲出来，一路向西走，待到正午时，已经将身后的追兵摆脱。
<br/>　　到这个时候，程大雷才算是松口气。他们奔波了一夜，现在都是筋疲力尽。
<br/>　　“大当家，咱们找个地方歇歇脚吧？”白元飞道。
<br/>　　程大雷点点头，他如今也是累得够呛，人吃得消，座下马未必能吃得消。
<br/>　　“估计追兵也不会赶过来，咱们往前走走，看哪里能够歇歇。”
<br/>　　最后三人寻到一处荒废的镇子。兵荒马乱的世道，镇子里的人死的死，走得走，如今帝国这样荒废破败的镇子有很多。
<br/>　　小白狼去野外打了两只野鸡，白元飞从井中打水清理灶台，三人一起动手，折腾一顿午饭吃。
<br/>　　到这时候，程大雷才向白元飞问起山寨的情况。昨晚匆匆遇上，又遇到一连串事，有很多话程大雷还没来得及问。
<br/>　　据白元飞所说，现在山寨一切正常，倒没有什么大事发生。不过，帝国最近可是不太平，各路诸侯之间磨刀霍霍，许多已动起手来。还有一些占山为王的巨贼大盗，扯起反旗造反。
<br/>　　程大雷点点头，这种事也不是想不到。乱世天下，许多人都不安分了，有些人是真活不下去，想要豁出去命不要拼一碗饱饭吃，当然也有些人是存着侥幸心理，想着能在这乱世中浑水摸鱼。
<br/>　　所以，如莫将难聚义这种事，在帝国很多地方都有发生。倒也不能说这些人自不量力，乱世天下许多事情说不好，未必在这波人中，就没有未来的帝国九五之尊。
<br/>　　“那个……徐军师如何？”
<br/>　　“徐军师一直都在闭关，就是不知道有没有什么收获。”
<br/>　　程大雷令黑衣老尼守着徐神机，就是担心自己不在的这段日子，徐神机不安生搞出什么幺蛾子来。其实听白元飞说蛤蟆城平安无事，程大雷就知道徐神机该是很安生。
<br/>　　不然的话，蛤蟆城也不可能一直正常。
<br/>　　三人在破落的镇子里歇息了一夜，养足精神后，第二天继续上路。一路无话，穿过宋伯康控制的区域后，在第四天的黄昏抵达蛤蟆城。
<br/>　　半路上已经遇见蛤蟆城的探子，程大雷令他们先回城传递消息，所以在程大雷没回城之前，刘悲一行人已经提前知道消息。
<br/>　　城门洞开，刘悲、关鱼、阿喜、高飞豹、秦蛮等等诸人都在城门口迎接。
<br/>　　远远就看到一头黑牛托着程大雷缓缓而来。
<br/>　　“来了，来了。”张肥先忍不住喊出声来。
<br/>　　等程大雷到城门前时，诸多人一起拱手，口中道：“大当家。”
<br/>　　程大雷一跃从牛背上跳下来，摆摆手道：“好啦，好啦，又没干什么大事，不至于如此。”
<br/>　　“听说大当家一把火将书院的藏书楼烧掉了，现在天下读书人都指着大当家骂娘，这还不是大事。”关鱼感慨道。
<br/>　　“胡说八道，又不是我烧的。”程大雷看了关鱼一眼，忽地吃惊的发现一个事实：关鱼竟然突破了，现在已经是绝世等级。
<br/>　　“你的身体好了？”
<br/>　　关鱼在鬼门关前走了一圈，现在想来还是心有余悸：“对亏了华大夫，与我刮骨放毒之后，身体才渐渐好转。”
<br/>　　程大雷还是琢磨不明白关鱼怎么突然突破的，关鱼这句话倒是令他想起华正则和刘发财。
<br/>　　“走喽，走喽，回城再说话。”
<br/>　　程大雷来到城主府，先由和珅与刘悲介绍了最近山寨的情况。倒也没有什么大事发生，一切都照常运转。
<br/>　　这时候，刘发财与华正则来到城主府，未见到他的面，程大雷就先听到一阵哈哈的笑声。
<br/>　　“大当家，你总算是回来了，可是让老刘我好想。”
<br/>　　刘发财迈着步子走进大厅，身后跟着华正则。来蛤蟆城的路上，他还忐忑不安，真正到了，方才知道蛤蟆城在凉州又如此大的势力。自给自足，安居乐业，如今在整个帝国，这样的地方都不好找了。这里简直就是一处世外桃源。
<br/>　　程大雷坐直身体，道：“坐，快坐，最近怎样，这里没人亏待你吧。”
<br/>　　“怎么可能，大家都是自家兄弟么。”刘发财顿了顿道：“大当家，你给我派个差事吧，我总要做些什么。”
<br/>　　刘发财一路从东海过来，在蛤蟆城自然不会有任何人亏待他，但也不会有人格外热情。如刘悲、关鱼这些人，都不是太好接触的人。所以他这些日子也没交到什么朋友，终日无所事事也有些无聊。
<br/>　　“这个……”程大雷顿了顿，让刘发财做什么，他倒是还没想好：“你先歇一歇，等回头我定给你安排一个重要差事。”
<br/>　　接着，程大雷面向华正则：“华大夫，今后可愿留在蛤蟆寨？”
<br/>　　“程当家……”华正则顿了顿，道：“我答应你的事已经做到，我还是想回到东海家中。”
<br/>　　听他对程大雷的称呼，已经可以猜到他的心思。很明显，他并不认为自己是蛤蟆城的人。
<br/>　　华正则胸无大志，虽然医术惊人，有妙手回春之能，却没有什么远大理想。无非想安生生做个大夫，平安度过此生。而成为一个山贼，毕竟为正人君子所不取。
<br/>　　虽然程大雷很希望华正则留下，但对于此，也没有什么法子。
<br/>　　“华大夫暂且留几天，等回头方便了，我送华大夫回去。”
<br/>　　程大雷施展拖字诀，心中还是想着怎么把华正则留下来。
<br/>　　几人在一起说着话，大厅内的人越来越少，只剩下刘发财还在与程大雷扯些闲话。
<br/>　　“咳咳。”
<br/>　　刘悲轻咳一声：“你来，我正好有件事与你商量。”
<br/>　　“与我商量什么……”刘发财一头雾水，在看到刘悲冲自己不停眨眼时，忽地恍然大悟。
<br/>　　程大雷去了好些日子，家中还有娇妻美妾，说不得他还有什么事要做，怎么能把这奢侈的时间浪费在自己身上。
<br/>　　刘发财连声答应，与刘悲一起离开。大厅瞬间变得空旷起来，程大雷来到后院，见过了苏樱与樊梨花，今晚在苏樱房中歇了。
<br/>(本章完)
