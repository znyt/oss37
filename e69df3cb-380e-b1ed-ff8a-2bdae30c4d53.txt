　　“不可能吧？那天我们商量这件事情的时候，根本没有外人，而且这段时间以来我们的人也没有时间与外界接触，消息是如何泄露的？”倭人青年狐疑的看了自己老爹一眼，有些不大确定的问道。
<br/>　　“现在的问题已经不在于消息是如何泄露的，而是如果真的泄露了我们应该怎么办！以李承乾的手段来说，如果他知道了我们的打算，估计这一次绝不会善罢干休。”秦吉胜关上窗子，将外面的一切隔绝在外。
<br/>　　“那怎么办？”倭人青年的眼神有些慌乱，在大唐的次出事给他留下了一定的心理阴影，做起事情总是会犹豫不决。
<br/>　　不过就在倭人青年犹豫不定的时候，秦吉胜一句话却让他大惊失色：“我打算让你先找个机会离开。”
<br/>　　“什么？父亲，你，你这是……”
<br/>　　“大唐有句俗话，叫留得青山在，不怕没柴烧。我服部氏只有你一个继承人，你不跟着着我一起走上这条绝路。”秦吉胜面色十分严肃，就好像是在交代后事一样。
<br/>　　倭人青年被秦吉胜吓坏了，颤抖的问道：“父亲，您到底在说什么啊，孩儿，孩儿怎么有些听不懂。”
<br/>　　“你闭嘴，先听我说。”一次又一次被打断之后，秦吉胜瞪了儿子一眼，然后正色说道：“我打算让你去大唐舰队告密，把我们想要偷袭大唐太上皇的事情告诉唐军，这样你就可以取得他们的信任，免于被这次的事情所波及。”
<br/>　　“不，我不去。父亲，我是不会出卖你的，而且唐人从来就没有看上过我们，就算是我去说了他们也不会相信我，就算是信了……，如果你们都不在了，我，我又能干什么。”
<br/>　　倭人青年听完秦吉胜的话之后，面色大变，他已经知道了自己老爹是想要牺牲自己来保全他。
<br/>　　秦吉胜的眼中闪过一丝挣扎，但最后还是坚定的说道：“这件事情就这么定了，一会儿你就趁着夜色下海，我会给你安排一条小船……”。
<br/>　　“不！”倭人青年坚定的摇了摇头：“父亲，我是不会离开你的，要死我们也要死在一起，服部氏没有怕死的族人！再说这一切还都是您的猜测，万一大唐其实并不知道这件事情有呢？万一他们是别有目的呢？您这样做岂不是自投罗网？”
<br/>　　“你这孩子怎么……”秦吉胜看着还在心存侥幸的儿子，心中不知道是应该生气还是欣慰。
<br/>　　从感情上来说，他当然不希望自己的儿子为了生存便将自己这个父亲弃之不顾；但从理智上来说，他还是希望儿子能够聪明一些，主动去举报自己，这样至少也可以为自己的家族留下一条根。
<br/>　　可是，秦吉胜不知道的是，在强大的实力面前，一切的计谋都是纸老虎。
<br/>　　李承乾的命令是血染十里海疆，那么大唐军队就会将这个命令彻底的执行下去，倭人的命在他们看来并不值钱，不管是秦吉胜也好，还是他的儿子也罢。
<br/>　　就算是倭人青年主动过去投降，主动检举揭发，最后的结局也不会有任何的改变。
<br/>　　不知道‘感恩’为何物的倭人，狼子野心的倭人必须死，这就是李承乾对自己手下将军们下达的绝杀令。
<br/>　　……
<br/>　　李震此时正坐在押送倭人的战舰之上，高高的舰楼之上，他可以透过望远镜看到自己想要看到一切。
<br/>　　“大都督，我们还在等什么？对于这种恩将仇报的东西，直接灭了他们不就完了么？何必这么跟他们耗着。”副将的刘仁轨站在李震的身后，端着一碗面条，有一口没一口的吃着，顺便不还忘了牢骚几句。
<br/>　　李震有些无奈的回头看了他一眼，没好气的说道：“你懂个屁，吃你的面吧！”
<br/>　　“呃……”刘仁轨被怼了一句，尴尬的咂咂嘴，眨眨眼睛说道：“刚刚是大都督你说不要的，现在怎么又来生气！”
<br/>　　“我看着你就来气。”李震咕哝了一句，指着外面那些倭国战船说道：“陛下说了，要让倭人的血染红十里海岸线，这些现成的‘染料’你就打算在这里杀了？万一到了倭国没办法完成任务怎么办？”
<br/>　　“倭岛上那么多倭人，也不差这么几个吧，把他们全都灭了我们速度还能快一点，说不定能提早完成任务呢。”刘仁轨无辜的眨眨眼睛，抱着自己的面条说道。
<br/>　　“行行行，你赶紧有多远给老子走多远，别在这里给老子呕气，看着你就烦！”李震实在看不下去刘仁轨吃货的样子，不耐烦的摆摆手。
<br/>　　这货应该是吃坏了脑子吧，也不想想，舰队能上岸么？上不了岸怎么可能完得成任务，真当倭人都是漂在海里的鱼呢，一炮下去炸死一大堆。
<br/>　　……
<br/>　　李震的北海舰队押送着秦吉胜等人回国，而在大唐的海参崴，一车又一车的装备被运了过来，整箱整箱的钢努，成捆成捆的三棱矢，堆的像小山一样高。
<br/>　　这些都是大唐已经淘汰下来的装备，不过用来打山地丛林战却是再好不过，相比于那些每打一枪都要装填一次的火铳，丛林里面，弩和弓才是靺鞨人最好的武器。
<br/>　　钦陵在十余天的时间里，已经集合了十万靺鞨士兵，这些人虽然没有经过很长时间的训练，但是长期的狩猎生活，让他们的个人战斗力丝毫不在任何人之下。
<br/>　　这一次他要去打的是一场山地丛林战，是一场绞杀战，可以用所过之处寸草不生来形容。
<br/>　　那些倭人的城市早已经在战争中被摧毁的差不多了，那些倭人的村落也是一样。
<br/>　　现在的倭国可以说山里面的人比城里的人多出不知多少倍，将他们找出来，杀死，便是此行的最终目的。
<br/>　　说来似乎有些残忍，因为那些躲起来的倭人其实很无辜，可是谁让他们的倭人中有人要打李二的主意呢，激怒了大唐皇帝之后，他们就需要承担这一切后果。
<br/>　　唐皇一怒，血流漂杵！
