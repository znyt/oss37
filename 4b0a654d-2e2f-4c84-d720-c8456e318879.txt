　　“弟子也时常听陵越师兄提起您。”
<br/>　　望着眼前的凝丹长老，百里屠苏说起话来都有些拘谨。
<br/>　　潜意识里，他就觉得慕清霄与他们根本不是同一类人，其中或多或少有些陵越的影响。
<br/>　　自他与陵越师兄相识以来，若说陵越口中提及最多的人，无疑是天墉城最年轻恐怖的天才，凝丹长老。
<br/>　　陵越对于慕清霄的崇拜与敬重，绝对是发自内心的，从他的语气中就能听出。
<br/>　　“陵越那小子，确实有几年没见了，以他的资质，修为想必已经不弱。”
<br/>　　慕清霄微微一笑，当年陵越与他同时进入天墉城，后者的性格他非常赏识，一直以来关系都不错。
<br/>　　甚至，紫胤真人闭关的时候，陵越曾经向他求教过剑术。
<br/>　　他也听说过，在半年前，陵越就已经迈入化神期，除了他以外，绝对是天墉城天赋最高的弟子。
<br/>　　突破化神期后，那小子就按耐不住，得到紫胤真人的允许，下山执行任务去了，短时间内估计不会回来。
<br/>　　在慕清霄与百里屠苏谈话之际，四周的新进弟子也窃窃私语，看向他的眼神，满是恭敬与崇拜。
<br/>　　“我可是听说了，那位可是天墉城大名鼎鼎的凝丹长老......”
<br/>　　“传闻凝丹长老是万年不遇的妖孽天才，二十岁突破练虚，更是一名七品炼丹师！”
<br/>　　“你们的消息早就过时了，我可是听说，凝丹长老已经是合道期真人，更是九品炼丹师，我的偶像啊！”
<br/>　　由于慕清霄的出现，新进弟子只是受了些皮外伤，惊吓过度，甚至有两名胆小的直接被吓晕了过去。
<br/>　　对于救了他们的慕清霄，他们自然是十分感激。
<br/>　　当然，在慕清霄看来，解决两只金丹期的姑获鸟，不过是顺手而为罢了，他来此是另有目的。
<br/>　　“这里便交给你处理，我还有些事情，需要去处理。”
<br/>　　百里屠苏拱了拱手，感激道：“长老慢走。”
<br/>　　慕清霄微微点头，身形化作剑芒便消失在原地，以极快的速度向翡翠谷外掠去。
<br/>　　...
<br/>　　...
<br/>　　此时，翡翠谷外的竹林小道上。
<br/>　　“二师兄，不如我们回去看看吧，要真是出了什么问题，我们可担不起责任啊，而且大家都是同门师兄弟......”
<br/>　　肇临走在竹林小道上，不时回头看一眼翡翠谷，心中极其忐忑，总觉得距离自己的道越来越远。
<br/>　　“哼，事已至此，若是怀疑到我们头上，也有大师兄来顶锅，哪怕真漏泄了，也有我来顶着，没你的事。”
<br/>　　陵端脸色阴沉，想到百里屠苏那副冷冰冰的模样，气不打一处来，道：“而且......”
<br/>　　“而且什么？”
<br/>　　就在这时，寂静的竹林小道中，陡然响起一道声音，令得陵端与肇临心都提到了嗓子眼上。
<br/>　　“凝......凝丹长老！”
<br/>　　当陵端抬起头来，看到站在前面的慕清霄时，脸色顿时变得煞白。
<br/>　　至于肇临早就吓得全身哆嗦起来，这若是让普通弟子听到也罢，但让凝丹长老给听见了，后果他已经不敢想象。
<br/>　　扫了两人一眼，慕清霄屈指一弹，两道恐怖的劲力，便将陵端与肇临击飞出去。
<br/>　　“噗噗——”
<br/>　　两人倒飞的同时，嘴中忍不住喷出两口鲜血，最后狠狠的摔在数十米外的地面上。
<br/>　　这两道劲力，他都控制在他们能够承受的范围，最多也就是落下个重伤，修养半个月就能恢复。
<br/>　　他虽然厌恶这种阴险狡诈的弟子，但两人毕竟是掌门真人的弟子，若他真的出手废了他们，至掌门真人于何地？
<br/>　　哪怕真想废了陵端与肇临，他也不能亲自出手。
<br/>　　“陷害同门，难道掌门真人就是这样教导你们的吗？”
<br/>　　慕清霄的语气平淡，传入两人耳中，却犹如晴天霹雳。
<br/>　　恐惧的同时，只能强忍着重伤，从地面上爬起，乖乖跪在地上，不断磕头求饶。
<br/>　　“请凝丹长老饶命，请凝丹长老饶命啊！”
<br/>　　“哼，心术不正，谈何修道，现在能够陷害同门，将他们至于危险中于不顾，下次是不是也能设计加害于他们？”
<br/>　　此时，肇临已经吓的面无人色，不敢接话，至于陵端更是不断磕头求饶，浑身冷汗直冒。
<br/>　　就在刚才，他真的以为慕清霄要出手杀了他们。
<br/>　　“念你们是初犯，自己去掌门真人那里请罪吧，至于如何处置你们，都由掌门真人来决定，听到了没有？”
<br/>　　闻言，陵端面如死灰，肇临也是脸色惨白，但不知为何，心中却是松了口气。
<br/>　　“是......是，凝丹长老。”陵端颤声开口道。
<br/>　　站在那里的可不是随意欺压的普通弟子，而是在天墉城有着崇高地位的凝丹长老。
<br/>　　只要他一句话，别说他是涵素真人的弟子，哪怕直接将他杀了，涵素真人恐怕也无话可说。
<br/>　　慕清霄负手而立，收回目光后，便化作一道剑芒消失于天际。
<br/>　　待慕清霄离开，十多分钟后，陵端才颤颤巍巍的站起身来，感觉全身的力气都被抽的一干二净。
<br/>　　肇临站起身来，脸色苍白，道：“二......二师兄，我们怎么办呐，掌门真人肯定不会轻饶我们。”
<br/>　　陵端闻言，脸色顿时变成了猪肝色，但他也没有办法，难道将凝丹长老的话当成耳旁风吗？
<br/>　　这岂不是找死！
<br/>　　跟凝丹长老比起来，他不过是掌门真人的弟子，若论重要性，一千个他恐怕都不如慕清霄一个。
<br/>　　“我们立刻就去掌门真人那里请罪，越快越好，重罚肯定是逃不了了，但总比被逐出天墉城的好。”
<br/>　　陵端心里清楚，若是他们不去掌门真人那里请罪，往后在天墉城的日子，绝对不会太好过。
<br/>　　如果真将凝丹长老给得罪死了，往后的丹药资源，绝对是没着落了。
<br/>　　他是涵素真人的弟子，资质也非常不错，看在这份情面上，涵素真人肯定不会将他们逐出天墉城。
<br/>　　若真是如此，丢的可是涵素真人自己的脸面。
<br/>　　至于重罚，陵端已经做好的心理准备，虽然有些忐忑，但这也是没办法的事情。
<br/>　　谁让他们运气如此差劲，事情居然让凝丹长老给听到了呢。
