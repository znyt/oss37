　　四季城，本身就武者极多，几乎没有几个普通人。
<br/>　　而作为城内霸主的白家，偌大的府邸庭院上，今日更是挤满了武者。
<br/>　　观这些武者的气息，最弱的都是先天境，洞玄境武者竟不在少数。
<br/>　　萧逸也来到了白家，当看到这个阵势时，也是微微吃惊。
<br/>　　在场的武者，无一不是百武城附近有名之人，或猎妖师，或家族武者，还有名气不小的炼药师。
<br/>　　众人都在议论纷纷，一副愁眉苦脸的样子。
<br/>　　萧逸大概也听明白了过来。
<br/>　　白家家主要雪魄丹，是为了救她唯一的女儿，白冰雪。
<br/>　　那白冰雪，传闻容貌绝美，更是北山郡有名的天才，年纪轻轻，位列北山榜十二。
<br/>　　她修炼的是一门强大而奇怪的寒冰属性功法，一旦练成，可造就雪魄之体，极为强大。
<br/>　　然而，雪魄之体，起码要洞玄境以上的修为，方能开始修炼。
<br/>　　她却在修为未足的情况下，仗着自己天赋过人，强行修炼，最终被反噬，昏迷不醒。
<br/>　　如今唯一能救她的，只有雪魄丹。
<br/>　　借助雪魄丹的冰属性力量，强行抵消掉反噬，方能让她苏醒。
<br/>　　然而，雪魄丹位列四品巅峰丹药，岂是一般炼药师能炼制的。
<br/>　　三品炼药师，想都不用想；四品炼药师，失败几率极大。
<br/>　　只有五品炼药师，方能有九成以上的把握炼制成功。
<br/>　　“这雪魄丹，位列四品巅峰，但只对修炼冰属性功法，或者冰属性武魂的武者有效，偏门得很，就算是四、五品的炼药师，也没有多少炼制经验阿。”
<br/>　　“可不是嘛，如此偏门的丹药，没有足够的炼药经验，失败几率太大了。”
<br/>　　在场的，大多是三品炼药师，一个个都束手无策，只能无奈地闲聊着。
<br/>　　“先不说失败几率大，想找一个五品炼药师已经够麻烦的了。”
<br/>　　“我听说裂天剑派中不是有一位炼药长老，乃是五品炼药师吗？”
<br/>　　“有是有，白家主早就去请过了，可不不巧的是，那长老在闭关，十天后才出关，如何能来得及阿？”
<br/>　　“以白家主女儿如今的情况，若三天之内无法得到雪魄丹，那种反噬足以要了她的命。”
<br/>　　萧逸在一旁听着，暗道难怪白家这么急，要在三天内找到五品炼药师，原来是白家家主女儿的伤势拖不下去了。
<br/>　　像雪魄丹这种偏门又高品的丹药，就算是大商行，一时半刻也很难找得到。
<br/>　　“静一静。”
<br/>　　忽然，一位管家模样的老者走到庭院，喝了一声。
<br/>　　一声轻喝之下，庭院中原本因谈论之声而产生的喧嚣瞬间停了下来。
<br/>　　“好强的气势。”萧逸暗暗惊讶，面前老者的修为怕是在洞玄七、八重以上。
<br/>　　白家一个下人尚且有如此修为，可想而知整个白家的武者力量非同小可。
<br/>　　“今日来我白家的，皆是附近十数城赫赫有名之辈，可有谁有那个能力炼制雪魄丹？”
<br/>　　管家模样的老者沉声问道。
<br/>　　在场的，皆是不弱的武者，洞玄境的不下数十人，连柳絮城的柳城主也赫然在此。
<br/>　　可是，他们都不是炼药师。
<br/>　　在场的炼药师，有十多人，但最高不过三品。
<br/>　　这些人，本来就没有办法。不过冲着白家的面子，想过来看看能否略尽绵力，再者就是交好白家。
<br/>　　在场所有人都沉默着。
<br/>　　老者脸上显然掠过一丝失望和着急。
<br/>　　“哎。”老者叹了口气，摆摆手，道，“诸位来白家一趟，有心了，待会儿我会命下人准备好酒席，好生招待诸位。”
<br/>　　说罢，老者准备离去。
<br/>　　恰在此时，白家府邸之外，传来一阵高亢的声音。
<br/>　　“百武城盛宝商行，慕容胤，前来拜访白家。”
<br/>　　话音落下不久，一群人快速走到庭院中。
<br/>　　为首的是两个中年人，后面跟着十数人。
<br/>　　管家模样的老者见到两位中年人，连忙上前对着其中一位拱了拱手。
<br/>　　“慕容胤，白家好大的面子，竟然将他请来了。”
<br/>　　“慕容胤可是百武城盛宝商行的负责人，加之还是一位成名已久的四品炼药师。”
<br/>　　在场武者看向慕容胤的目光，充满了惊讶。
<br/>　　一旁的萧逸在人群中暗暗皱眉，北山慕容家的人？
<br/>　　盛宝商行，遍布北山郡每一城。而像破玄城、百武城这种较为强大的大城中，负责人一般由北山慕容家的直系族人担任。
<br/>　　这慕容胤，便是北山慕容家中的一位内门执事，而且还是权利比较大的那种，被派遣至百武城担任负责人。
<br/>　　在百武城附近十数城，乃是大人物之一。
<br/>　　“白管事，给你介绍一下，这位是慕容杀执事，刚刚从家族那边来到百武城。”慕容胤对客气地对老者说道。
<br/>　　北山慕容家，位于郡都。
<br/>　　郡都位于全郡的正中央，距离破玄城、百武城很远。
<br/>　　“慕容杀执事，有礼了。”老者，也就是白管事，客气地说了一声。
<br/>　　慕容胤问道，“白管事，现在可找到五品炼药师了？”
<br/>　　“未曾。”白管事无奈地摇了摇头，“家主如今烦心不已，就不出来招呼两位了。”
<br/>　　“白管事言重了。”慕容胤连忙摆手。
<br/>　　白家乃是北山郡有名的大家族之一，虽然比不上北山慕容家、暗影楼这等霸主势力，但也是大势力之一。
<br/>　　慕容胤小小一个慕容家执事，自然要客客气气的。
<br/>　　“若是有什么在下能帮的，白管事尽管吩咐便是。”慕容胤说道。
<br/>　　白管事点点头，道，“有劳慕容胤执事了，若是再找不到五品炼药师，便要劳烦慕容胤执事炼制雪魄丹了。”
<br/>　　这时，白家府邸外，再次传来一阵高亢的声音。
<br/>　　“方某来也。”
<br/>　　话音刚落下，一个老者飞跃而来。
<br/>　　当老者落下，在场武者的目光，比之前看到慕容胤出现时更为惊讶。
<br/>　　“方鹤？那位云游炼药师，百武城炼药术最高之人。”
<br/>　　“听说此人行踪飘忽，很难找到，这次竟也主动来了。”
<br/>　　方鹤，四品巅峰炼药师，传闻此人曾经炼制出五品丹药，虽说只是五品中级别最低的破玄丹，但也非常了不起了。
<br/>　　论炼药之术，比慕容胤还要强上数分。
<br/>　　“方大师。”白管事连忙上前打招呼。
<br/>　　“白管事。”方鹤同样拱了拱手，回礼。
<br/>　　“方鹤。”这时，一旁的慕容胤冷声说道，“你也来凑这热闹？”
<br/>　　“慕容胤？你也来了。”方鹤冷笑道，“也是，哪里有好处，哪里就会有你们北山慕容家的人。”
<br/>　　“白家的一个承诺，自然也把你吸引来了。”
<br/>　　方鹤丝毫不卖慕容胤面子，反而话里有话。
<br/>　　慕容胤冷声道，“方鹤，你不也是为了这个承诺而来？别把自己说得那么高尚。”
<br/>　　两人皆是百武城有名的炼药师，传闻两人有些过节。
<br/>　　“哼，老夫懒得理你这种小人。”方鹤不屑地说了一声，而后看向白管事，问道，“可找到五品炼药师了？”
<br/>　　“我听闻白世侄女的情况刻不容缓阿，可需我即可开炉炼丹？”
<br/>　　白总管回答道，“还未找到五品炼药师，不过方大师还需再等一会儿。”
<br/>　　“哦？”方鹤疑惑道，“还有其他炼药师来吗？白管事，方某也是白家的常客了，我的炼药术你信不过？还需老夫等其他炼药师？”
<br/>　　“方大师稍安勿躁。”白管事道，“家主派人去请一位炼药师过来，按着时间，应该差不多到了。”
<br/>　　“哦？白家主派人请的？”方鹤虽然疑惑，但却没有再多说什么。
<br/>　　萧逸则在人群中极不显眼地待着，没人留意他。
<br/>　　半个时辰后，一道人影如期而至。
<br/>　　来人是个年轻人，约莫二十出头。一裘白衣，翩翩而落。
<br/>　　在场众武者的神色，顷刻间大变。
<br/>　　“叶铭，裂天剑派炼药堂首席弟子。”
<br/>　　“他可是那位炼药圣手坐下的第一高徒，竟也来了？”
<br/>　　叶铭落下，白管事快步前来行了一礼。
<br/>　　“白管事不必客气。”叶铭轻声道，“白家坐落于裂天剑派之外，彼此也算邻居了。”
<br/>　　“此番师傅闭关，便由在下过来，看能否略尽绵力。”
<br/>　　原来等候之人，竟是裂天剑派炼药堂首席弟子，方鹤和慕容胤顿时没有半分不满。
<br/>　　“有劳。”白管事行了一礼，而后环视了四周武者一眼，高声问道，“可还有哪位有本事炼制雪魄丹？”
<br/>　　在场，除了慕容胤、方鹤以及叶铭三人神色自若外，其余所有武者，纷纷面露无奈，没人应话。
<br/>　　白管事皱了皱眉头，但也只能摇摇头。
<br/>　　恰在此时，一道稚嫩的声音响起，一道人影走了出来，“还有我。”
<br/>　　“你是？”白管事面露疑惑，明显不知道面前的是何人。
<br/>　　不过面前之人一身黑袍，戴着面具，眉宇间透漏着一丝坚毅和冷酷，看起来，似乎只是个少年。
<br/>　　“在下易霄。”
<br/>　　此话一出，在场武者顿时一惊。
<br/>　　“易霄？那个天才猎妖师？”
<br/>　　“那个打败了无忧公子的紫炎易霄？”
<br/>　　一身黑袍，戴着面具的武者，并不少。但这身装扮，又叫易霄的，只有一人。
<br/>　　正是最近声名鹊起的紫炎易霄。
<br/>　　“噗...”在场武者忽然哄堂大笑。
<br/>　　“易霄也来了，他也是炼药师？哈哈，一品的还是二品的？”
<br/>　　“虽说他是个武道天才，但武道跟炼药可是两回事。”
<br/>　　“如此年轻，就算真的会炼药，无名师指导，怕也只有一品二品了。”
<br/>　　慕容胤更是不屑地哼了一声，“哪来的毛头小子，滚一边去，浪费时间。”
<br/>(本章完)
