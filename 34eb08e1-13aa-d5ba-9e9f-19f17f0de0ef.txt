
<br/>　　<br/>　　    平日里的抚仙湖碧波荡漾，湖水清澈，好一片山水美景。<br/>　　<br/>　　    但阴雨下的抚仙湖，也有它狰狞的一面，风吹水浪起，怒吼连连，王惊蛰站在岸边的时候，都感觉到了如果此时深入这片湖泊的话，该是怎样一种翻腾的场景，那就是身在其中心有余而力不足了。<br/>　　<br/>　　    雨越下越大，渐渐地视线都有点模糊了，打着伞都已经不管用了，雨滴一个劲的往身上拍着，片刻后就已经都湿透了，陈三岁在他旁边说，这次的雨比他之前遇见的那几次都要大多了。<br/>　　<br/>　　    “这下了雨的抚仙湖，到底能带来什么一番景象呢？”王惊蛰不解的轻声嘀咕了一句。<br/>　　<br/>　　    在岸边站了一个多小时，浑身上下都已经被彻底湿透了，湖面上全是跳跃着的雨滴，水浪一个劲的往岸上冲刷着，但除此以外抚仙湖依旧看不出有什么不同来，看的王惊蛰都有点皱眉了。<br/>　　<br/>　　    “要不是巧合？”陈三岁诧异的说道。<br/>　　<br/>　　    “哪里有那么多的巧合，你碰上了两次，几十年前也有，这是商量好了，都挑着下雨天来凑热闹啊，接着看吧……”王惊蛰抹了把脸上的雨水，眯着眼睛往湖中心望去，那一片区域的浪花翻滚的尤其大。<br/>　　<br/>　　    就在这时，陈三岁和王惊蛰忽然对视一眼，两人惊愕的扭过头又看向了湖面，此时湖水上方绵延出了一层淡淡的雾气，如果要是寻常人来看的话，怕不是以为这是下雨拍打在湖面上升起的水雾了，可在他们去看，这是雾气昭昭的阴气，整座抚仙湖在顷刻间都被笼罩在了一团阴气中，更诡异的是这阴气只是漂浮在水面上，一点都没往外渗透，哪怕是站在岸上的他俩一点都没感觉得到，就仿佛被局限在了一定的范围里。<br/>　　<br/>　　    “踏踏踏……”王惊蛰迈步就朝着岸边的湖水中走去，蹲下身子手就伸进了水里，入手冰凉刺骨，让人打了个冷颤。<br/>　　<br/>　　    “王哥？”后面的陈三岁忽然叫了一声，王惊蛰回头，就看见他手抬起来指着村东面的一处峭壁，那座山头上不知何时出现一片虚影，影影绰绰的能看出来似乎是个城市的轮廓，阴气昭昭的，弥漫在了山头附近。<br/>　　<br/>　　    “哗啦，哗啦”王惊蛰趟着水就回来了，把伞扔了后说道：“走，过去”<br/>　　<br/>　　    两人拔腿就跑，在雨中一路狂奔向了村东面的那座山峰上，山峰三古城轮廓的影子很淡，如果不仔细看的话都未必能发现得了，常人根本看不出那是一片阴气。<br/>　　<br/>　　    半个多小时后，两人一路疾驰到了山头上，这一片山林中果真阴气连绵，但从这个地方的角度，正好能俯视到下面的抚仙湖。<br/>　　<br/>　　    王惊蛰站在峭壁的边缘，朝着四周观望了片刻后，眼神就落在了湖中心，之前在岸边时抚仙湖中间一带的水域，水花就翻滚的大了不少。<br/>　　<br/>　　    从山顶向下，看向湖中心的时候，那一片浪花翻滚的区域，那座古城的轮廓又再次显现出来，而这一次出现的古城，远比他们刚才在山头上看见的还要清晰，全面。<br/>　　<br/>　　    王惊蛰，陈三岁顿时惊得瞠目结舌，呆立在风雨中，久久的连眼睛都没眨一下，因为不光是那座古城太震撼了，城中还站着数不清的人，男的身子朝前倾斜，女子向后仰着，这就跟一副电影的画面被按住了暂停键一样，太诡异了。<br/>　　<br/>　　    这种情形完全是在阴气笼罩下显现出来的，如果要是海市蜃楼那种状况的话恐怕千百年来早就被当地人给发现了，毕竟常人是没办法看出阴气的。<br/>　　<br/>　　    古城的规模很宏大，壮观，翻腾在浪花中一眼都看不见头，城中的建筑错落有致，一条条的街道和房屋井然有序的排列着，这种城市的规划格局一点都不必现代的城区要差。<br/>　　<br/>　　    “城中，有一座辉煌的建筑，跟故玛雅人的金字塔差不多，高能有十层楼左右……”王惊蛰想起了之前黄九郎跟他描述过的古城景象，眼神一下子就到了古城最正中央的那栋巨大建筑，一看之下确实是一栋金字塔模样的建筑那个高度特别显眼，醒目，你想看不见都不行。<br/>　　<br/>　　    陈三岁倒吸了一口冷气，惊骇的说道：“那座金字塔，哪怕就是现在都不一定能轻易的建的起来，两年多年前的人是怎么搭起来的，这他么的是不是幻觉，也太不真实了”<br/>　　<br/>　　    王惊蛰没有吭声，皱着眉头继续观望，在那栋金字塔的四周，还有几座高耸的建筑，有点类似于古罗马斗兽场一样，虽然没有那座金字塔高，但在古代来说的话，仍旧能够惊掉人下巴的了，就这座古城中的一切，如果能原原本本的从抚仙湖底展现出来的话，怕不是世界八大奇迹，又得再加一项了，这可一点都不比秦始皇兵马俑或者万里长城差到哪去了。<br/>　　<br/>　　    王惊蛰的眼神逐渐的扫了一圈看了个大概后，忽然间木然一惊，他一把拉住陈三岁的胳膊，急促的说道：“快点，尽你可能的记忆，快把古城的遗址记在脑子里，能记多少记多少，我从东往南，你从北往西，咱俩一起来记”<br/>　　<br/>　　    陈三岁愣了下，随即也反应了过来，他连忙扭头看向抚仙湖中间，两人一前一后开始迅速的将古城轮廓记在脑子里。<br/>　　<br/>　　    不知过去多久，这场大雨来的快去的也不慢，差不多应该是临近到中午的时候，雨势开始慢慢减小了，湖面上的风浪也没之前那么大了，渐渐地，随着雨越来越小以后，抚仙湖中古城的影响也开始变淡了不少，一直到雨停歇，就彻底消逝了。<br/>　　<br/>　　    雨停了，王惊蛰从口袋里掏出烟，发现烟全都湿了，他叹了口气随手扔向了山坡，说道：“你记下来多少了？”<br/>　　<br/>　　    “零零散散的，大概有七八成左右吧”陈三岁羞涩的说道。<br/>　　<br/>　　    “啊？”王惊蛰惊讶的我擦了一声，说道：“这么多呢？”<br/>　　<br/>　　    陈三岁说道：“我从小的记性就比较好，以前看出差不多都是过目不忘的，现在基本也是，我看过一眼的人不管隔多久，我再看见都能想起来”<br/>　　<br/>　　    王惊蛰感叹的说道：“也是，你人是痴呆蠢笨了点，记性好了，也算弥补下你的缺失了，老天还是公平的……”
