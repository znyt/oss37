　　他堂堂云丹宗十大长老之一，当着这么多人的面，自然不可能去杀了一个小辈，那样就太丢份了。
<br/>　　更何况，他之所以想要收叶尘为徒，也并非是出自真心，而是觉得叶尘的炼丹手法，包含着无尽的奥妙，这才是他的真正目的。
<br/>　　“小子，既然你如此不识时务，那就休怪老夫无情了！！”
<br/>　　火龙真人在心里，已经开始暗暗筹划着，如何把叶尘口中的那本古籍弄到手了。
<br/>　　叶尘自然也能看出，这老头不安好心，不过他并没有太放在心上。
<br/>　　区区一个半步化神境而已，还对他构不成威胁，如果对方不知死活，那他自然也不会手下留情。
<br/>　　从丹师盟会的手中接过二品丹师宝卷之后，叶尘缓步走向高台。
<br/>　　哗啦啦！
<br/>　　台下众人顿时纷纷躲避开来。
<br/>　　原本，那些还想和叶尘套近乎的人，目睹了他拒绝火龙真人的场景，自然也不敢再上前，生怕因此得罪了火龙真人。
<br/>　　叶尘也乐得如此，直接在台下站定，准备继续参加三品炼丹师的考核。
<br/>　　就在这时，冷蝉衣忽然缓步来到叶尘的跟前，主动开口道：
<br/>　　“你叫叶尘对吧？我是冷蝉衣，你刚刚为什么要拒绝火龙长老？你知不知道，你错过了一个千载难逢的机会！”
<br/>　　冷蝉衣是发自内心的，为叶尘感到可惜。
<br/>　　虽然她也清楚，云丹宗多一名核心弟子，对她来说，就多一分威胁，不过她并不在乎，因为她更喜欢挑战。
<br/>　　而叶尘，一连两次炼制出完美品阶的丹药，即便是她，自问也很难办到，所以她是真的希望叶尘加入云丹宗。
<br/>　　相比于屠苏公子，眼前的这个少年，或许现在还不够资格成为她的对手，但是在不久的将来，一定可以！
<br/>　　叶尘望着眼前这个，犹如水兰花一般素雅的女人，不由得想起前世，两人第一次相遇的场景。
<br/>　　记得那时，叶尘还尚未成为凶名赫赫的一代狂帝，偶然参加了一次炼药师大赛，和冷蝉衣在决赛中成为了对手。
<br/>　　而那个时候的冷蝉衣，已经在炼丹界小有名气，是名动一方的丹道天才。
<br/>　　然而在那场比试之中，踌躇满志的冷蝉衣，惨败于叶尘之手，沦为第二。
<br/>　　自那之后，叶尘去到哪里，这女人便追到哪里，非要在炼丹术上胜过他才肯罢休。
<br/>　　就这样，被她一连纠缠了数月之后，叶尘实在不胜其扰，最终只好故意输给了她一次，此事才算了结。
<br/>　　在叶尘看来，这个女人，就是名副其实的丹痴，在她的眼中，只有炼丹术，再也容不下其他东西。
<br/>　　而事实上，也正是因为她的这种极度专注的性格，在数百年之后，终成一代丹圣，带领云丹宗，再现了昔日云丹仙子的辉煌。
<br/>　　这些念头，在叶尘脑海中迅速闪过的同时，脸上不由得露出一抹笑容，淡淡道：
<br/>　　“不为什么，我不想做的事情，没有人可以逼迫我！”
<br/>　　冷蝉衣微微一愣，
<br/>　　不知道为何，眼前这个少年，她明明是第一见，但对方说话的语气和眼神，倒像是老朋友一般，给她一种莫名的熟悉感。
<br/>　　冷蝉衣心头感到微微诧异的同时，忍不住蹙眉道：
<br/>　　“你这话未免太狂妄了些吧？火龙长老的炼丹水平，虽然在十大长老之中，只处于中等，可却也是货真价实的五品炼丹宗师，而在五品炼丹宗师里面，绝对是最顶尖的存在！”
<br/>　　“再加上他老人家，一身修为，已经达到半步化神境，不知道有多少人，想要拜在他的门下，难道还不配当你的老师？”
<br/>　　叶尘淡淡一笑，不置可否。
<br/>　　他如果说是，冷蝉衣肯定不信，所以也不想和她做口舌之争，索性闭上眼，不再言语。
<br/>　　冷蝉衣见状，顿时气得不行，
<br/>　　她在云丹城，是出了名的冰山美人，一向眼高于顶，对任何男人都不假辞色，如今屈尊降贵，好心规劝于他，他竟然丝毫不领情！
<br/>　　至于周围其他人，看到这一幕，早就惊的眼珠子都掉下来了。
<br/>　　火龙长老主动提出收他为徒，被他一口回绝！
<br/>　　无数人心中的完美女神，主动和他搭讪，他居然还爱理不理！
<br/>　　这家伙，也太彪悍了吧！
<br/>　　……
<br/>　　而此时，另外一边，
<br/>　　二品炼丹师的考核，也已经结束。
<br/>　　“接下来，我们开始三品炼丹师的考核，想要参加考核的，请到台上来！”
<br/>　　随着丹师盟会的长老宣布出来，叶尘再次缓缓登上考核大台，
<br/>　　“快看！那家伙又上去了！他这是准备一口气晋升到三品丹师吗？”
<br/>　　“按照他之前的表现来看，应该没有什么问题吧？”
<br/>　　“那可不一定！炼丹之道，每晋升一品，难度都要提升十倍不止，纵然他可以炼制出一枚完美品阶的二品丹药，可倘若这一次，如果运气不好，抽到一个偏僻的药方，恐怕也未必能够通过！”
<br/>　　“不错！三品以上的炼丹师，不但对炼丹技术有很高的要求，还要具备渊博的药方知识，要从一百多副药方之中，随即抽取三个，然后从中再挑选其中一个，来作为考核的试题，难度不是一般的大！”
<br/>　　……
<br/>　　在众人的议论声中，丹师盟会的长老宣布完考核规则之后，三品炼丹师考核，便正式开始了。
<br/>　　参加这场考核的人数，比上一次再次大幅度降低，仅仅只有三十来人。
<br/>　　而且一连十人过后，竟然无一人通过考核，全部折戟沉沙！
<br/>　　一直到第十三人，方才成功炼制出了一枚粗制级别的三品丹药，勉强通过了考核。
<br/>　　之后又一连数人，全军覆没。
<br/>　　终于，等到了叶尘登场，
<br/>　　全场顿时一下子安静了下来。
<br/>　　而就在叶尘准备登上考核台，抽取考核题目的时候，他忽然看到，坐在考官席上的火龙真人，嘴巴忽然动了几下。
<br/>　　而那位负责抽题的考官，身体微微一滞，旋即冲着火龙真人点了点头，两人应该是通过神念传音，进行了某种交流。
<br/>　　叶尘见状，眉头微微一皱，猜测火龙真人应该要针对自己。
<br/>　　果不其然，
<br/>　　他登台之后，一连抽取了三道试题，分别是噬生丹、冰灵丹、失心丹。
<br/>　　而这三种丹药，可以说是众多三品丹药之中，最为偏僻三种！
<br/>　　甚至在场的很多人，都从来没有听说过。
<br/>　　很显然，那位主考官，在火龙真人的示意下，悄悄对叶尘的试题做了手脚，故意给了他三道最难的药方！
