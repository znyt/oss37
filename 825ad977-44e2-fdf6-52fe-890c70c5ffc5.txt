　　　　玄天一斧开天，一息灭世，灭世大磨转动，天地都将覆灭。
<br/>　　覆巢之下，岂有完卵？
<br/>　　帝俊此刻正在那充满大寂灭的世界之中，若是他不能逃出，便会随着那世界一起毁灭。
<br/>　　“嘶！”
<br/>　　这一刻，宇宙间各路大神通们纷纷倒吸一口气，集体失声了，他们看到了什么，妖族之主帝俊与人族武祖大战，竟然落入了下风？这怎么可能？但这又是事实，就发生在他们眼前，容不得反驳。
<br/>　　这是一场震惊万古的大战，人族武祖玄天大战天庭之主帝俊，帝俊竟然落入下风，快要被玄天灭了。这是一种不可思议的场面，说出去谁都不会相信。
<br/>　　这一战就这么的突然爆发了，帝俊竟然落入下风。
<br/>　　“大哥！”
<br/>　　远处，东皇太一见此，大声惊叫，眼中满是担忧，想要上前去帮助，可是，他就连自己都快保不住了，还怎么帮助别人。因为之前红云的自爆，到如今他的伤势都还没有好转，此刻又被五位人族武皇夹击，使得他的伤势更重了。
<br/>　　玄天满脸杀气，神色淡漠的看着眼前这一切，他知道，帝俊是不会那么容易就死的。果然，如他所料，只见那毁灭的世界之中。帝俊身化一轮金色大日，煌煌之威，大日破天。
<br/>　　一轮煌煌大日光耀天下，携着无尽神威，破开世界，飞出灭世大磨的笼罩，来到洪荒之中。大日东升，越来越高，越来越璀璨，神光迸发，烈日炎炎，照破山河万朵。那毁灭天地的神光，正向着玄天照射过来。
<br/>　　“嗡！”
<br/>　　玄天出手了，他一出手便是将周边虚空化为一片黑暗，黑暗笼罩而去，向着大日包围而去。漆黑一片，湮灭一切光明，寂灭一切灵魂，天地将归于黑暗，黑暗一出，光明退避。
<br/>　　光明是黑暗的克星，黑暗有河床不是光明的克星呢？两者之间只不过没有绝对的强弱，有的只是施道者所掌握的道则强弱罢了。
<br/>　　那轮大日被无尽的黑暗包围，漆黑一片，不见丝毫光明，天地沉沦。
<br/>　　“嗡！”
<br/>　　就在此时，玄天不停，伸出一指，手指顿时化为一道巨大的天地之柱，镇压世间一切。这一指犹如混沌魔神的一指，寂灭天地万道，群邪退避，诛尽万妖。
<br/>　　古神一指！
<br/>　　天地静止，虚空禁锢，世间停止，岁月枯竭，万道轰鸣，道则相合，镇压诸天，万界俯首。
<br/>　　“轰！”
<br/>　　一指点出，从上而下，从虚无到实质，这一指从虚空之中点了出来，化为无量神光，轰在那正在被无尽黑暗腐蚀的大日之上。
<br/>　　“咔擦”
<br/>　　随着一声轻响，大日之上传一道裂缝声，紧接着，裂缝不断，不过瞬息之间，那一轮大日便化为无尽碎片，从虚空之中洒落下来。
<br/>　　“咳咳……”
<br/>　　一阵咳血声响起，帝俊的身影从虚空之中跌落出来，此刻的帝俊，早已没有了之前那般霸气雄伟，只见他全身气息萎靡，满脸苍白，身上一个巨大血洞，贯穿其胸口，血肉变得模糊，他身上的金色帝袍也变得破败不堪，再也不复之前那般威严了。
<br/>　　“死！”
<br/>　　玄天淡漠出声，神色冷淡，眼中杀机爆闪，人族死去这么多人，完全是因为妖族的屠杀，对于帝俊与东皇太一，他并不想放过，人族之仇，需要用鲜血来洗刷，刻骨铭心，试问，他又怎能放过他们？
<br/>　　玄天再次凝聚出遮天巨掌，一掌如同天网一般，覆盖苍穹，翻天覆地，巨掌之下，一切都将毁灭。
<br/>　　他这一掌带着无尽神威，对着帝俊轰下，破开星云万朵，震荡苍穹，欲要一掌覆灭帝俊。
<br/>　　然而，正在此时，突如其来，一只晶莹玉手，穿过层层虚空限制，自外向内，于那三十三天之外，九天之上，携带着无尽圣威与浩瀚无比的伟力，直奔玄天而来。
<br/>　　“女娲！”
<br/>　　玄天冷声大喝，声音之中夹杂着无尽愤怒，响彻九霄，轰动洪荒。
<br/>　　骤然遭逢强敌越界来袭，玄天心有所感，口中一声冷喝，只见他那高举的惊天巨掌依旧毫不犹豫的向着帝俊方向轰落下去，而另一只手，则是擎天而起，带着无尽的愤怒，毫不示弱的向着女娲圣人的那只玉手对轰上去。
<br/>　　“轰！”
<br/>　　惊天动地，一声巨响，来自混沌天外的天道圣人女娲的一击，人族武祖以准圣圆满修为，堪比圣人的神威，必须以全力对之。双方强势一击，掀起无边惊涛骇浪，一时之间，乾坤震裂，天穹破碎，地水火风翻滚不断，竟然再现无尽混沌的景象。
<br/>　　当然，这还只是圣人随意一击，若是女娲真的认真起来，玄天绝不会如此轻松的接下这样一击。
<br/>　　“轰隆隆！”
<br/>　　与此同时，玄天的那一惊天巨掌，带着无法抵挡的霸道神威，轰击在了帝俊身上。
<br/>　　“噗——”
<br/>　　一口逆血喷出，帝俊被击落跌倒在数千里之外的大地之上，掀起无尽烟尘。
<br/>　　玄天皱了皱眉头，因为他发现帝俊并没有死，果然——
<br/>　　“咳咳……”
<br/>　　强自支撑起自己的身体站起，昔日纵横洪荒天地的妖族妖帝，此时此刻已如风中残烛，摇曳的光火，随时可能有熄灭的迹象。帝俊的身体已经半残，血肉模糊，元神也暗淡无光，好似随时可能倒地死亡。
<br/>　　“难怪……”
<br/>　　玄天恍然，他发现，在帝俊身上竟然穿着一件顶级的铠甲，而且已经融入到身体之内，他之前并没有发现，现在看来，就是那铠甲帮助他御去一部分力道，所以他才能活下来。
<br/>　　“哼！既然不死，那我便再赐你一掌，看你死不死！”
<br/>　　玄天一声冷哼，于无边风浪之中，再次凝聚惊天一掌，恐怖的气息弥漫，无情的杀意蔓延，顷刻之间，便充斥周遭。
<br/>　　“玄天道友，住手！”
<br/>　　眼见玄天还要继续轰杀帝俊，九天之上，祥云翻滚之间，圣人女娲，浮现出身影，顿时，无穷无尽的力量汇聚，造化之能，乾坤之变，尽在其中。无可估量的恐怖圣威，强势压向玄天而来。
<br/>　　“拜见女娲娘娘！”
<br/>　　“拜见圣母！”
<br/>　　乍一见圣人女娲到此，不管是巫族、妖族、还是人族，都毫不犹豫地对着女娲跪拜了下来。
<br/>　　“免礼！”
<br/>　　女娲轻轻一拂衣袖，顿时，所有人都感受到一个不可抗拒的力量，将他们扶起。
<br/>　　“多谢娘娘（圣母）！”
<br/>　　众人再次拜谢。女娲却是没有管其他人，而是将目光投向玄天。
<br/>　　“嗯？”
<br/>　　女娲娘娘的到来，令玄天不由得为之眉头一皱，手中动作也不由得停了下来，他心中暗叹一声，看来今天想要杀了帝俊与东皇太一，是做不到了，可惜，若是女娲再晚来一分钟，他便能将帝俊杀死。
<br/>　　真是可惜，错过这样的机会，还不知道下一次要等到何时呢？
<br/>　　唉……正是人算不如天算！
