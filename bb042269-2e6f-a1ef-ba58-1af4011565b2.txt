　　其实若论真实实力，叶尘最多也就和绝无敌旗鼓相当，甚至还要稍微逊色半筹。
<br/>　　怪只怪，绝无敌和龙辰两个人，都是先入为主，认定了叶尘的实力，远远不如绝无敌，所以才会接连两次都惨被打脸。
<br/>　　而叶尘，虽然先后两次逼退绝无敌，其实也都是利用了对方轻敌之心，而且又有所顾忌，不敢对叶尘下杀手，所以才能出其不备，但其实对绝无敌根本没有造成任何伤势。
<br/>　　可是绝无敌就不会这么想了，
<br/>　　他自信满满的提出三招之约，结果两次都被叶尘逼退，这种羞辱对于他来说，简直比杀了他还要难受！
<br/>　　“啊！！！”
<br/>　　羞愤交加的绝无敌，顿时扬天大吼了一声，双眸都变得赤红无比，杀气腾腾，
<br/>　　“小子！我今天非杀你不可！！”
<br/>　　轰隆！
<br/>　　伴随着绝无敌一声暴喝，一道金芒自他的体内爆射而出！
<br/>　　这居然是一条金色巨龙！
<br/>　　不过，这条金色巨龙虚影，在空中盘旋了片刻之后，变成了一杆金色长枪，应该绝无敌的本命法宝。
<br/>　　“黄金圣龙枪！！”
<br/>　　“这可是一件真正的仙器啊！”
<br/>　　“能够逼迫绝无敌使出黄金圣龙枪，这小子也算是厉害了！”
<br/>　　……
<br/>　　人群中，顿时有人开始惊呼了起来！
<br/>　　一旁的龙辰见状，也不由得两眼一瞪，脸上露出一抹凝重之色，
<br/>　　很显然，经过两次羞辱之后，绝无敌对叶尘已经彻底动了杀心！
<br/>　　龙辰下意识的想要阻止，可是只迈出一步，却又缓缓退了回去。
<br/>　　之前他一直觉得，眼前这个叫做叶尘的无名小卒，肯定远远不是绝无敌的对手，所以才会出言相帮，一再告诫绝无敌。
<br/>　　可结果呢，反而让绝无敌蒙受羞辱。
<br/>　　而事实上，这个只有元婴境的小子，显然并没有他们想象的那么简单！
<br/>　　‘从他之前的表现来看，纵然实力不如绝无敌，也相差不会太厉害，绝无敌想要杀他，只怕也没有那么简单！不如等他们三招过后，我再出手阻止也不迟！’
<br/>　　打定主意之后，龙辰便又缓缓朝后方退了数步，不再开口阻止绝无敌。
<br/>　　而此时，绝无敌悬空而立，手握黄金圣龙枪，枪尖直指叶尘，冷冷道：
<br/>　　“小子！亮出你的法宝来吧！本公子即便要杀你，也绝对不会占你的便宜！！”
<br/>　　不料，叶尘冷冷一笑，
<br/>　　“凭你，还不配让我动用法宝！！”
<br/>　　虽然明知道，绝无敌是一个十分强悍的对手，可不到万不得已，叶尘还是不想动用灭世魔刀，因为这个目前，是他最大的底牌！
<br/>　　而且，眼前的这些人，应该还不知道他已经得到了灭世魔刀，一旦暴露出来的话，难免会引人觊觎，等进入迷罗星海之后，成为众矢之的。
<br/>　　轰！
<br/>　　叶尘这话一出，全场顿时再次哗然。
<br/>　　面对手持黄金圣龙枪的绝无敌，他居然还拒绝动用法宝！
<br/>　　他是疯了吗？
<br/>　　就连龙辰，也感觉叶尘有些过于狂妄了！
<br/>　　要知道，即便是他，空手对战拿着黄金圣龙枪的绝无敌，也不敢说有百之百的把握，可以战胜他！
<br/>　　毕竟，黄金圣龙枪乃是真正的仙器，足以提升绝无敌近一倍的战斗力！
<br/>　　“好好好！”
<br/>　　绝无敌听到叶尘这话，顿时气得怒极反笑，
<br/>　　“小子，你果然够狂！就算本公子不用法宝，你以为就杀不了你了吗？”
<br/>　　绝无敌说着，竟然又将黄金圣龙枪收回了丹田之中。
<br/>　　他刚刚也是气急了，所以才将本命法宝祭出，倘若叶尘真的不动用法宝，而他即便杀了对方，依旧无法洗刷之前的屈辱，反而会被人说是仗着仙器法宝取胜。
<br/>　　所以，他必须要光明正大、公公正正的战胜叶尘，才能找回自己丢掉的面子！
<br/>　　“小子！今天就让你见识一下，我擎苍宗的无上拳法！”
<br/>　　轰！
<br/>　　随着绝无敌声音落下，只见他双臂微微一颤，顿时幻化出无数道虚影，身上仿佛一下子生出了无数只手臂一般！
<br/>　　“万佛千影！！”
<br/>　　绝无敌一声暴喝，
<br/>　　轰轰轰轰！！
<br/>　　那九天之上，顿时一下子出现了成千上万道拳影，每一道拳影都犹如一座巨山，如同一枚枚巨大的炮弹，朝下方的叶尘爆射而去！
<br/>　　“啊！！”
<br/>　　高台上的围观众人，看到这惊天的一幕，顿时再次纷纷倒退，甚至有的人已经远远退到角落。
<br/>　　而叶尘，则瞬间被那成千上万道硕大的拳影淹没在了其中。
<br/>　　绝无敌的这一招，的确很强！
<br/>　　不但封住了叶尘所有的退路，而且拳势滔滔，绵延不绝，他每挡下一拳，脚下的岩石，就会下陷数十米！
<br/>　　一连数十拳下来，叶尘整个人，已经完全陷入岩石之中，甚至是地底之下，足足千米深！
<br/>　　而且频繁的撞击，让叶尘体内气血翻涌，如果不是他修炼九天焚身诀，肉身强大大大提升，恐怕在对方这一招之下，已经受了不轻的伤势！
<br/>　　而此时，高台上众人，看到那高台中央，硬生生被绝无敌，打出了一个深不见底的深渊，顿时个个满脸骇然之色。
<br/>　　“想不到绝无敌的拳法，也如此精妙！”
<br/>　　“如此强大的一招，即便是返虚真君，恐怕也挡不住吧？”
<br/>　　“那小子，不会已经死了吧？”
<br/>　　……
<br/>　　众人又是惊叹，又是担忧。
<br/>　　倘若叶尘真的死了，那迷罗星海的通道，还会不会开启？难道他们要白跑一趟了？
<br/>　　“绝无敌！你下手也太狠了吧！忘记龙老大是怎么说的了？”
<br/>　　人群中，十大天骄之一的燕狂生，忍不住怒喝了一句。
<br/>　　“不错！若是因为你耽误，让大伙无法进入迷罗星海，这个责任，你担当的起吗？”
<br/>　　风华舞也忍不住呵斥道。
<br/>　　绝无敌冷冷一笑，
<br/>　　“我已经收了几分力，他要是死了，只能怪他自己太弱，这可不能怪我！”
<br/>　　不料，绝无敌这话才刚刚说，一道淡漠的声音，陡然自那深渊之中响起，
<br/>　　“就凭你这不入流的拳法，连给我挠痒都不够，也想杀我？”
<br/>　　声音落下，
<br/>　　轰！
<br/>　　叶尘的身影，已经自深渊中爆射而出！
