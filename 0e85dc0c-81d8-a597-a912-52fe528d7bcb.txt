<br/>　　“都滚开，快让我进去，这所有的宝藏都是我的！”
<br/>　　无数武者，都是发疯了一般，众人纷纷催动遁光，想要一下子进入到密室之中去，争夺宝藏。

<br/>　　那落长恨五人，在打破这禁制的一瞬间，便是纷纷化为遁光，一下子冲入到了密室之中去，在密室的门口，突然又是布下了无数禁制。
<br/>　　将所有众人，除了他们五人之外，全部拦在了外面。
<br/>　　“我们五人打破的禁制，你们这些废物，也敢来凑热闹？”
<br/>　　“敢越界者！杀无赦！”
<br/>　　落长恨冷冷的看着众人，冷喝一声道。
<br/>　　这五人，俨然已经是把持住了这密室的入口，让人根本无法进入！
<br/>　　“什么！他们五个居然打算独吞了这密室中的所有宝藏！”
<br/>　　“我们杀，杀进去，我就不信他们五人，可以挡住我们这么多人！”
<br/>　　无数武者，纷纷出手，他们的实力，虽然不如这五人，但是也都是一方天才。
<br/>　　在这无数宝藏的诱惑之下，哪里还管什么理智，纷纷出手，冲着那落长恨，便是轰击了过去。
