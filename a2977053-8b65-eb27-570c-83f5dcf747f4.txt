        而那晚，马智杰的事件，网络上只字不提，太过平静得异常诡异，好像这件事情从未发生过一样。
<br/>
<br/>        大哥傅夜沉说，警察会查出真相，临海大学闹出命案，临海城上级领导封锁消息，不让在网络上传播负面消息，也可以理解。
<br/>
<br/>        可是，童筱颖想知道结果，警察到底有没有把何明旭抓起来，安慰马智杰的在天之灵。
<br/>
<br/>        实在是一个人在傅家大庄园里待不住了，童筱颖身上塞了手机和钱包，只身离开了傅家大庄园。
<br/>
<br/>        临海大学的校门口，一如既往地人来人往，恍若什么事也没发生过一样。
<br/>
<br/>        今天是周五，一大清早，童筱颖就叩响了马智杰和何明旭曾经所在的宿舍门。
<br/>
<br/>        面对突如其来的敲门声，里面传来了不满地呵斥：“谁啊！一大清早的，敲什么敲？”
<br/>
<br/>        “是我！筱颖！”童筱颖大声喊道。
<br/>
<br/>        没过一会儿，宿舍门便打开了。
<br/>
<br/>        给她开门的人是章海昌。
<br/>
<br/>        章海昌看到童筱颖一脸诧异，方浩博也跟着凑了过来，好奇地问：“小筱颖，你怎么来呢？”
<br/>
<br/>        “我——”
<br/>
<br/>        “你该不会是来找何明旭的吧！”章海昌直接打断了童筱颖的话。
<br/>
<br/>        方浩博补充道：“小筱颖，你还忘不了何明旭那渣男啊？他现在都已经不住寝室了。”
<br/>
<br/>        “那马智杰呢？”童筱颖弱弱地问，因为她发现，他们对她的反应有点儿反常。
<br/>
<br/>        “他被学校开除了啊！”
<br/>
<br/>        章海昌口无遮拦的话刚一出，方浩博一把捂住他的嘴，冲着童筱颖咧嘴一笑：“别听海昌胡说，马智杰只不过是休学回家了。”
<br/>
<br/>        “你们不知道他已经……”童筱颖欲言又止，看着章海昌和方浩博的反应，心里大概猜到他们可能真的不知情，便转开了话题，心情沉闷地问道，“他为什么要休学？”
<br/>
<br/>        “哎呀！这种事情，还是告诉小筱颖吧！”章海昌挪开了方浩博的手，接着滔滔不绝地说道，“那天马智杰生日，我们一起喝了白酒，而且还喝大了。你和何明旭分手后，不是连手机号码都换了吗？我们联系不上你，然后马智杰突然提到你，抱怨何明旭没有好好珍惜你。而何明旭不服气，当着我们的面，骂你是贱货。还说你家负债累累，谁娶了你谁倒霉，说你辍学回家去相亲是为了骗彩礼钱，这事儿闹得他们那个镇上的人全都知道。还说你要嫁给什么糟老头骗取人家的彩礼钱后，又悔婚，结果你的名声太臭，无人敢娶。反正，何明旭那天说了你很多的坏话。马智杰气不过，就和何明旭厮打了起来，我们拉都拉不住。马智杰当天是真的喝多了，还把何明旭打成重伤送进了医院。后来，何明旭依仗着他那个有钱有势的女友林若琴，把马智杰给告了，马智杰的父母赔了钱，才免去了马智杰的牢狱之灾。可结果，学校也把马智杰给开除了。”
<br/>
<br/>        “这是什么时候的事情？”童筱颖听完章海昌的话，鼻子一酸，泪水瞬间溢满眼眶，声音有点儿哽咽地问道。
<br/>
<br/>        方浩博紧接着抢着回答道：“这大概是好几个月前的事情吧！”
<br/>
<br/>        童筱颖这才恍然大悟，那个时候，正是她被告知不是梁家的亲生女儿，被童家换回去后，为童爸童妈还债不得不去相亲的那段时间。
<br/>
<br/>        她确实是在辍学相亲，但她也没有何明旭说得那么犯贱。
<br/>
<br/>        何明旭捏造是非，黑白跌倒，毁她一个女孩子的名声，他到底图什么？
<br/>
<br/>        童筱颖难以想象，自己曾经居然喜欢过这样一个人面兽心的男生！
<br/>
<br/>        “我们也没想到何明旭是这么渣的一个男人，而且，自从马智杰被学校里开除后。我们就再也没跟何明旭有任何来往了！”方浩博接着说。
<br/>
<br/>        章海昌一脸不屑地补充道：“是我们不屑跟他那种人来往！宿舍自家兄弟，只不过是喝酒喝多了，一场打闹，事后彼此握手言和就是。而且，当时，马智杰酒醒后，已经跟何明旭诚恳地道过歉了。何明旭住院的那段时间，马智杰也是鞍前马后地照顾他，他的医药费，还有这个学期的学费和伙食费，马智杰全都出了。而那个何明旭，痊愈出院后，居然听信那个林若琴的话，拿着法检医院开具的伤残鉴定，事后背地里又去法庭上告了马智杰，向马智杰的父母索要了一笔巨额赔偿费。我们真的是对何明旭无语了！”
<br/>
<br/>        “我看他是想钱想疯了！”方浩博谩骂道。
