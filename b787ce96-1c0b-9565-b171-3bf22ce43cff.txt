　　　　这其中，就有几个梨山弟子。叶少阳在人群中一眼看到了当初打过交道的那对情侣，两人披头散发，看上去很是疲惫，不过眼中却是带着一股杀气，与自己对视着。
<br/>　　黎山老母望定了道风，道：“叶少阳已脱困，我也不可能杀他，刚才你为什么不趁机走？”
<br/>　　“我要你跟我走。”道风说道。
<br/>　　这句很容易引起歧义的话，让黎山老母嫣然一笑，“凭什么，凭这个吗？”
<br/>　　黎山老母抬起右手，纤细洁白的手腕上，带着一个翠绿色的手镯。
<br/>　　黎山老母把手镯摘下来，对道风晃了晃，说道：“金刚琢？”
<br/>　　道风隐然意识到什么，没有作声。
<br/>　　黎山老母耸了一下肩膀，说道：“这玉镯，的确是我贴身法器，不过……你上当了，我根本就不是青牛。”
<br/>　　“不可能！”叶少阳率先叫了起来，“你亲口告诉过我，你就是青牛！”
<br/>　　黎山老母一笑，“我没说我就是青牛，我只说你们找到找对了……因为我知道青牛是谁。”
<br/>　　叶少阳等人傻傻地看着她，震惊之余，也是感到了深深的愤怒。
<br/>　　“你骗我，身为阐教金仙，你居然用这种手段，你不嫌丢人吗，脸呢？”叶少阳大声质问。
<br/>　　黎山老母收敛笑容，道：“以我之身份，这么做确实不妥，但为了你诛灭转世鬼童，不得已只好出此下策……”
<br/>　　道风打断她这番悲天悯人讲述，说道：“青牛在哪？”
<br/>　　黎山老母笑而不语。
<br/>　　一道人影，从身后的联盟阵营里走出来，赫然正是那个一直站在苏沫身边的黄脸男子。在跟钟馗说了一句话之后，他一直沉默站在边上，即使崔府君从出现到走人，他也没说过一句话。
<br/>　　黄脸男子抬起左手，手里握着一枚金属环，有脖颈粗细，漆黑透亮，灵光闪烁，一看就不是凡物。
<br/>　　“我，才是你找的青牛祖师。”
<br/>　　众人大惊。
<br/>　　黄脸男子咧开嘴，露出一个比哭还难看的笑容，说道：“我历经轮回，斩却两尸，到这一世，我还有个名字……”
<br/>　　道风默然道：“我知道你是谁了。我找了你很久。”
<br/>　　“不错，我就是李浩然。”
<br/>　　李浩然！
<br/>　　无极天师嫡传内门大弟子，三十岁得道，飞升青冥界，是道门有记载的近代最后一个飞升青冥界的法师。虽然在人间声明不显，也没跟任何门派打过交道，但光凭这份出身和履历，没有人敢怀疑他的实力。
<br/>　　在人间的时候，基本上就差不多已经无敌了，飞升青冥界之后，以轮回入道，斩却两尸，又修炼了多少年，他现在的实力，至少跟他师父无极天师生前是一个等级的，甚至只高不低。
<br/>　　李浩然一只手指着道风，脸上表情不变，眼中却是有一股杀气，直射出来，冲道风说道：“多谢你灭了我在人间的道宗，省却我诸多牵挂。此等大恩，不可不报。”
<br/>　　他说的当然是反话。
<br/>　　道风冷冷道：“你是李浩然，又是青牛祖师，很好，两个都是我要找的人。”
<br/>　　李浩然淡淡一笑。
<br/>　　道风继续说道：“你既然是青牛祖师，就应该知道天命，我已找到白狐朱雀和玄武，只差你。”
<br/>　　李浩然道：“你是想凑齐我们四个，一起灭杀呢，还是想利用我们做与应劫相反的事？”
<br/>　　苏沫狞笑道：“自然是后者，他是转世鬼童，来人间自然是了推动天劫降临，不然他一介普通法师，为何这么执着地关心天劫的事。”
<br/>　　叶少阳本来还想呛她，听她这么说，实在有点没法开口，虽然他相信道风这么做绝不是为了灭世，但他毕竟真的是转世鬼童，自己不管怎么说，别人也不会相信。
<br/>　　摇光仙子抬起手中的长笛，吹奏出一串高亢的音符，等了不一会，谷口方向走来了数十道人影。叶少阳等人惊愕地转头望去，发现这些人里有僧有道，年纪都很老，一个个身上都带着些宗师的气度。
<br/>　　“是你们！”小九惊叹出声。
<br/>　　“他们是谁？”叶少阳急忙问。
<br/>　　“几大宗门的宗主和长老辈分的。”小九环顾过去，“有七八个门派的人都来了，都是阐教和佛宗的门派。”
<br/>　　叶少阳陡然心惊，他是万万没想到，对方居然还有援兵，而且是这么多的强者，登时紧张起来，低声嘱咐自己一干门人：“待会不可厮杀，能跑就跑。”
<br/>　　芮冷玉奇道：“怪了，既然有这么多援手，为什么之前不一起上，否则我们早就没机会了。”
<br/>　　像是为了回答她的疑惑，摇光仙子说道：“这几位是谁，大家都清楚的，他们也想要抵抗天劫出一份力，他们之前没有现身，只因要确定道风转世鬼童的身份。”
<br/>　　小九听她这么说，立刻明白是怎么回事了，冷笑出声。这些宗门的大佬们，她再了解不过，在受到黎山老母和摇光仙子的邀请之后，这些狡猾的家伙的打算是一致的：一定要参与！
<br/>　　假如成功绞杀转世鬼童，功劳就是大家的，说不定还能得到不少现实的好处，如果不来，道理上也过不去。但是在双方局面未定之前，这些家伙是绝不会出手的，如果星宿海这个大的门派被别人灭掉，他们也很乐意看到这个结局。
<br/>　　如果联盟占据优势，有大概率绞杀转世鬼童，他们就会一拥而上，争抢功劳……
<br/>　　现在不管是人数上还是实力上，空界联盟都占据了优势，再加上他们这些人，一起上的话，基本上不会出什么意外了。因此才一个个赶紧现身出来。
<br/>　　先动手的那几个门派的大佬，黎山老母和摇光仙子等，骨子里也是恨透了这帮滑头——眼看着几大门派消耗了死伤这么多弟子，不现身来救，这时候见尘埃落定了，才想要出来分一杯羹，不过恨归恨，也是没有办法，毕竟还要仰仗他们的力量一起来对付敌人。
<br/>　　这时候，星宿海七座山峰的灵力，在长期运行之后，总算是耗得差不多了。北斗七星阵越来越暗淡，最终完全熄灭。
