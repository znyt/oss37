
<br/>　　<br/>　　    “啪”王惊蛰生硬的挤开了尸体的嘴，眼神一瞥之后，他这股气已然力竭要喘不上气了，随后他松开手脚下一蹬，人就迅速朝着水面升了上去。<br/>　　<br/>　　    “哗啦”王惊蛰人从水中钻出来，大口大口的喘了几下，就靠着小船游了过去，翻身就爬到了船上。<br/>　　<br/>　　    陈三岁随即也浮了上来，扒着船沿问道：“你看见什么了？”<br/>　　<br/>　　    “回去再说……”王惊蛰皱眉说道。<br/>　　<br/>　　    半个小时后，两人回到村里，冲了个水换了身干爽的衣服，王惊蛰一直拧着眉头抽着烟。<br/>　　<br/>　　    王惊蛰是想判断，诗远有没有出现尸变的可能，要是有这种情况的话，最首要的特征就是嘴里肯定已经开始长出獠牙了，然后才有逐渐蜕变成僵尸的可能，但掰开尸体的嘴，獠牙倒是没有发现，但一瞥之间令他惊异的是，对方嘴中的舌头上居然出现了舌苔，这是啥情况呢，说白了就是只有人才可能有舌苔，死人的舌头是发紫的，这么一看这明显是违反生物的生理现象了。<br/>　　<br/>　　    如此一来，反倒是把王惊蛰给整懵逼了，完全搞不清头绪了。<br/>　　<br/>　　    地上已经扔了几根烟头，回来半天了王惊蛰和陈三岁都没怎么交流，因为不知道从哪里起话头了。<br/>　　<br/>　　    “王哥，有个细节你注意到没有？”陈三岁忽然问道。<br/>　　<br/>　　    “嗯？”<br/>　　<br/>　　    陈三岁说道：“我之前发现诗远三次有异样，全都是在阴雨天，三个月来平时我也去过多次，但从来都没有发现什么不同，还有刚才问村里那些老人，你记得他们怎么说的吧，发现抚仙湖尸体的时候，那天也同样是在下雨，而且风浪还比较大……”<br/>　　<br/>　　    陈三岁说到这里，王惊蛰顿时倒吸了一口冷气，点头道：“还真是这么一回事，照你这么说，这变故是跟天气有关了？”<br/>　　<br/>　　    “我看看的”陈三岁拿出手机，看了下天气预报，说道：“明天阴天，后天就有雨了，还是中到大雨，到时咱俩再过去一趟”<br/>　　<br/>　　    王惊蛰点头道：“那就等等的，后天在过去”<br/>　　<br/>　　    说实话，抚仙湖这个事把王惊蛰也给难住了，单从表面上看吧，这其实也不是什么麻烦事，因为尸体虽然有变可终究没出什么乱子，抚仙湖附近也没有出现什么天灾人祸，就单单因为陈三岁这个事，也没啥可说的。<br/>　　<br/>　　    可不知道为啥，他心里总有点直敲鼓，隐约觉得抚仙湖下可能还真有一桩惊天的秘密。<br/>　　<br/>　　    两人这边刚研究完，准备后天下雨之后再去湖上，这时王惊蛰的手机忽然响了，他拿起一看，显示的是京城的号码，就接了起来。<br/>　　<br/>　　    他还以为是贺三刀或者常文书谁有事找他呢，但没想到电话里传过来的声音，还有对方一开口说的，就给他惊了一下。<br/>　　<br/>　　    “你在滇西，是么？”<br/>　　<br/>　　    王惊蛰说道：“你查我？”<br/>　　<br/>　　    “我不用刻意查你，韩观山之前没和你说过么，我们这个部门有自己的信息搜集系统，你是我关注的人，你有什么动向我这里自然清楚，不用着重刻意去查，我们只需要得到大数据的汇报就可以了……比如，你乘坐飞机去了滇西”<br/>　　<br/>　　    王惊蛰言语间态度不善的冷笑道：“说白了，不还是有人在留意我么？唉，那我就想问问了，我他么早上起来在哪拉的屎，是啥色的，消化好不好，你们是不是也得派个人去闻闻啊？”<br/>　　<br/>　　    “你消化好不好我确实不知道，但我知道你来滇西之前在咸阳机场候机的时候，吃了一碗面肉丝面，似乎对口味明显不太满意”<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    黄九郎叹了口气说道：“你不知道我们国家有个天眼监控系统么？还有不要对我抱有这么大的敌意，在火车上我已经和你说过了，对你我是打算从朋友做起的，跟韩观山的出发点截然相反”<br/>　　<br/>　　    王惊蛰说道：“那你这次找我，有什么事？”<br/>　　<br/>　　    “你在抚仙湖吧？滇西，除了墓崖村和八拐里坡，也就抚仙湖这个地方能让你跑一趟了”<br/>　　<br/>　　    王惊蛰顿了顿，说道：“对，在这”<br/>　　<br/>　　    “等着我，今天时间有点赶不及了，明天我过去一趟，你在什么位置？”<br/>　　<br/>　　    “澄江县，抚仙湖附近的一个村子里”王惊蛰犹豫了下，还是告诉了他。<br/>　　<br/>　　    “好，我明天过去，直接在那见”黄九郎顿了顿，又接着说道：“在我没去之前，你就在湖边面朝大海春暖花开的赏赏景吧，最好别太深研究那个湖，也最好别尝试着下水，特别是往深水区去”<br/>　　<br/>　　    王惊蛰当即讶异的问道：“你知道这里有什么状况？”<br/>　　<br/>　　    “等我过去了再说吧……”<br/>　　<br/>　　    黄九郎说到这里就挂了电话，王惊蛰脸上顿时露出了疑惑，陈三岁也听见电话里的动静了，诧异的问道：“王哥，谁找的你啊？”<br/>　　<br/>　　    “传说中的有关部门”<br/>　　<br/>　　    黄九郎的突然联系，让王惊蛰暂时放下了去打探抚仙湖的意图，多等一天无所谓，而且明显对方知道的可能还有着某些内幕。<br/>　　<br/>　　    这一天，王惊蛰和陈三岁就没有再动了，在村里等着黄九郎的到来，对方的速度确实也很快，一天过后，第二天的下午，一辆挂着白牌的越野车就开了过来，除了开车的司机以外，车里还有两个人，黄九郎和一个五十岁左右的中年男子。<br/>　　<br/>　　    这两人还是轻车简从来的，王惊蛰还以为黄九郎出场，那不得前簇后拥的呢，没想到两次见到他，这人都相当低调了。<br/>　　<br/>　　    两人下了车，王惊蛰脸色不善的盯着对方，黄九郎背着手大踏步的走来，仿佛也没看见他的表情，来到他俩身前后，他一转身，指着旁边的中年，说道。<br/>　　<br/>　　    “介绍下，国家一级潜水员，耿明先生”<br/>　　<br/>　　    王惊蛰和陈三岁顿时一愣。<br/>　　<br/>　　    黄九郎接着说道：“二十二年前到现在，他也是唯一一个潜入过抚仙湖湖底的人……”
