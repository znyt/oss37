        “筱颍，你要不要喝口水？”傅昼景一边开车，一边用余光注意着童筱颖的动静。
<br/>
<br/>        童筱颖上车后，就一句话也没说过。
<br/>
<br/>        她像个无助的孩子，躲在副驾驶座上，完全沉浸在自己那悲伤而封闭的空间里。
<br/>
<br/>        傅昼景此刻已经心知肚明，童筱颖一定已经相信了他所说的话。
<br/>
<br/>        他说的是事实，没必要骗她。
<br/>
<br/>        时间在一分一秒中过去。
<br/>
<br/>        车子抵达临海城，在入傅家大庄园的那条路上，马路两边停满了各种私家车，就连两边的绿化带上都隔那么几米便挂有一朵白色的纸花。
<br/>
<br/>        前面有交警站岗，似乎是堵车了无法通行，傅昼景不得不靠边停车。
<br/>
<br/>        童筱颖不等傅昼景把车停稳，便直接解了安全带，拉开了安全锁，直接打开车门跳了下去。
<br/>
<br/>        “筱颍，你等等。”傅昼景迫不及待地将车停好，连忙下了车，去追童筱颖。
<br/>
<br/>        傅家大庄园门口，有穿着黑色素衣的男女老少陆陆续续地进进出出。
<br/>
<br/>        两边的石狮上挂着大白花，大门上挂着白绸。
<br/>
<br/>        门口还有黑衣墨镜的保镖看守。
<br/>
<br/>        因为所有门厅都是敞开的，所以就现在大门外，可以直视到里面的灵堂。
<br/>
<br/>        童筱颖刚想进去，却被追上来的傅昼景给拦住。
<br/>
<br/>        他抓住她的手腕，将她拽到了路边的一棵大榕树后。
<br/>
<br/>        “爷爷交代，你只能在外看着，不能踏进大门一步！”傅昼景凝眉劝阻。
<br/>
<br/>        因为大哥的意外身亡，爷爷傅炎廷悲痛住院。
<br/>
<br/>        他去医院探望爷爷的时候，爷爷抓着他的手，气若游丝地警告，不许童筱颖踏进庄园半步，不许童筱颖替傅夜沉守灵。
<br/>
<br/>        只是，他为了让童筱颖跟他走，才谎称是爷爷要他来接她的意思。
<br/>
<br/>        童筱颖看着那张大门，望眼欲穿，嘴唇忍不住地随着上下门牙在“咯嚓”打颤。
<br/>
<br/>        “里面现在守灵的女人是苏漫雪，你进去的话，只会添乱。”傅昼景找了个理由敷衍似的补充道。
<br/>
<br/>        而现在在里面守灵的女人，作为傅家明面上的大少奶奶，确实是苏漫雪。
<br/>
<br/>        童筱颖声音暗哑地质问：“为什么不是我？”
<br/>
<br/>        “不知道。爷爷安排的。”傅昼景皱起了眉头，“筱颍，我先带你去附近的酒店开间房。你在房间里等我，哪都不要去。我安顿好你后，我要去替大哥守灵，大哥的灵柩后天上山。”
<br/>
<br/>        “我自己去就行了。”童筱颖黯然失色地答。
<br/>
<br/>        傅昼景抬起手来，看了看手腕上手表的时间，抿了抿唇：“好！那你自己去！记住了！不要到处乱跑。我会给你电话。”
<br/>
<br/>        “等等！那个女秘书，叫落以柔，我想见她。”童筱颖又叫住了傅昼景。
<br/>
<br/>        傅昼景明白童筱颖的意思，深吸了口气，回答道：“好！我进去后，叫她出来见你。”
<br/>
<br/>        她不信，傅夜沉是为了救落以柔而牺牲了自己。
<br/>
<br/>        明明，那么多人告诫过她，他傅夜沉的命比任何人都重要。
<br/>
<br/>        可他为什么会选择去救落以柔？
<br/>
<br/>        傅昼景进了傅家大庄园没多久，落以柔一身黑色连衣裙，胸戴白花，从大门内走了出来。
<br/>
<br/>        她四下张望了一下，在对面马路的大榕树下看到童筱颖后，凝着眉头，聋拉着脑袋朝童筱颖走了过去。
<br/>
<br/>        当落以柔走近时，童筱颖才看到她双眼已经红肿，面色惨白，就连嘴唇都已经干涸到龟裂。
<br/>
<br/>        “我要一个真相！”童筱颖不跟她拐弯抹角。
<br/>
<br/>        落以柔抿了抿唇，声音嘶哑地回答道：“我们的车子被人做了手脚……”
<br/>
<br/>        “我要的是具体过程。”童筱颖疾言厉色起来。
<br/>
<br/>        落以柔鼻子一酸，豆大的眼泪，从眼眶里潸然而下：“我们到美国的第二天早上，我租了辆豪车回来去接傅总见客户，但是途中，车子的刹车和油门失灵，本来是我在开车，傅总坐在驾驶座后面，得知车子失灵后，傅总说他来开车。傅总让我去后座，解开后座的坐垫绑在身前当缓冲，他会找个合适的地方和时机，命令我跳车。后来，傅总让我跳到了路边的草地。事后，警方调取了监控录像，视频里，傅总似乎是为了不伤及他人，他自己将车开向了海边公路，结果直接冲破了栏杆，连人带车一起掉进了海里，而且，车子在海里爆炸了。我……呜呜——”
<br/>
<br/>        落以柔说着说着，掩面痛哭了起来。
<br/>
<br/>        说到底，这还真像他傅夜沉的作风。
<br/>
<br/>        看着泣不成声的落以柔，童筱颖却只是悲痛地握紧了双拳，暗哑着嗓音，问道：“连华生也在里面吗？”
