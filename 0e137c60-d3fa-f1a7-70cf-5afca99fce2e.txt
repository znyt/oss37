　　第377章
<br/>　　程大雷怔了怔，问道：“你们要去长安？”
<br/>　　奥黛琳娜点点头：“我们要去面见帝国皇帝，告诉他来自诸神的神谕。”
<br/>　　程大雷和徐神机对视一眼，都觉得匪夷所思。合着人家根本看不上自己这琴川之主，目标是明帝那老头儿。
<br/>　　“这个……此地距离长安可有不近的路程，况且现在兵荒马乱的，路上可是不太平。”程大雷好心提醒。
<br/>　　“所以才要请程当家派人护送我们进京。”
<br/>　　程大雷揉了揉眼睛，想要确认是自己没睡醒，还是着奥戴琳娜还迷糊着呢。
<br/>　　“可我为啥要护送你们呢？”
<br/>　　奥黛琳娜用无法理解的眼神看着程大雷：“这可是难得的荣耀，你们如果能够平安送我们到京城，你们将是神圣的护教军，诸神将永远祝福你们。”
<br/>　　凯恩向程大雷解释道：“就像教廷的神圣骑士团一样，拥有封地，供奉，世代传承的荣耀。”
<br/>　　程大雷无话可说，诧异的盯着凯恩。
<br/>　　昨天和凯恩聊天，感觉他听聪明的啊，怎么今天说话，也有些四六不着调的意思。
<br/>　　“既然诸位志不在此，那我也就不强留了。”程大雷站起身道：“徐军师，安排人送客吧。”
<br/>　　奥黛琳娜也站起身：“感谢程当家的相助，诸神将永远祝福你，不知阁下会派多少人护送我们？”
<br/>　　程大雷眨眨眼睛：“我没说要护送你们啊。”
<br/>　　“呃……”这次怔住的是奥黛琳娜，她道：“这无上荣耀，诸神永远的祝福……”
<br/>　　“我不关心。”程大雷顿了顿，忽然郑重道：“尊敬的圣女阁下，其实我也有我的信仰，我信仰伟大的蛤蟆神，黑暗笼罩世界，蛤蟆大神终将吞掉所有黑暗，还世界光明。”
<br/>　　奥黛琳娜和凯恩一脸懵。
<br/>　　徐神机侃侃而谈：“二位怕还是不知道，我们大当家乃是仙人的徒弟，他一身本事都是神仙梦中所授。”
<br/>　　奥黛琳娜愣了很久，最后吐出一句话：“那都是邪神。”
<br/>　　“……”程大雷。
<br/>　　这些教徒的心理的确不能以正常人的思维揣测，当然他们的信仰也不能也不容怀疑。奥黛琳娜一路过来，克服重重困难，为的就是去见帝国皇帝。
<br/>　　据传说，这东方大国，没有自己的信仰，原始淳朴，天真烂漫。这简直就是一片没开垦过的处女地呐，只要自己张开神光的翅膀，他们还不像渴奶的娃娃一样，哇哇叫着扑进母亲的怀抱中。
<br/>　　当然，即便他们千辛万苦可以抵达京城，能不能见着明帝都不好说。就算最终见着明帝，明帝那老头儿……可没有程大雷好说话。
<br/>　　要不说，这些人脑袋不正常呢。
<br/>　　最不正常的就是奥黛琳娜，完全一副四海之内皆我爹的精神头。相比之下，凯恩还算是比较正常的，奔波这么久，大家都是筋疲力尽，已经不愿再走下去。
<br/>　　凯恩用他们的语言对奥黛琳娜说了几句话，奥黛琳娜显得很纠结，最后还是点头答应了，表情简直委屈极了。
<br/>　　“那我们就暂时留在此地。”奥黛琳娜道：“不过我还有几个要求。”
<br/>　　“什么要求啊，说来听听。”程大雷。
<br/>　　“第一、琴川关内要允许我们自由传教；第二、要划出我们单独的生活区域，其他人不许靠近；第三、要为我们建造一座神殿，我们的生活所需，由你们提供。”
<br/>　　“就这些？”程大雷好奇问。
<br/>　　“目前就这些，过后我们会拿出一份正式的文件，约定具体细则。”
<br/>　　“我还以为什么呢，原来就这些啊。”程大雷伸了个懒腰。
<br/>　　“你都答应了？”
<br/>　　“都不答应。”
<br/>　　“呃……”奥黛琳娜。
<br/>　　程大雷无辜的看着她，如果她了解程大雷，就该明白程大雷此刻眼神中的意思：快别扯淡了。
<br/>　　程大雷目前也在思索，让这些人加入蛤蟆寨，究竟是对是错。
<br/>　　“把和珅叫来。”程大雷想了想道。
<br/>　　谈判的事情，还是由和珅这种专业人才来负责。至于奥黛琳娜，程大雷已经没有心情和她聊下去了。
<br/>　　等和珅进来，徐神机和程大雷起身离开。接下来和奥黛琳娜聊的就是和珅，这下她立刻感觉到和珅和程大雷的不同。
<br/>　　和珅礼貌又客气，甚至耐心询问了新教的教义，可紧跟着，和珅就用强大的逻辑能力，绕得奥黛琳娜五迷三道，甚至差点对自己的信仰产生动摇。
<br/>　　程大雷还没离开多久，和珅就来到程大雷的书房。
<br/>　　“谈妥了，这么快？”程大雷。
<br/>　　“也没什么困难，反正他们没有什么地方可以去。”
<br/>　　和珅不愧是谈判的高手，轻而易举就搞掂了奥黛琳娜，将对方的底线一压再压。
<br/>　　具体谈好的细则是，允许他们留下来，蛤蟆寨安排住房，以及一开始的生活保障。但到后来，他们想要在这里生活下去，就必须参加工作。业余时间他们想也传教，也就随他们便。
<br/>　　至于奥黛琳娜要求的神殿、单独生活区域，建造神殿，享受供奉……和珅根本没过脑子就给否定了。
<br/>　　对于这些人的具体工作，由刘悲负责安排，刘悲目前掌握山寨内的人事大权。程大雷也亲自参与，看看这些人都有什么技能。
<br/>　　这帮人约有五十几位，包括的职业有：航海家、冒险家、猎人、皮匠、酿酒师、吟游诗人、药剂师、炼金术师。
<br/>　　程大雷还亲自看了看这魔法师，果然和自己估计得不错，还不如叫做魔术师，几乎就是江湖骗子，可程大雷手里不缺这些人。
<br/>　　在这些职业中，程大雷最看重的职业有：能够绘制地图的冒险家，搭配出各种药剂的药剂师，研究物资构成的炼金术师。
<br/>　　至于奥黛琳娜，程大雷惊讶的发现她的职业竟然是预言家。而且这个预言家的职业不是和魔法师一样扯淡职业，而且真还是有一定功能呢。
<br/>　　根据系统介绍，假若预言家预测你有好运，那么你就可以出去转转，说不得会碰到什么绝世武器，宝藏之类的。而预言家预测你运气不好时，最后在家里窝着不要出去。
<br/>　　这几乎就是一个运气检测仪。
<br/>
