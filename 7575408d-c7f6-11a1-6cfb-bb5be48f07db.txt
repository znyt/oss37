　　蔚蓝的天空，高楼林立，喧嚣的气氛，温暖的阳光，来往的车辆，忙碌的人群......
<br/>　　阴凉的街道上，柔和的清风拂面而来。
<br/>　　看着周围的景象，慕清霄心情轻松异常，仿佛回到母亲怀抱的孩童，紧绷的心弦也渐渐松懈下来。
<br/>　　将气息完全收敛，顺着人流行走在街道上，却没有任何一人察觉到他的存在。
<br/>　　“两个月的时间吗？”
<br/>　　望着手机上的时间，慕清霄眼中闪过一丝精芒，他总算是清楚了一些，越是高等的位面，时间流逝的速度就越快。
<br/>　　比如，他在低武位面待上十年，地球也就过去两个月而已。
<br/>　　但这次不同，他在斗破位面也就待了一年零两个月而已，地球依旧是过去两个月，但比例却有了变化。
<br/>　　若是待上十年的时间，地球岂不是已经过去两年。
<br/>　　只要想到，往后的位面会越来越高，危险系数也会随之增加，慕清霄越是觉得，自己太过渺小。
<br/>　　“时间还早，先去集团看一看吧。”
<br/>　　回过神来，抬头望了一眼矗立在千米外，静海市中心地区，那最为显眼的高楼，脚步轻轻一迈，身影就自原地消失。
<br/>　　几秒钟后，慕清霄就来到集团门口。
<br/>　　只是随意的瞥了眼，保安室中执勤的并非林雷，而是独狼佣兵团的队员，气息倒是比一年前强上许多。
<br/>　　如果当初独狼佣兵团的成员，相当于普通特种兵级别。
<br/>　　那么，现在的独狼佣兵团成员，已经属于精英特种兵级别，两者根本不是一个层次。
<br/>　　根据他们的气息来看，自己不再的时间里，这些佣兵也都在勤奋苦修啊。
<br/>　　说来也是，他们原本就是过着刀剑舔血的生活。
<br/>　　现在生活虽安稳了下来，但强大的实力，对骨子里好强的佣兵们来说，绝对是致命的诱惑。
<br/>　　放弃如此好的修炼机会，绝对是傻子的行为。
<br/>　　独狼佣兵团也算是辉煌集团的一道安全保障，慕清霄也没有吝啬，当初直接将《易筋经》传授给了林雷。
<br/>　　当然，传授的不过只是，简单的段体方式，能够修炼出内力的心法，他并没有传授。
<br/>　　林雷此人也算是有情有义，知道照顾底下的弟兄。
<br/>　　林雷他们不在保安室执勤，慕清霄就没有惊动执勤人员，身影一闪，就向集团内部掠去。
<br/>　　“董......董事长，早......早上好。”
<br/>　　前脚刚踏进集团，门口的迎宾小姐，神色一滞，回过神来，连忙鞠躬行礼，结结巴巴的说道。
<br/>　　“嗯。”
<br/>　　瞥了眼礼貌的迎宾小姐，慕清霄微微点头，还是比较满意的，随后直接向电梯走去。
<br/>　　由于现在早就过了上班时间，电梯里空荡荡的，按下楼层按钮后，电梯就向集团大厦顶部窜去。
<br/>　　很快，电梯就在二十六层，总裁专用楼层停下。
<br/>　　集团大厦内部走廊，基本每天都有清洁人员打扫，所以里面极其的整洁，走廊上弥漫着淡淡清香，使人身心舒畅。
<br/>　　细微的脚步声，在走廊上响起，转过拐角，向前走了几步，慕清霄就出现在总裁办公室门口。
<br/>　　“咚咚...!”
<br/>　　敲门声轻轻响起，旋即，办公室内就响起女子的声音：“请进。”
<br/>　　推开房门，一股幽香就钻进鼻孔中，慕清霄顿时觉得有些痒痒的。
<br/>　　抬头望去，映入眼帘的是，端坐在办公桌前，阅览着文件，雍容丰满的佳人。
<br/>　　完美无瑕的容颜，不经意间透着一抹妖精般的妩媚，青丝随意的洒在香肩，直达腰际，释放着让人口干舌燥的诱惑。
<br/>　　纤纤玉指捋了捋额前的青丝，忽然动作一滞，望着站在门口的青年，美眸闪烁着温柔的光芒。
<br/>　　“回来了？”
<br/>　　清脆的声音在耳边响起，温柔无比，如同关心丈夫的妻子，正是这股温柔，令得慕清霄找到了家的感觉。
<br/>　　“嗯，回来了。”
<br/>　　说着，慕清霄就走上前去，来到办公桌前，在她身上打量了一圈，道：“是不是胖了一点。”
<br/>　　话语刚落，欧阳紫嫣眼角一抽，攥起粉拳，就在他胸前锤了一下，道：“你才胖了，你全家都胖了！”
<br/>　　“嘿嘿。”
<br/>　　慕清霄趁机抓住她的手腕，目光盯着她胸前，那两座高低起伏的峰峦，坏笑一声，道：“胖没胖，只有摸过才知道。”
<br/>　　“哼，依旧没个正经，在外面又骗回来几个？”
<br/>　　“咳咳，这说的真难听。”
<br/>　　干咳两声以示尴尬，慕清霄将她一把从办公椅上抱起，惹的她一阵惊呼。
<br/>　　来到沙发上，欧阳紫嫣莲藕般的玉臂搂住他的脖颈，完美无瑕的脸颊上，涌上一抹酡红，美眸扑闪扑闪的。
<br/>　　“想什么呢，妖精？”
<br/>　　“你......你不是想做那事吗?”
<br/>　　话语刚落，慕清霄差点被呛死，眼角抽了抽，道：“你丈夫，我是那样的人吗？而且现在可是白天。”
<br/>　　“你那样起来不是人！而且，你以前不也这么荒唐吗？”
<br/>　　盯着慕清霄，欧阳紫嫣斩钉截铁的说道，美眸中闪过一抹幽怨，早就将他性格摸了个底朝天。
<br/>　　“那事，晚上再说，我可是给你们带了不少好东西。”
<br/>　　说着，慕清霄就自储存空间中，取出一枚天蓝色的戒指，戒指不算多华贵，却也有些卖相。
<br/>　　“戒指？”看到戒指，欧阳紫嫣拿在手中，脸颊上闪过一抹欣喜。
<br/>　　毕竟，老夫老妻三年了，慕清霄可是连结婚戒指都没给她们买一枚，神经真是大条的可以。
<br/>　　慕清霄温柔一笑，道：“这可不是普通戒指，按照我说的试试看......”
<br/>　　旋即，他便将使用方法与步骤，告诉了欧阳紫嫣。
<br/>　　按照慕清霄说的方法，欧阳紫嫣轻轻划破手指，逼出一缕鲜血，滴落在戒指上，随后，奇异的一幕就出现了。
<br/>　　只见，鲜血落在戒指上的瞬间，下一秒就融入戒指中。
<br/>　　“好神奇啊！”
<br/>　　几秒钟后，欧阳紫嫣脸上涌上激动之色。
<br/>　　因为，她能清晰的感觉到，自己与手中的戒指，有了一股莫名的联系，心念一动，便出现一个储存空间。
<br/>　　“这是纳戒，算是中等层次，其中空间足以装下一幢小型别墅，喜欢吗？”
<br/>　　“算你有点良心。”
<br/>　　虽然喜欢的不得了，欧阳紫嫣美眸却是瞪了他一眼，然后就将戒指戴上，心里美滋滋的。
