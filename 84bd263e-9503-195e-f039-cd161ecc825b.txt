　　从大唐方向过来的商队是一个由上百商旅自由组合的商队，这是为了躲避一些小股的马贼侵扰，必竟商队大了，护卫也多，团结起来应付一些小股的马贼问题并不大。
<br/>　　但是当商到遇到西域最大的一股马贼时，除了自求多福之外，其它的一切准备都变成了徒劳。
<br/>　　“尊敬的头领，请问您有什么事情么？”商队被薛仁贵带人拦下来之后，一位众人推选出来的负责人哆嗦着走出队伍，颤抖着来到薛仁贵的马前。
<br/>　　“老家伙，你拿老子开涮是吧？”高展一提马缰，越众而出，围着商队负责人转了两圈之后嗤声问道。
<br/>　　“不敢，不敢啊！”负责人吓的连连摆手，似乎知道自己问错话了。这特么可是马贼啊，而且是拿着手弩的马贼，拦下商队总不会是因为迷路找不到家。
<br/>　　“把他们都关起来，让他们写清楚自己都带着什么货。”薛仁贵给高展打了一个眼色，示意他适可而止。
<br/>　　“老大，这帮家伙滑着呢，让他们自己写一定会有所隐瞒。”高展手中横刀搭在商队负责人的脖子上，略带着一些不满说道：“依我看不如全都杀了，货让崽子们全都拉走，省心省事。”
<br/>　　“杀鸡取卵的买卖还是少做，人杀光了下次就没人走这条路了，到时候让手下弟兄都去喝风么？”
<br/>　　“是是是，这位好汉爷，小的一定如实写，一定如实写！”商队负责人听了薛仁贵的话，立刻像是抓住了救命稻草一般，连连拱手，示意自己不会作假。
<br/>　　“老小子，算你命大，不过你最好让其他人也老老实实的，否则，只要抓到一个敢隐瞒的，老子不介意屠了你们所有人，滚！”高展虎着一张脸，哼了一声，装成火气很大的样子，厉声威胁着，同时还用刀背在商队负责人的脖子上砍了一下。
<br/>　　“是是是，小人这就去，这就去。”商队负责人狠狠一缩脖子，连滚带爬的向商队跑去，同时口中胡乱答应声。
<br/>　　片刻之后，负责人已经跑回到商队当中，随着他将薛仁贵的要求一说，人群中立刻就掀起一片喧哗之声，但是很快声音就又低了下去，只有商队负责人的声音还在响着：“大家忍忍吧，老夫经常走这条路，知道这伙马贼，只要按照他们说的办了，大家还是不会有太大损失的。”
<br/>　　“你说的轻松，如果他们看我们的货眼红起了杀人越货的心思呢？”一个声音在人群中反驳道。
<br/>　　“老夫不敢保证他们一定不会杀人，但是按他们说的做我们还有一线生机，否则的话，你们自己看看这周围，再看看那边的山上，后果还用我说么？”商队负责人用手向四下里指了一指，示意围在一起的那些商贩向四下看看。
<br/>　　“就算是他们人多，我们人也不少啊！”又一个声音嘀咕道。
<br/>　　“不少？我们有多少护卫？三千够不够？你再看看周围，至少有五千人马，那边的山上影影绰绰还不知道藏了多少人。真打起来怕是所有人都加起到起也不够人家一刻钟杀的。”一个中年人主动站到了商队负责人的一面。
<br/>　　“是啊，这些马贼装备这么好，很可能就是西域哪个国家的军队，护卫们就是再厉害也打不过军队吧？”又一个人站到了负责人的一面。
<br/>　　“写就写吧，至少可以赌一把不是么？”
<br/>　　“是啊，总好过马上就死，而且，护卫们也不一定就会真的和那些马贼对上。”
<br/>　　一刻钟后，商队里所有人终于达成了一致，在马贼强大的武力震慑下选则顺从。
<br/>　　接着，连商人带护卫，在一群马贼的押送下转道上路边的一座小山，而在山的另一侧，他们见到再多的马贼正在扎营。
<br/>　　乌泱泱的一群马贼让商队所有人看的心惊胆颤，暗暗庆幸刚刚没有拼死反抗，否则现在只怕已经被万马踏成肉泥了吧。
<br/>　　随着一路前行，那些忙着扎营的马贼竟然没有发出一丝的声音，除了偶尔投射过来桀骜不驯的一瞥之外，没有任何一个人理会他们，这更加让商队众人肯定了这批人是军队的想法。
<br/>　　商队最后被带到一片空地之中安顿，马贼留下百十个人远远的看着他们，其他人就各自归位，各行其是，不再来理会他们。
<br/>　　不过那些商人却统统被带走，去了不远处的一个帐篷，在那里他们需要写下他们自己所带的货物都有什么东西。
<br/>　　而商队的负责人，则被带入了营地中最大的一顶帐篷。
<br/>　　“高展，你个驴日的，你特么把老子的脖子砸肿了！”刚刚一进帐篷，商队的负责人声音就变成了一个年轻人，一把撸下头上的帽子往地上一摔，就向高展扑了过去。
<br/>　　“老子不也是为了演的像一点么，你见过马贼心平气和说话的么？”高展在窜到薛仁贵身后，露出半个脑袋，抗声说道。
<br/>　　“那你咋不用刀刃砍呢，你砍死老子正好立威。”围着薛仁贵转了几圈，商队负责人见抓不到高展，也就顺势放弃，扯过老薛递过来的水袋仰头咕咕的灌了几口。
<br/>　　“怎么样，路上还顺利吧？”薛仁贵接过递回来的水袋，指指不远处的小凳，示意商队负责人坐下说。
<br/>　　“除了遇到你们这股马贼，一切都很顺利。”商队负责人大咧咧的坐下，就着边上一个马贼送上来的水盆将脸上的易容洗了下去。
<br/>　　等他抬头将脸擦干净之后，露出的是竟然是吴辰那张臭脸。
<br/>　　薛仁贵从贴身的衣服里拿出半块牌子，递向吴辰，同时问道：“这次你怎么亲自来了？”
<br/>　　吴辰同样拿出半块牌子，与薛仁贵手中的牌子对了一下，之后才满脸鄙夷的说道：“当然是太子殿下有事情交待，否则你当老子愿意来你这鸟不拉屎的破地方？”
<br/>　　“哦？殿下有任务交待下来？快说说，是不是要进攻吐蕃了？”一边的高展听到李承乾那边有任务交待，立刻来了兴致。
<br/>　　吴辰脸上的鄙夷变成了坏笑，兴灾乐祸的说道：“吐蕃的事殿下没提，只是让我给你们带五十个人来，而且是五十个酸丁。”
