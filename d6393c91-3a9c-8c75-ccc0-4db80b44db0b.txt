        季思妍毫不犹豫地应道：“好！我给你两百万，你签了这些文件以及离婚协议书后，立马给我离开阿景的身边，离开海滨城。”
<br/>
<br/>        “成交！”苏漫雪扯了下红唇，精致而娇俏的脸上露出了几分不屑。
<br/>
<br/>        要论这办事的速度，没有人能比她季思妍更快了。
<br/>
<br/>        离婚协议书是季思妍叫了一个律师过来临时草拟的，修订成正稿后，打印了出来。
<br/>
<br/>        苏漫雪把该签的字，都签了后，季思妍当即就给苏漫雪转了两百万。
<br/>
<br/>        虽然用的是童筱颖的银行卡，但是苏漫雪先前在银行里已经把密码给改了，现在完全为她所用。
<br/>
<br/>        另一边，临海城，依山别苑里。
<br/>
<br/>        而中午的时候，刘管家并未回来吃饭，童筱颖只好自己动手做饭吃，但是当她把饭菜做好端上桌，这位冷面女保镖却不肯吃她做的饭菜。
<br/>
<br/>        难道是怕她在饭菜里下毒吗？！
<br/>
<br/>        作为一个监视者，敬业到这样的地步，童筱颖也不得不佩服。
<br/>
<br/>        既然这女保镖不吃，那她就一个人吃好了。
<br/>
<br/>        虽然被苏漫雪偷换了身份，令她很生气很无助很悲伤，但是日子，她还得过。只有让自己好好的，她才有从这逃出去，去揭穿苏漫雪的机会。
<br/>
<br/>        直到下午，刘管家才从外回来，而那女保镖只吃刘管家做的饭菜。
<br/>
<br/>        童筱颖无言以对，但心里却在盘算着，晚上等他们俩都睡着了，她就从这里逃出去。
<br/>
<br/>        谁知，这才晚上七点，她便困意来袭，更在不知不觉中，见床就倒，呼呼睡起了大觉。

<br/>        滨江城，傅氏集团旗下地利置业大厦。
<br/>
<br/>        办公室里的傅昼景，刚刚遣了那几个高层出去，仰躺在自己的老板椅闭着眼睛不想说话时，助理赵斌就正好叩响了他办公室的大门。
<br/>
<br/>        “进来。”
<br/>
<br/>        “傅总，季小姐……”
<br/>
<br/>        “打发她走。”傅昼景头也没抬地继续闭目靠在那里，工作上的事已经足够烦心，他确实是不想再浪费私人时间的一分一秒。
<br/>
<br/>        赵斌微微错愕了一下，讷讷地回答道：“她已经进来了。”
<br/>
<br/>        “傅总，虽然我已经不在地利置业工作了，但您也用不着，连电话都不接我的吧！”季思妍甩了甩柔顺的秀发，娇俏地说道。
<br/>
<br/>        她今天穿着一身合体剪裁的连身宝蓝色短裙，大波浪的长发被高高束起，左手手腕上一只纯黑限量款香包，脚上同样一双纯黑色凉皮的鱼嘴高跟鞋。
<br/>
<br/>        赵斌见状，欠身退了出去。
<br/>
<br/>        傅昼景闻言，似乎是想到了什么，皱了皱眉头，冰冷地说道：“现在是关键时期，你难道不知道要避嫌吗？”
<br/>
<br/>        他已经被上头的人暗中调查这件事情，是大哥傅夜沉给他“通风报信”。
<br/>
<br/>        大哥果然是向着他的，但是他不明白，明明他暗中“拿走”了傅氏集团的业务，为什么大哥不但不怪他，还要帮着他呢？
<br/>
<br/>        “你就这么怕吗？而且，你突然跟我断绝那方面的关系，你就不怕我到头来反咬你一口呀？”季思妍扬眉说道，有一股威胁的味道在里边。
<br/>
<br/>        傅昼景眯了眯深邃的眼眸，遮住了里边的犀利跟杀气，冷声说道：“有话直说，没必要在这跟我拐弯抹角。”
<br/>
<br/>        “阿景，这是筱颖让我转交给你的离婚协议书！”季思妍勾唇一笑，从自己的香包里拿出一叠资料，走到傅昼景的办公桌，挺了挺她那傲人的胸脯，俏声说道。

<br/>        傅昼景接过季思妍递过来的资料，从头到尾，一字不落地翻看了一遍后，顿时恼火地将这些资料甩到了办公桌上。

<br/>        就在季思妍感到洋洋得意的时候，傅昼景却怒吼一声：“季思妍，你以为，你随便找个人代签，就能糊弄我吗？你当我是三岁小孩？”
