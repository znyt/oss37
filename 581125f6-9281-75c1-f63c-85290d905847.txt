        &lt;h3&gt;第三卷第一百七十七章  江俞轩别样的婚礼&lt;/h3&gt;
<br/>
<br/>        江俞轩连轴转了两天，赶着两个公司开年会，然后随着张倩楠拜访了岳父。
<br/>
<br/>        张家老一辈兄弟三人，大房就是张函他们家（爷爷几年前已经走了），二房的老人不闻世事（因为二房的不闻世事，他们张家才过得艰难），三房就是张倩楠家，张倩楠家里就她一个女孩子，张倩楠既有女孩子的温婉也有男孩子的铮铮铁骨。
<br/>
<br/>        他们张家三个孩子，另外一个就是落家那个哥哥落白，张函比她大了，也是从小品学兼优，她是哥哥的小迷妹，俩个人感情很好，所以就一直跟着哥哥的步伐。
<br/>
<br/>        江俞轩提前和张倩楠商量，节前要去拜访张家的人，现在爷爷辈就是二房的还健在，到了张家是要去拜访老爷爷的。
<br/>
<br/>        江俞轩和张倩楠这种柏拉图式的恋爱已经进行了很多年了，两家大人也是知晓，只是他们之间都是各种忙，走动的也不是很多。
<br/>
<br/>        江俞轩也是和江爸爸商量了一下，决定再去和张家商量商量，定个日子结婚。
<br/>
<br/>        江爸爸从广州飞到北京，随着儿子江俞轩去张家提亲。
<br/>
<br/>        这是江爸爸第二次到张家，第一次也是因为儿子的婚礼，父子俩人同样的感觉心里有那么一丝愧疚，当初定下的婚期，因为工作的原因，取消了婚礼，不是说他们江家爱财，而是当时手头上的项目确实很多，基本都进入白热化阶段，光江俞轩负责西城开发的项目都是拿多少个亿来说事。
<br/>
<br/>        江爸爸：“轩轩，你这臭小子，放出去了连老爸都不要了啊。”
<br/>
<br/>        江俞轩：“我跑得再远还不是翻不出你这如来佛的五指山。”
<br/>
<br/>        江爸爸：“为了工作，取消了你们的婚礼是不是心里不舒服？”
<br/>
<br/>        江俞轩：“有啥不舒服的？我和我老婆经常在一起。”
<br/>
<br/>        江爸爸：“.......”莫名奇妙让他这个老爷子感觉有点不好意思。
<br/>
<br/>        江爸爸：“我以为你一直在陈子昂的公司是因为陈子昂，所以我也就没有催促过你，原来是我看走了眼。”
<br/>
<br/>        江俞轩：“陈子昂太傻了。”被爸爸这样一问，江俞轩突然就觉得心里有点酸涩的感觉。
<br/>
<br/>        “爸，假如我那个时候追求陈子昂你会愿意吗？”
<br/>
<br/>        江爸爸看了一眼江俞轩：“如果陈子昂是单身应该能够接受。”
<br/>
<br/>        江俞轩看了一眼疾驰而过的街道，他在北京呆了很多年，这里有他青春的印记，也有他随着陈子昂走过的足迹。
<br/>
<br/>        陈子昂的模样在他的脑海里清晰，倔强的陈子昂、强势的陈子昂、喝醉了的陈子昂、温柔的陈子昂、哭着的陈子昂、笑着的陈子昂、还有那流着哈喇子酣睡的陈子昂、那个吧唧他脸的陈子昂，那个憨憨唧唧的陈子昂，江俞轩的心里越发的酸涩，他闭上了眼。
<br/>
<br/>        “轩轩，人生有很多美好的事情，也有很多的迫不得已，还有很多的错过，我们不能都一一的抓在手中，我们只能也只有找到合适自己的拥有他珍惜他。”江爸爸语重心长的说。
<br/>
<br/>        “嗯。”江俞轩是一个理智的人，将自己的感情能很好的分割，他也知道他自己需要什么，没有结果的他永远都不会去设想。
<br/>
<br/>        到达张倩楠的家里，张函家里的人都在，看来大家都很注重张倩楠的这一次婚礼，几家人之前是见过一次面的，这一次，他们还要去拜访二房的爷爷，现在家里的老人就是二房的爷爷和奶奶健在，小辈的婚礼是需要得到他们的祝福的。
<br/>
<br/>        离去二房的家里还有一段距离，几个长辈坐一辆车，江俞轩和张倩楠、张函坐一辆车。
<br/>
<br/>        张函开车，江俞轩和张倩楠坐在后排，张倩楠将手放在了江俞轩的手中，俩个人十指相扣，江俞轩捏了捏张倩楠的手，给了张倩楠一个微笑。
<br/>
<br/>        “二爷以前是很厉害的，只是二叔去了之后二爷才什么都不管的，这次你去见见二爷。”张函对江俞轩说。
<br/>
<br/>        江俞轩知道张家之前最强的就是二房，到现在虽然二房不管事，余威还在，心里倒是有点忐忑，毕竟是第一次见那样的一个存在。
<br/>
<br/>        “爷爷很好的，也很爱我，你放心。”张倩楠似乎能感觉到江俞轩的心。
<br/>
<br/>        “好的。”
<br/>
<br/>        张函：“俞轩，我怕是得将孩子送走了。”
<br/>
<br/>        江俞轩和张倩楠一起问：“为什么？”
<br/>
<br/>        张函：“冯睿已经将孩子的消息透露给蔡少芬和我妈了，我对我妈承认了孩子是我的。”
<br/>
<br/>        江俞轩：“那你就这样承认了是不是给你和陈子昂都带来了麻烦？”
<br/>
<br/>        “如果我不承认，会给你带来麻烦，你们俩个现在是公众人物，又牵扯到我们家倩楠。如果被人利用，后果不堪设想。”
<br/>
<br/>        江俞轩沉默了一会：“我当时只是为了陈子昂打抱不平，所以就那样怼了冯睿，没有想到给你们添麻烦了。”
<br/>
<br/>        “该来的总是会来，躲也躲不掉，陈子昂和孩子的事情迟早都会曝光，只是我没有想到会这么快。”
<br/>
<br/>        “那你准备怎么办？如果让嫂子知道了怕是会不高兴的。”张倩楠也忧心的问。
<br/>
<br/>        “我想让子昂暂时将孩子送出国，今天我就给落白说一声。落家在国外有好几个庄园，让孩子过去我还能放心。”
<br/>
<br/>        “陈子昂上一次也和我们说过这个事情，大哥还说他帮忙问问他弟弟。”江俞轩对张函说，心里确想着，陈子昂和张函不愧曾经相爱过，俩个人的想法出奇的一致。
<br/>
<br/>        张函：“不用了，落白其实就是二叔的孩子，你今天也能见到，他也知道我和陈子昂之间的事情，让他出面比什么都好。”
<br/>
<br/>        “就让函哥自己解决吧。”张倩楠看着江俞轩。
<br/>
<br/>        “好！”
<br/>
<br/>        几个人到了张家二房爷爷家，江俞轩看了一下，如果没有张函他们，外人是进不去的，爷爷身体健康完全看不出来是八十多岁的人了，奶奶更是身材苗条，对，没错，是苗条，只是偶尔好像思维有些混乱。
<br/>
<br/>        几个人坐在一起寒暄了一阵，转入了正题。
<br/>
<br/>        张倩楠的父亲对二爷爷说：“二伯二妈，倩楠也有这么大的年龄，明年想结婚，俩个孩子想征得您的同意和祝福。”
<br/>
<br/>        “明年准备什么时候结婚？”二爷看着张倩楠的父亲。
<br/>
<br/>        “俩个孩子准备五一结婚。”
<br/>
<br/>        “明年下半年结婚吧，现在事情不明朗，你们，都低调点，另外结婚也没有必要铺张浪费。”大家听了爷爷的话都不吭气了。
<br/>
<br/>        江俞轩和张倩楠年龄已经不小了，俩个人拖拖拉拉好多年。
<br/>
<br/>        “你们不是结婚证都领了吗？不就是一个仪式的问题？来，一人给我端一杯茶，晚上在家里一起吃顿饭吧，你们就回去过日子去，等明年下半年挑个好日子再举行婚礼吧！”
<br/>
<br/>        大家都觉得不可思议，尤其是江俞轩父子，江俞轩刚想反驳，张沛尧和张沛毓同时说：“谢谢二叔，谢谢二叔。”
<br/>
<br/>        江俞轩和张倩楠按照古老的仪式给二爷和奶奶端了茶，就算是举行了结婚仪式，从张家出来，心里总是感觉不对劲。
<br/>
<br/>        张倩楠说：“落家和吴家的联姻会引起什么样的效应我们都不知道，二爷大概担心我们吧，毕竟落家和我们张家也有关系，俞轩，我觉得这样也很好。”
<br/>
<br/>        “可是，倩楠，现在是二十一世纪啊，我.......”江俞轩心里很不是滋味。
<br/>
<br/>        “明天，我和你一起回广州！”张倩楠坚定的说，无论此生如何，她会伴着江俞轩的脚步和他一起前进。
<br/>
<br/>        人生无常，江俞轩想给张倩楠一个美好的婚礼的想法只是成了一个无法完成的梦想。
