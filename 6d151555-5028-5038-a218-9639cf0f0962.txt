        “妈，怎么连你也说这种话？”傅夜沉不悦地蹙起了眉头。
<br/>
<br/>        冷夫人苦口婆心地解释道：“妈不跟你说两家话，小景那孩子一直是妈妈心头的一根刺。你若无后，傅氏集团不可能交给小景去继承，也更加不可能交给小景他儿子。你妈我现在很想抱孙子，更别提你爷爷想抱曾孙。妈知道你深爱筱颖，但找女人代孕，只是做试管婴儿，并不影响你和筱颖的夫妻感情。代孕所生的孩子，我和你爷爷会对外声称是筱颖跟你的孩子。筱颖向来懂事，我想她不会介意。”
<br/>
<br/>        “但我介意。”傅夜沉斩钉截铁地说。
<br/>
<br/>        冷夫人见自己劝不动傅夜沉，便不再多说了。
<br/>
<br/>        两人的话题不欢而散。
<br/>
<br/>        但是，冷夫人并未就此罢手。
<br/>
<br/>        她想着，既然劝不动儿子，那就去劝儿媳妇。
<br/>
<br/>        所以，本来在国外旅游的童筱颖，在接到冷夫人的一通电话后，提前回国了。
<br/>
<br/>        于童筱颖来说，这是她名义上的“第一次”见婆婆，见面礼自然是少不了，她专挑了一些高档的护肤品以及首饰，打包成大大小小的礼物盒带去给冷夫人。
<br/>
<br/>        临海城，傅家大庄园。
<br/>
<br/>        傅夜沉前天才离开，冷夫人便把童筱颖给召了回来。
<br/>
<br/>        童筱颖下了私家车，在管家的引领下去了冷夫人的房间。
<br/>
<br/>        冷夫人早已让女佣备好了茶，等着童筱颖来见自己。
<br/>
<br/>        童筱颖跨过门槛，走进屋内时，看到冷夫人在高椅上坐着，落落大方地问候了一声：“婆婆好！”
<br/>
<br/>        起初，在电话里的时候，冷夫人自称是她婆婆的时候，她一时间还未反应过来，直到冷夫人又换了个自我介绍的方式，说自己是傅夜沉的母亲，童筱颖才恍然大悟地掩饰自己的尴尬。
<br/>
<br/>        有老公的这件事情，她都还没适应过来，就要适应自己还有婆婆，她心里虽然觉得难为情，但也学会了去调整顺其自然的心态。
<br/>
<br/>        “筱颖，你这段日子，都去哪儿玩呢？”冷夫人向童筱颖招了招手，示意她在自己的身边坐下，“快来说给我听听。”
<br/>
<br/>        童筱颖将手里的大小礼盒袋，放在了一旁的方桌上，然后坐到了冷夫人的身边：“我暂时只去了威尼斯。”
<br/>
<br/>        她还只游玩到一半，就被这位“婆婆大人”给召回来了。
<br/>
<br/>        “威尼斯是水城，那里一定很好玩。我都已经年过半百了，却还从来没有出过国。”冷夫人微笑着说。
<br/>
<br/>        童筱颖随即挽着冷夫人的手臂，微微一笑：“要不，婆婆和我搭个伴，我们一起去环游世界。”
<br/>
<br/>        其实，环游世界这件事情，是她当时在实验室里苏醒过来后，在电视里看到的。她觉得外面的世界很大也很美，她想出去看看，但是实验室里那群医生不允许，特别是那个连老头。
<br/>
<br/>        不知道为什么，他们越是想将她困在实验室里，她就越是想出去，向往外面的世界。
<br/>
<br/>        冷夫人按住童筱颖的手，轻轻地拍了拍她的手背，苦笑：“我这年纪，就不出去折腾了。倒是你，趁着现在还年轻，多出去玩玩，看看外面的世界，是好事。只不过……”
<br/>
<br/>        “只不过什么？”童筱颖眨巴着眼睛看着冷夫人。
<br/>
<br/>        冷夫人接着说道：“我知道，你失去了以前的记忆。所以，你也一定不记得自己不能生育的这件事情。”
<br/>
<br/>        “我不能生育？”童筱颖诧异，这事她还真不知道。
<br/>
<br/>        冷夫人看了童筱颖一眼，忧心忡忡地接着说：“傅氏集团的未来，必须交到小沉的孩子手中。如果，你不能……”
<br/>
<br/>        “婆婆是要我跟大哥离婚吗？”童筱颖不假思索地就问出了这句话来。
<br/>
<br/>        冷夫人微微摇头，语重心长地说道：“你们不用离婚。我只是希望，你能接受代孕。”
<br/>
<br/>        “婆婆的意思是，要找另外一个女人，给大哥生孩子？”童筱颖觉得有些尴尬了。
<br/>
<br/>        “做试管婴儿，不会影响到你和小沉的。”冷夫人接着说道。
<br/>
<br/>        童筱颖定了定神，心里却总觉得这事挺别扭。
<br/>
<br/>        “筱颖，我希望你能体谅。”冷夫人深沉道。
<br/>
<br/>        “我觉得代孕这事，对那个代孕的女人不太公平，更何况，代孕不合法吧？如果我真不能生，傅家又需要继承人的话，我可以选择离婚。”童筱颖经过深思熟虑之后，十分淡定地回答了冷夫人的话。
<br/>
<br/>        冷夫人一脸诧异地看着童筱颖。
<br/>
<br/>        童筱颖嬉皮笑脸地起身，将自己买的礼物提到了冷夫人的面前：“婆婆还是先看看我给您买了什么好东西，希望您会喜欢。”
