        爱宝贝老婆？！
<br/>
<br/>        难道是童筱颖！
<br/>
<br/>        落以柔心里十分确定，毕竟这是连华生今早交给她保管的，属于傅夜沉的手机。
<br/>
<br/>        “滋滋滋——”
<br/>
<br/>        手机一直在振动。
<br/>
<br/>        落以柔一直在犹豫要不要接听。
<br/>
<br/>        “滋滋滋——”
<br/>
<br/>        手机还在持续振动着。
<br/>
<br/>        看样子，童筱颖是打算等到这边接电话为止。
<br/>
<br/>        落以柔滑动到接听键，将手机贴到了耳边。
<br/>
<br/>        “大哥，你……身体好些了吗？”电话接通后，童筱颖弱弱地问道。
<br/>
<br/>        今早她有给刘婶打电话，刘婶说大少爷还在医院里住院没回来，还说，如果她真关心大少爷，就应该主动给大少爷打电话。
<br/>
<br/>        她听得出，刘婶在生她的气，怪她不该把傅夜沉气到吐血。
<br/>
<br/>        童筱颖跟刘婶打完电话后，犹豫了许久，才鼓足勇气给傅夜沉打了这通电话。
<br/>
<br/>        “傅总现在还在休息，你晚点再打过来吧！”落以柔没好气地说。
<br/>
<br/>        童筱颖听到是一个女人的声音，脑子里第一反应，想到的人就是落以柔。
<br/>
<br/>        “你怎么会拿着大哥的手机？”童筱颖纳闷道。
<br/>
<br/>        落以柔暗自得意地回答：“是连医生让我替傅总保管的，连医生说，不希望任何打扰傅总休息。”
<br/>
<br/>        “那你告诉我，大哥所住院的医院以及病房号。我会赶回来去医院里照顾他……”
<br/>
<br/>        “童小姐，你是打算脚踏两条船吗？为了傅总的身心健康，我劝你还是不要再出现在傅总的面前比较好。”落以柔毫不客气地打断了童筱颖的话，说完就把电话给挂断了，顺便报了昨天她先挂断她电话之仇。
<br/>
<br/>        现在这情形，还真是风水轮流转。
<br/>
<br/>        今早她去公司取资料的时候，故意在师父郑忠威那儿套了话，才从师父郑忠威口中得知，前天晚上童筱颖跟傅夜沉的弟弟傅昼景私奔了。
<br/>
<br/>        傅夜沉彻夜未眠，兴师动众、担惊受怕地寻了他们一整晚，最后才得知童筱颖和傅昼景他们两居然私奔到一个信号不好的山窝窝里去了。
<br/>
<br/>        啧啧啧，这堪比年度大戏啊！
<br/>
<br/>        童筱颖这女人真的是贱到骨子里去了！
<br/>
<br/>        落以柔十分鄙夷童筱颖的所作所为，亏她童筱颖的名字里还跟她落以柔共用了一个“以”字，让她恨不得替她童筱颖去改个名，别玷污了她落以柔名字里的“以”。
<br/>
<br/>        另一边，童筱颖看着手机屏幕上的通话结束，心里顿时很不是滋味。
<br/>
<br/>        她并不是因为那个落以柔挂断了她的电话而不开心，而是因为傅夜沉生病了还未好，让她心里莫名其妙地感到担忧与不安。
<br/>
<br/>        现在的她，又能为他做什么呢？

<br/>        除了三年前在澳洲，得知傅昼景要做肾移植手术时，她都从未如此为一个男人感到揪心过。
<br/>
<br/>        是因为担心他的缘故吗？
<br/>
<br/>        而她又是以什么身份在担心他呢？
<br/>
<br/>        弟妹？朋友？亲人？妻子？
<br/>
<br/>        童筱颖懊恼地挠了挠长发，然后手脚开始不听大脑使唤地在忙于收拾东西，准备下山赶巴车，返回滨江。

<br/>        长途漫漫，这是童筱颖第一次嫌路途遥远。

<br/>        “我……”童筱颖欲言又止。
