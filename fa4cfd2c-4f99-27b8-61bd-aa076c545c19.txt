        “您的包厢，我已经为您准备好了，请随我来。”女服务生迎了迎手，走在了前头。
<br/>
<br/>        进门后走过一段绿意盎然的玄关走廊，绕过一堵精致的缠绕着绿叶蔓藤的木隔断，抵到大厅。
<br/>
<br/>        大厅像偌大的玻璃花房，屋顶可以直观天气，绿色植被簇拥着一个又一个原木色的雅座，清新舒适，大厅中央的尽头有一波银色的池水，池水上架起小桥，直通一块褐色巨石面前。
<br/>
<br/>        傅夜沉叫住了那个领头的女服务生，让她先去包厢里准备，女服务生点头，先行离开。
<br/>
<br/>        他则牵着童筱颖的手，走过那座小桥，来到这块褐色巨石跟前。
<br/>
<br/>        童筱颖好奇地仰着头，看着这块巨石，巨石上刻满了整整齐齐的心形图案，而图案里镌刻着每一对情侣或是夫妻的名字。
<br/>
<br/>        不用傅夜沉提醒，童筱颖在众多名字里，找到了她和他的名字。
<br/>
<br/>        傅夜沉睐了身旁的童筱颖一眼，顺着她的目光，看到彼此的名字时，他情不自禁地嘴角微扬。
<br/>
<br/>        他以为她要找好久才会看到，没想到她一眼就看到了。
<br/>
<br/>        一个男人在追求一个女人的时候，无非就是陪逛街、陪吃饭、送浪漫。
<br/>
<br/>        他会把所有追求女人的招数，都用在筱颖的身上，然后剩下地就只有默默地等待着她的回应了。
<br/>
<br/>        看到这，童筱颖若是还看不明白傅夜沉的心意，那她就是真的眼瞎了。
<br/>
<br/>        他一直在努力地向她示爱，明里暗里，直接的、间接的，只要是行动上能做的，他都去做了。
<br/>
<br/>        “大哥，我肚子饿了。”童筱颖收回目光，顾左右而言他，完全看不出她心里作何感想。
<br/>
<br/>        傅夜沉顺着她的意思，牵着她去了早已预定好的卡座里。
<br/>
<br/>        这是个比较僻静的角落，卡座四周是枝繁叶茂的植物，肉眼看不出它们的真假，餐桌上的鲜花像插花艺术那样错落精致，人坐在里面，有点像是坐在百花盛开的森林里的错觉。
<br/>
<br/>        这一路，他都是紧紧地牵着她，这会儿，他总算是放开了她的手，在她优先坐下后，他才走到她的对面，优雅地落座。
<br/>
<br/>        他们刚坐下，立马就有服务员推着餐车走过来上菜。
<br/>
<br/>        菜的式样中西结合，每道菜的分量少，品种却很多，而且精致的骨瓷配上精致的菜式，就像欣赏精美的艺术品一样，格外令人赏心悦目。
<br/>
<br/>        服务员给他俩一人盛了一小碗汤，放在他俩的跟前后，就欠身退了出去。
<br/>
<br/>        童筱颖垂眸看着自己面前的这小碗汤，忽然间想到了什么，抬眸看向傅夜沉，微笑道：“落秘书，她人真的很不错，心细又体贴。今天中午，我做的荤菜和汤，过于大补。落秘书看到后，发现不适合刚出院的你吃。她就主动跟我提了这事，但是，我又不可能重新回去做，她便让我拿她的便当给你，她吃我做的那份便是。”
<br/>
<br/>        “是这样？”傅夜沉身心一怔，狐疑地看着童筱颖。
<br/>
<br/>        童筱颖点了点头，脸上保持着微笑，淡定地接着说道：“我在她办公室里待了一会儿，发现她不仅生活上心细，就连工作上也很认真、一丝不苟，特别上进。公司能招到这样的女孩，是公司的荣幸。”
<br/>
<br/>        “……”傅夜沉不由地皱起了剑眉。
<br/>
<br/>        难道是他误会了落以柔？
<br/>
<br/>        童筱颖看着傅夜沉，见他皱着眉头不说话，心里摸不准他的态度，于是继续替落以柔说好话：“她虽然只是一个秘书，但不仅能把上司交代的工作做好，还能关照到上司的身体健康，在公司里任劳任怨。真的很不错！所以，公司里一定要好好待她，别让其他公司的老板把她给挖走了。”
<br/>
<br/>        “你为什么会突然提到她？”傅夜沉的敏感神经一上来，就开始怀疑，他的这个小女人不会是想把他推给别的女人吧？
<br/>
<br/>        她安安静静地不再跟他闹离婚，其实是在想法设法地替他找别的女人来上位，让他先移情别恋，主动放弃她？！
<br/>
<br/>        “因为现在像她这么年轻的女孩子，能吃苦耐劳的，已经很少了。贤才，应为己所用。”童筱颖嘴角微扬，从容不迫地回答。
<br/>
<br/>        “你真的，只是这么想的？”傅夜沉再次追问。
<br/>
<br/>        童筱颖很认真地点了点头，顿了一下，又接着说道：“要不，公司再招一批新人进来，公司好好培养他们，与公司一起发展，把该给他们的福利都给他们，稳住他们的人心。人最怕自己怀才不遇，也不会忘记自己的知遇之恩。花钱去挖人才，还不如花钱去培养人才。大哥，你觉得呢？”
