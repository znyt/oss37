        童筱颖回到自己的房间里后，就在思忖着如何装病。
<br/>
<br/>        这病不大好装吧！
<br/>
<br/>        得真病了才行！
<br/>
<br/>        晚上季雅珊原本说好要来陪她的，但是因为十爷需要，季雅珊便陪十爷去了。
<br/>
<br/>        其实，在季雅珊的心里，爱情应该比友情更重要。
<br/>
<br/>        童筱颖并不计较这些，反倒是一门心思地在想如何让自己生病。
<br/>
<br/>        她先是在冷水里泡了一个小时，又将自己屋里的中央空调的温度调到最低。
<br/>
<br/>        这大半夜的她强迫自己不睡觉，来回折腾，终于把自己给整出了发烧。
<br/>
<br/>        到了第二天的起床时间点，季雅珊见童筱颖还未起床，便主动来房间里叩门。
<br/>
<br/>        季雅珊一边叩门一边呼喊着童筱颖的名字，见童筱颖不答应，只好自己压下门上的手柄，推门而入。
<br/>
<br/>        “筱颖，不许赖床噢！得起床跟我一起打扫……”季雅珊的话才到一半，看到躺在床上的童筱颖面色红润，整个人却一副无精打采的样子，连忙在她床边坐下，下意识地摸了摸她的额头，“好烫！”
<br/>
<br/>        童筱颖迷迷糊糊地有了些反应，半睁着眼睛，看到来人是季雅珊，只觉头脑昏沉，气若游丝地说道：“雅珊……我……头好痛……”
<br/>
<br/>        “你怎么突然病呢？”季雅珊收了手，立马从床边起身，莫名地紧张起来。
<br/>
<br/>        “我、我昨晚……看到了……何明旭、还有……还有林若琴……和……他们的孩子。”童筱颖无力地回答道。
<br/>
<br/>        季雅珊听后，脸色大变：“我看你是烧糊涂了！我这就叫阿十送你去医院！”
<br/>
<br/>        “我……”童筱颖欲言又止。
<br/>
<br/>        季雅珊似乎也无心听她把话说完，起身急匆匆地就走。
<br/>
<br/>        童筱颖只觉自己头脑昏沉，想睡觉，迷迷糊糊里，有人将她的身子抱起。
<br/>
<br/>        再次醒来的时候，童筱颖看着天花板上的吊灯，只觉这里很熟悉。
<br/>
<br/>        童筱颖坐起身来，环顾了一下四周。
<br/>
<br/>        这房间的布局，和傅夜沉私宅的房间布局一样。
<br/>
<br/>        她这是做梦了吗？
<br/>
<br/>        为了确定自己不是在做梦，童筱颖抚着额头从床上下来。
<br/>
<br/>        将这房间一一打量了一番后，才确定自己是在大哥傅夜沉的私宅里。
<br/>
<br/>        “阿沉！”门外忽然传来连华生的呼唤声。
<br/>
<br/>        童筱颖得知傅夜沉也在门外，欣喜地往门边走去。
<br/>
<br/>        “你看我带谁来呢？”连华生接着说。
<br/>
<br/>        房门虚掩着，开了一道缝。
<br/>
<br/>        童筱颖刚走到门边，透过这道缝隙，就看到了傅夜沉的身影。
<br/>
<br/>        然而，下一秒，另一抹倩影，直接扑到了傅夜沉的身上。
<br/>
<br/>        “大哥，太好了！你没有死！呜呜——”落以柔哭得梨花带雨，紧紧地抱住傅夜沉的腰身。
<br/>
<br/>        傅夜沉刚想推开落以柔，落以柔却越抱越紧。
<br/>
<br/>        连华生在一旁帮腔道：“以柔为了你，吞安眠药殉情，虽然救了条命回来，但她总为你伤心落泪，茶不思饭不想，为你日渐消沉。我于心不忍，就只能带她来见你了。”
<br/>
<br/>        听着连华生道出了她的相思之痛，落以柔的眼泪止不住地夺眶而出，紧紧地抱着傅夜沉就是不放手。
<br/>
<br/>        她的眼泪，染湿了他胸膛的衣襟。
<br/>
<br/>        仅仅只是一门之隔，看到这样的画面，童筱颖忽然间觉得，自己与傅夜沉之间似乎已经隔了十万八千里那么远。
<br/>
<br/>        第一次，她在心底开始怀疑自己，她是不是不如落以柔，对傅夜沉的爱，没有落以柔那么深？
<br/>
<br/>        “筱颖什么时候能醒过来？”傅夜沉还是挪开了落以柔的束缚，看向一旁的连华生问道。
<br/>
<br/>        连华生悠悠然地开口，淡漠地回答道：“哪个女人更爱你，你分不清楚吗？”
<br/>
<br/>        “你不要扯开话题！”傅夜沉一脸不悦。
<br/>
<br/>        “童筱颖和那个十爷是什么关系，已经显而易见了。她给你的那个微型炸弹，好在只是炸毁了一间信息室，弄伤了两名同志，幸好没炸死人。上头不抓童筱颖，完全是看在你的面子上。如果不是上头非要执行这个计划，只怕你还是看不清童筱颖的真面目。”
<br/>
<br/>        “我问你，筱颖什么时候能醒过来？”傅夜沉忽然疾言厉色起来。
<br/>
<br/>        连华生“啐”了声，不耐烦地回答：“今晚。”
<br/>
<br/>        “其他话，去你实验室里谈。不要打扰筱颖休息。”傅夜沉说完，便离开了。
<br/>
<br/>        童筱颖随后也听到了他们几个人离去的脚步声。
<br/>
<br/>        虽然她对连华生的那些话听得有些不明不白，但是她大致能猜到，或许是连华生认为，她和十爷他们是一伙的。
<br/>
<br/>        她这是莫名其妙地被树敌了吗？
<br/>
<br/>        童筱颖躺回了床上，闭上眼睛，打算装作什么都不知道，安安静静地等傅夜沉回来。
