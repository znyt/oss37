　　人们都说生死有命，富贵在天，李承乾不想做缩头乌龟，大唐皇帝也不可以当缩头乌龟，就算明知道外面有一千把步枪对着自己，这个时候他也必须站出来面对这一切。
<br/>　　至于说后不后悔搞出步枪这种不应该出现在大唐这个时代的武器，这一点李承乾从不后悔，所谓屁股决定脑袋，站在大唐皇帝的立场上看，武器的进步与发展是推动大唐这辆列车滚滚向前的动力，无论如何也不能因噎废食。
<br/>　　队伍继续前行，不多时已经穿城而过，竞技场已经出现在前面不远的地方。
<br/>　　一路上风平浪静的结果大大出乎所有人的意料，黑子和称心两人也是一头雾水。
<br/>　　但是车驾已经到了停车的位置，四周也没有什么明显可以藏人的建筑，他们已经无法在车里继续拖延时间。
<br/>　　最后只能在小白和夜魅的陪同下走下马车，又在一群护卫的保护下进入竞技场的休息室，而在他们的身后，则是程琳等人以及带了一层面罩的李承乾。
<br/>　　等到进了休息室，李承乾的身份瞬间就被苏玫等人认了出来，惊讶的问道：“陛下，您……您怎么来了？”
<br/>　　“别管我怎么来的，衣服换过来吧，刚刚发生的事情朕就当成不知道！”李承乾并没有与苏玫多说什么，神情严肃的叫过黑子：“一会儿加派人手，注意会场四周的动静，除了主看台的方向，其他三个方向都要严查。”
<br/>　　“是，臣明白。”黑子也知道这个时候没有时间争论，一边在旁人的帮助下将属于李承乾的龙袍脱下来，一边紧张的答道。
<br/>　　“别那么紧张，人这东西其实就是命，该你河里死，沟里绝对死不了，如果老天今天真的要收了朕的命去，你就是再紧张也没用。”李承乾借着这个机会在黑子的肩膀上拍了一把，末了还不忘调侃道：“还有，你未来一年的俸禄没有了，权当你租借朕这一身衣服的租金。”
<br/>　　“租金？！”黑子愣了一下，旋即苦笑道：“陛下，您，您这个时候还有心思开玩笑？”
<br/>　　“朕可没跟你开玩笑，扣你一年的俸禄也是你占便宜！”李承乾像是没有听懂黑子的话一般，又与他调笑了一句，然后才正色说道：“小心一些，千万不要出了什么岔子，不到万不得以不要拼命。”
<br/>　　“是！”黑子郑重点头，随后便与已经换好衣服的称心两人一起走了出去。
<br/>　　“长乐呢？为什么没有看到她？”李承乾等到黑子和称心出去了，这才转头看了一眼小白问道。
<br/>　　“不知道，我们进来的时候长公主就不在这里。”小白摇摇头。
<br/>　　李承乾瞳孔微微一缩：“去找！马上！活要见人……”。
<br/>　　“是！”小白也不等李承乾把话说完，答应一声就出去吩咐人去了，毕竟接下来的那一句不是什么好话，说出来有些不吉利。
<br/>　　等到把所有事情都安排完，李承乾也换好了衣服，头顶冕旒，在苏玫与程琳的陪同下离开休息室，前往场内观礼台。
<br/>　　皇帝出行，礼仪自然是非同一般，一阵鼓乐声中，竞技场瞬间变的鸦鹊无声，先是两组各百人的禁军开路，而后七七四十九对宫娥彩女持各式礼器前行，接着才是皇帝与妃嫔，再后面无数宫女以及禁军。
<br/>　　这还是考虑到竞技场上没有太大的空间，在李承乾强烈要求下缩水了近一多半的阵式，如果真的按照唐礼而行，估计整个这一面看台都装不下他的随员。
<br/>　　不过就算是这样，也依旧把其他各国的使节唬的一愣一愣的，那华美的衣饰，大气的排场，还有无数的奇珍让那些使节无不感到自惭形秽。
<br/>　　中原大地有着数千年的文化传承，仅仅这一份传承可足以使无数人望之心生沮丧。
<br/>　　或许那些异族的国家有钱，但是比礼仪，比底蕴却要比大唐不知差上多少。
<br/>　　只不过李承乾却并不在乎这些，他现在最着急的是知道长乐的下落，那个丫头已经很长时间没有消息了，在这个关键时刻，容不得他不着急。
<br/>　　可就在他来到观礼台属于他的位置之后，原本一直提着的心总算是放下了，因为一直让他十分担心的长乐就端端正正的坐在那里，巧笑倩兮的看着他。
<br/>　　看到如此表现的长乐，李承乾也顾不上什么礼节之类的事情，上前两步来到长乐的面前问道：“丽质，你怎么样？这几天有没有发现身边有异常的情况？”
<br/>　　“哥，你怎么了？什么异常情况啊？”长乐看着李承乾，惊讶的问道。
<br/>　　“别说那么多了，一会儿你尽量不要一个人露面太久，如果能躲最好到能退到后面的休息室，我怕今天有大事要发生。”李承乾一时间也不知道怎么跟长乐解释，草草的说了一句之后，便转过身，面对竞技场中心含笑而立。
<br/>　　“参见吾皇陛下，吾皇万岁万岁万万岁！”
<br/>　　“参见天可汗皇下，天可汗陛下万岁万岁万万岁！”
<br/>　　属于大唐百姓和异族番邦的礼拜声贯长空，如山呼海啸一般响起。
<br/>　　“平身！”李承乾高举双臂，在半空虚扶，但其声音却透过他面前的一套诡异装置传便整个竞技场。
<br/>　　‘哗……’从来不知道什么叫扩音装置的百姓和官员还有异族使节全都被惊到了，一个两个目瞪口呆，几乎以为自己见到了神明一般。
<br/>　　只有几个从科学院赶来，维护设备的大唐科学院的工程师脸上露出得意的笑容。
<br/>　　不过李承乾眼下却顾不上这些，现在的他全部注意力都集中在观察四周异状上面，毕竟枪丢了两枝，子弹丢了百来发，如果有人想要杀他，眼下就是最好的机会。
<br/>　　可事情的发展再次出乎了李承乾的预料，他想像中的攻击并没有到来，甚至直到他发表完了一篇感人肺腑的演讲之后，依旧没有发现竞技场上有任何的异常情况出现。
<br/>　　这让李承乾不禁在想，是不是情报科搞错了？又或者自己搞错了？那丢失的枪械并没有被拿到这里，而是被那些歹徒带走了？
