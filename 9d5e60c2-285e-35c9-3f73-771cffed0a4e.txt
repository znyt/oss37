　　徐锋的话让众人都安静下来，突击，围杀，有些词让他们异常熟悉。
<br/>　　徐锋道：“这超级丧尸队已经有了自己的主观意识和战略想法，你们可以理解为那是一群与战队相等的军队，而且你们不觉得它们的作战方式很熟悉么？”
<br/>　　“好像在那里见过，隐藏，暗伏。”一人开口，声音有些惊异。
<br/>　　“跟狼牙三战队学的。”徐锋的笑容带着丝丝冷意：“而且我不知道是江凌若太强，还是这丧尸首领太弱，刚才那群超级丧尸队竟然对一只首领丧尸的死亡无动于衷，反而更像是用它当诱饵。”
<br/>　　众人沉默不语，望着地上那只丧尸首领的尸体全体深思，如此一分析事件便诡异的不正常起来。
<br/>　　第一小队遇到超级丧尸队后先是往前走了一段路，但却只有四个人往前走，剩下的一人折回原地静待，而就是这么临时的变动，还真让第一小队发现了一丝不同寻常。
<br/>　　首先就是那队超级丧尸，在离开后的一段时间后又原路返回，寻到了那只二代丧尸烧掉的痕迹，这完全不属于正常丧尸的行为立即让徐锋警惕起来，便带着全体第一小队成员一路追踪。
<br/>　　这大概是以徐锋为首小队隐匿追踪的一次爆发，不仅完全没让那队超级丧尸发现，甚至连狼牙战团的侦查队都差无所觉，完美消失在这片森林。
<br/>　　一路的跟踪也让徐锋等人发现了越来越多的可疑点，这队超级丧尸数量只有几十，但却跟这片地带的其他丧尸没有联系，单独行动模式与战队作战极像。
<br/>　　尤其是这队超级丧尸绕了这片地带一整圈，不仅远远围观了其他小队的移动轨迹，更折回来追上了江凌若所在的第五小队，见到江凌若独自解决了一头丧尸首领之后，它们便另则方向追了过去。
<br/>　　很像一批暗杀队伍！
<br/>　　尤其像黑芒！
<br/>　　徐锋不禁暗自思考，他们到底是哪一场战役里用过这种手段？竟被丧尸偷学了过去！
<br/>　　“眼下我们该如何？”第一小队有人出声询问。
<br/>　　当下他们所有人都被打乱，三战队中没有任何一队集合在一起，那是因为他们没有想到埋骨之地的边缘地带就遇到这种诡异情况，也没料到一路上竟然还有二代丧尸在暗中观察。
<br/>　　徐锋手握金枪毫不示弱：“第五小队能引开丧尸群，我们自然能引开超级丧尸队。”
<br/>　　“干掉？”一名来自杀羽战队的成员眼冒精光。
<br/>　　徐锋看了其余的四人一眼，冷静下令：“干掉。”
<br/>　　三名并非来自杀羽战队的成员都心头一抖，对名震天下的杀羽战队队长不禁有了更深的认识，果然激进的不像话。
<br/>　　——————
<br/>　　此时早已移动了一段距离的江凌若，则是忽然发现前方道路变得崎岖曲折，泥泞不堪以至于脚踩下深陷半公分之多，极其影响移动不说，周围也到处都是拦腰而断的木桩，一个个立起严重阻碍了前行。
<br/>　　江凌若一下子停下，望着这带有极强人工刻意的场景皱起眉。
<br/>　　她确定狼牙没有这项计划。
<br/>　　忽的，一道微弱的风声从耳畔而过，让江凌若猛然身躯侧移躲开，只见一柄类似于匕首的铁器一闪而过，‘咻’的一声没入她身后的木桩之内，但那匕首射来的方向却一片黑暗什么都看不见。
<br/>　　就在江凌若微惊之际，忽的感受到身旁传来阻碍感，紧接着便与一根木桩重重撞上。
<br/>　　嘭！
<br/>　　十阶新人类的力量何其大，一下子就将这木桩撞飞，而相撞过后的用力也让她的一只脚深深陷入泥泞之内，‘咔’的一声似乎踩到了什么东西。
<br/>　　江凌若稳住身形低头一看，顿时瞳孔一缩。
<br/>　　那是一只丧尸的脑袋！
<br/>　　心忽然下沉，江凌若深吸一口气仔细观察了一下地面，很快便发现这整个泥泞地面之下都是丧尸的尸骨，而这些泥泞竟然都是由丧尸的黑血浸染而成。
<br/>　　这是一片腐血沼泽！
<br/>　　但是狼牙一路而来的推进并未有造成任何地带的腐血沼泽，但凡遇到大数量的丧尸群，不是当场烧尽了那些丧尸的尸体，就是避开再避开，尤其是这片地带是整个狼牙战团十万大军的必经之路，前方打头阵的人怎么可能留下这么难走的路给队友？
<br/>　　所以！
<br/>　　事态已然诡异的令人发指！
<br/>　　江凌若的表情看不出任何情绪，她只是轻轻抬头看向了上方黑夜中的微红月光，耳听八方将感知力放到了最大。
<br/>　　呼！
<br/>　　又是一道微弱的风声。
<br/>　　刷！
<br/>　　快速踩着沼泽地躲过。
<br/>　　咻！
<br/>　　又是一柄铁质匕首没入树桩。
<br/>　　她被埋伏了，对方是人是鬼都不知道！
<br/>　　——————
<br/>　　此时已经行动的徐锋等人，则是一路追着前方那批超级丧尸队狂奔，他们计算好了双方的速度和距离，但奔走了极长时间都没有见到超级丧尸的影子。
<br/>　　又过了几分钟后，徐锋猛然停下看了眼四周的一片漆黑，静悄悄的什么都没有。
<br/>　　其余四人也觉得事情不对，有人蹲下摸了摸地面，而后起身道：“没有移动痕迹。”
<br/>　　“我们没有走错，我确定方向正确。”另一人也立即开口。
<br/>　　徐锋鼻子嗅了嗅，空气中弥散着埋骨之地边缘固有的腐臭气味，看起来一切正常。
<br/>　　“重新推理一遍。”忽的，徐锋瞳孔微缩，问道：“我们追上超级丧尸队之前江凌若在与丧尸首领战斗，我们到达之际超级丧尸队正好离开，战场上只有打斗痕迹与那只丧尸首领的尸体。”
<br/>　　有人直愣愣的看着徐锋，忍不住的脱口而出：“其余丧尸都追着第五小队过去了吗？”
<br/>　　丧尸首领战斗时身旁不一定会有丧尸群围聚，但这一整片地带只有一只丧尸首领，其他的全部都是听从它号令的丧尸群，正常智力的丧尸首领一般不会让大部队离开它的视线，尤其是它被江凌若的领域锁在其内之时。
<br/>　　忽然想到这一可能性细节的徐锋深深皱起眉，看着周围的一片无尽黑暗，总觉得有什么东西在看着他。
