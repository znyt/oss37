　　只见，
<br/>　　那道巨拳虚影，还没有靠近叶尘跟前三米，就被一道金色光罩所阻挡，然后瞬间破碎开来。
<br/>　　“什么！”
<br/>　　所有人目瞪口呆，就连那位东门统领，也被眼前的场景惊呆了。
<br/>　　他可是御空境强者！
<br/>　　虽然只有御空境二重，可是能轻松将他的攻击化解，至少也得有御空境后期的实力吧？
<br/>　　但对方明明如此年轻，怎么可能具备如此强大的实力？
<br/>　　“不！这不可能！”
<br/>　　东门统领顿时大吼一声，手掌一翻，已经从空间法器中，取出一边巨锤，然后纵身一跃，狠狠朝叶尘砸落下来！
<br/>　　“不自量力！”
<br/>　　叶尘连看都没有看一眼，直接随手一抓，
<br/>　　轰！
<br/>　　空中陡然出现了一个，巨大的金色手掌，一把便将东门统领死死擒住。
<br/>　　“死！”
<br/>　　叶尘手掌猛然一握，
<br/>　　彭！
<br/>　　东门统领竟然直接被叶尘一手生生捏爆！
<br/>　　“啊！！”
<br/>　　周围的魔族众人，看到一幕，顿时吓得纷纷尖叫倒退，满脸的不可思议之色。
<br/>　　罗刹城在九大势力之中，实力本来就属于垫底的存在，御空境之上的强者，满打满算也就十几人。
<br/>　　因而这东门统领，在罗刹城中，绝对属于实力最顶级的存在。
<br/>　　然而现在，竟然被眼前的这个普通侍卫，随手捏爆！
<br/>　　这家伙的实力得有多强？
<br/>　　还有，
<br/>　　他真的只是一名普通侍卫？
<br/>　　叶尘轻描淡写的杀了东门统领，随手拍了拍自己的手掌，转而又望向那位十七公主。
<br/>　　后者顿时吓得娇躯一颤，脸色煞白无比，险些从灵兽背上栽倒下来，结结巴巴道：
<br/>　　“你你……你别乱来啊！我父亲可是罗刹城的城主，御空境巅峰强者，你若是杀了我，就是和整个罗刹族为敌！”
<br/>　　叶尘不由得摇了摇头，
<br/>　　虽然他对魔族之人向来不喜，不过眼前的这个小丫头，毕竟才只有十四五岁，他也懒得和她一般计较，直接转而望向一旁的戴安娜，道：
<br/>　　“还愣着做什么？我让你查的事情，还不敢赶快去查！”
<br/>　　戴安娜被叶尘目光扫到，顿时吓得浑身一激灵，连忙道：
<br/>　　“是是！我这就去查！”
<br/>　　她原本以为，这个来历神秘人族男子，顶多也就只有御空境初期的实力，亲眼看到他将东门统领捏爆，这才意识到对方的实力，竟然恐怖到如此程度！
<br/>　　只怕是和他们城主一个级别的存在！
<br/>　　戴安娜此时哪里还敢反抗，当即乖乖朝城中走去。
<br/>　　叶尘紧跟其后，直接无视那位十七公主和众位的众人，大摇大摆的走进了罗刹城。
<br/>　　众人见状，顿时再次愕然，
<br/>　　这家伙当众拍死了北山家的嫡子，又灭杀了罗刹城十大统领中的一位，竟然还敢大摇大摆的进城？
<br/>　　嚣张！
<br/>　　简直太嚣张了！
<br/>　　直到叶尘的身影，已经完全消失在城门后方，之前已经被吓傻掉的那位十七公主，这才渐渐回过神来，想起之前羞辱的一幕，顿时气得不行，俏脸上不由得浮现出一抹寒色，心中恨恨道：
<br/>　　“这可是你自己找死，等我回到城主府，请父亲大人出马，一定将你千刀万剐！”
<br/>　　……
<br/>　　叶尘哪里知道十七公主的想法，就算知道了，也不会放在心上。
<br/>　　此时的他，随着戴安娜，一同进入罗刹城中后，正直奔城主府而去。
<br/>　　根据戴安娜所说，当初那块美人冰雕，先是送到了城主府，后来又被魔皇城的一位贵族看中，高价从罗刹城城主手中购走。
<br/>　　而这位魔皇城贵族的身份，也只有这位城主大人最为清楚。
<br/>　　大约二十来分钟后，
<br/>　　两人来到罗刹城正中央，一座华丽的宫殿之前，
<br/>　　“到了！叶先生，前面就是城主府！”
<br/>　　“您先在这里稍等，我进去打探清楚之后就出来！”
<br/>　　叶尘点了点头，
<br/>　　反正戴安娜的身上，被他种下了火种，性命捏在自己的手里，倒也不怕她耍什么花招。
<br/>　　然而就在这时，
<br/>　　忽然一队人马，从远处呼啸而来！
<br/>　　戴安娜顿时脸色一变，当即拉住叶尘的胳膊，道：
<br/>　　“不好！北山将军来了，应该是听说了之前在城门口的事情！”
<br/>　　“叶先生，要不您还是先暂时避一避吧！”
<br/>　　戴安娜现在虽然恨叶尘恨的要死，可是两人现在已经在一条船上，倘若叶尘出了什么事，恐怕她也难逃罪责。
<br/>　　然而，
<br/>　　叶尘却纹丝未动，根本没有要躲避的意思，依旧一脸的悠然之色。
<br/>　　眨眼之间，
<br/>　　那群人马已经奔至跟前，将两人迅速包围起来。
<br/>　　这些人全都一身盔甲，骑着灵兽，威风凛凛。
<br/>　　继而，
<br/>　　从人群之中，走出一头全身白毛的狼形灵兽。
<br/>　　而在那灵兽的背上，则坐在一名五十来岁的老者。
<br/>　　老者目光如电，气势惊人，在两人身上扫了一眼后，目光最终落在叶尘的身上，身上升起一股浓浓的杀气！
<br/>　　“小子！就是你，杀我的爱子？”
<br/>　　还没等叶尘开口，后方忽然又传来一道清脆动听的声音，
<br/>　　“北山将军，就是他！就是他杀了铁男哥哥，您快快出手杀了他，为铁男哥哥报仇！”
<br/>　　叶尘眉头微微一挑，回头一看，
<br/>　　只见说话之人，赫然正是那位十七公主。
<br/>　　这小丫头，原来一直远远跟在叶尘的身后，如今看到已经回到城主府门前，顿时底气也足了起来。
<br/>　　老者听到这话，顿时两眼几欲喷出火来，
<br/>　　“小子！你杀我爱子，我要让你死！”
<br/>　　说完这话，那老者抬手一挥，
<br/>　　唰唰唰！
<br/>　　随老者一同前来的七八名手下，顿时纷纷从灵兽背上跃起，然后朝叶尘飞扑而来。
<br/>　　叶尘见状，不由得摇了摇头，
<br/>　　“为什么总有人喜欢找死？”
<br/>　　声音落下的同时，叶尘已经随手拍出几掌，
<br/>　　嘭嘭嘭！
<br/>　　老者的七八名手下，几乎同时被叶尘打爆，化成了一团团血雾。
<br/>　　“嗯？”
<br/>　　周围的行人看到这一幕，顿时纷纷倒退，就连那老者，脸上也露出一抹吃惊之色。
<br/>　　“好！难怪敢这么嚣张！”
<br/>　　“可惜，你遇到了本将军，今天你非死不可！”
<br/>　　说完这话，老者两眼猛然一瞪，
<br/>　　“赤焰魔瞳！”
<br/>　　呼！
<br/>　　老者的双眼，瞬间变成了金黄色，继而两道炙热的火焰喷出，犹如两条火龙般，朝叶尘呼啸而去。
