　　六道魔门，身为魔道至尊，什么时候受到过如此威胁？
<br/>　　在宫殿之中的无数老祖，脸色都是勃然大怒，看向楚风眠，眼神中充斥着无尽杀意。
<br/>　　似乎只要是有人开口，他们便是会一拥而上，来对付楚风眠。
<br/>　　楚风眠站在原地，脸色依然是极为平静，他的目光扫视着无数老祖，都是充满着冷笑。
<br/>　　他倒是要看看，谁还敢上前来。
<br/>　　那些老祖，虽然是对楚风眠的话极为恼怒，可是却并没有任何一人，敢主动上前，出手对付楚风眠。
<br/>　　楚风眠刚刚击败血袍老祖，已经是奠定了他无上威严，血袍老祖，无暇之境的强者，都在楚风眠的手中被轻而易举的给击败。
<br/>　　若是没有洛汐阻拦，更是现在很有可能都已经是被楚风眠轰杀了。
<br/>　　面对有着如此实力的楚风眠，他们这些老祖虽然都一个个极为愤怒，却都没有任何一人，还敢出手对付楚风眠。
<br/>　　这便是实力的代表。
<br/>　　若是在刚刚楚风眠没有将血袍老祖逼迫到如此地步，那以现在这些老祖的怒火，只怕是早就要将楚风眠给轰杀了，不会让楚风眠站在如此的轻松。
<br/>　　“好，楚公子的话，洛汐知道了。”
<br/>　　坐在王座上的洛汐，缓缓的开口道。
<br/>　　听到这句话，楚风眠便是点了点头，身躯一动，直接便是离开了这宫殿之中。
<br/>　　看来这洛汐，还没有是想要跟楚风眠为敌的想法，既然如此，楚风眠也懒得去主动招惹六道魔门。
<br/>　　就算是现在的六道魔门，无法是跟数万年前的那庞然大物一般的六道魔门相提并论。
<br/>　　可是只怕依然是极为恐怖的存在，在没有理由之前，楚风眠也不想去招惹是非。
<br/>　　等到楚风眠的身影，从这宫殿之中离开的一刻，那些老祖的目光，便是全部落入到了洛汐的王座之上。
<br/>　　“魅魔尊主，为何要放跑这小子？这小子实在是欺人太甚，今日不杀了他，有损我六道魔门的威严！”
<br/>　　一尊老祖，怒声开口道，正是那血袍老祖。
<br/>　　身为一尊老祖，他被仅仅只有生死境巅峰实力的楚风眠，给逼迫到了如此地步了，甚至是差一点都要死在了楚风眠的手中，这一切现在让他的心中都是无比愤怒。
<br/>　　恨不得现在就要将楚风眠给碎尸万段，看着楚风眠淡然离去，他自然是极为愤怒。
<br/>　　“不错，这小子是少主点名要杀之人，若是无法招揽他，就要先除掉他，莫非魅魔尊主忘记了少主的命令了吗？”
<br/>　　又是一尊老祖，开口道，他的目光也是露出了几分怀疑。
<br/>　　刚刚楚风眠跟洛汐的交谈之中，显然便是代表着，楚风眠之前跟洛汐见过，所以他才会突然怀疑起来。
<br/>　　“你们以为，你们出手，真的足以杀的了此人？”
<br/>　　洛汐目光闪烁，冰冷的开口道。
<br/>　　“此子的实力，比起你们想象中的，还要可怕的多，就算是刚刚想要出手，也根本不可能杀的了此人。”
<br/>　　“不可能！我们在场如此之多的老祖，莫非就连一个区区生死境的小子，都杀不了？”
<br/>　　血袍老祖满脸不信，开口道。
<br/>　　“就算是此子实力再强，也不可能是我们如此之多老祖的对手。”
<br/>　　“哼，此子既然是敢单枪匹马的前来，也自然不怕我们杀他，此人不除，的确将来很有可能是少主的心腹大患，就连七大宗门之中，都寻找不到一个像是此子一般妖孽的武者出来。”
<br/>　　洛汐的眼神中，也是露出了几分凶光，缓缓开口道。
<br/>　　“一个宁州古城，我们暂且不取，先去取其他城池也无妨，现在还不是跟此人撕破脸皮的时候。”
<br/>　　“走！”
<br/>　　随着洛汐大手一挥，这宫殿便是突然缩小，落入到了她的手心之中去。
<br/>　　这宫殿其实也是一件如同剑皇殿一般的灵器，可以随时消失缩小，随后在这宫殿之中的无数老祖，便是也纷纷催动遁光，离开了。
<br/>　　宁州古城之中。
<br/>　　这外面的一举一动，显然自然还是在楚风眠的观察之中，一直是看到那些武者全部离开，楚风眠心中才松了口气。
<br/>　　“六道魔门再度出世，不知道是福是祸，不过他们的出现，也足以是给我争取一段时间了，当务之急，还是要先踏入到生死台阶再说！”
<br/>　　楚风眠心中暗自想到。
<br/>　　一旦是踏入到生死台阶，便是就有资格称为老祖，就算是在七大宗门之中，都是地位极高。
<br/>　　达到这种境界，就算是在正面面对七大宗门，楚风眠也不会任何忌惮了，等到那个时候，七大宗门的人在想要对付楚风眠，也要忌惮几分。
<br/>　　楚风眠现在的实力，最为弱的，便是他的境界，只要是楚风眠的境界可以晋升，他的实力便是会一日千里。
<br/>　　“楚公子。”
<br/>　　就在楚风眠从这宫殿之中休息的时候，突然一座空间隧道，突然打通，两道人影，从这空间隧道之中，突然现身出来。
<br/>　　这两个人，其中一人正是楚风眠所熟悉的，古家家主，而另一人，却是一副陌生男子，不过此人的实力，也是跟古家家主类似，有着生死台阶第六重，炼心之境的境界。
<br/>　　显然此人的地位，在这宁州古城之中，也是极高，地位显赫之人。
<br/>　　“古家主，这位是？”
<br/>　　楚风眠看到这陌生男子，开口询问道。
<br/>　　“哦，这位是我宁州古城，神泉商会的会长，神泉老祖，楚公子所要的圣药，神泉老祖听到后，主动找到我来，说是他有，想要来跟楚公子做个交易。”
<br/>　　古家家主笑着开口道。
<br/>　　“神泉商会。”
<br/>　　楚风眠搜索脑海，这个名字，并非是曾经万年之前就崛起的商会，应该是在这个万年之中，先崛起的商会，这神泉老祖，应该便是其中的开创者。
<br/>　　不过这炼心之境的境界，开宗立派，的确是已经做到，而且一般小一些的势力之中，甚至都找不到一位炼心之境的强者。
<br/>　　这神泉商会，在中域的商会之中，已经排到中上游了，虽然不能跟那些老牌万年招牌的商会相比，不过也算是有些名气了。
