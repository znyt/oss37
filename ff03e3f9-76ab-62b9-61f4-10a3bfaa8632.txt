　　看到站在门口，头发还有些湿的慕容飞，佟千雪手一松，原本抱在她手里的毛绒玩具掉落在地上，眼眶立刻就红了，握紧拳头就要朝着站在门口的慕容飞砸去，佟千雪虽然气势很足，但是真的当拳头落下的时候，她的动作还是变得轻柔起来，轻轻地落在慕容飞肩膀上。
<br/>　　“坏家伙！”
<br/>　　佟千雪声音中带着明显的哭腔。
<br/>　　慕容飞上前一步，一把抱住佟千雪，低头就吻了上去，用脚后跟把门关上。两个人一边亲吻着，一边往客厅里挪动……
<br/>　　这一吻，吻得是天雷勾地火，如果不是因为知道佟千雪怀有身孕，现在不能进行房事，慕容飞一定会把她摁倒在沙发上，狠狠的蹂躏一番。
<br/>　　“怎么不打个电话就来了？”佟千雪靠在慕容飞怀里问道。
<br/>　　“打了电话就没有惊喜了。”慕容飞回答道：“而且我们平时不能用手机，就算是手机带在身上也想不到要去使用。”
<br/>　　“那你也不怕我不在家，白跑一趟。”佟千雪问道。
<br/>　　“我知道备用钥匙藏在哪儿，我可以自己开门进来。”慕容飞满不在乎的说道：“而且我知道，这段时间你肯定在家休息，我妈肯定不会让你去上班的。”
<br/>　　佟千雪察觉到慕容飞身上的衣服还是湿的，便站起身对他说道：“赶紧起来把衣服换了，外面下雨你都没打伞吗？还是在外面风餐夜宿习惯了，忘记从‘原始人’回归回来了？”
<br/>　　“出地铁站的时候买了把伞，可质量太差，下出租车的时候，被风一吹就翻了，只能把伞扔了冲过来。”慕容飞回答道。
<br/>　　“还特种部队精英呢，就这么被一个卖劣质雨伞的无良小贩给骗了。”佟千雪噗嗤一笑，在慕容飞额头上点了一下，说道：“我去给你拿衣服，先去洗一个澡，不然就该感冒了。”
<br/>　　说着就转身去卧室给慕容飞拿给他买的换洗衣服。
<br/>　　佟千雪拿了衣服，在卧室里那个浴室里给慕容飞放了热水，招呼他去洗澡。
<br/>　　慕容飞光着上半身，手里拿着那件被雨水淋湿的短袖，走进卧室。佟千雪往慕容飞下身扫了眼，轻笑着说道：“你的火气似乎有点大啊。”
<br/>　　“你现在还是别挑衅我。”慕容飞颇有些苦恼的说道：“真要把我逼急了，我会做出什么事情来，我自己都害怕。”
<br/>　　佟千雪上前一步，贴上慕容飞胸膛，故意挑逗着他，“我挑衅你，又怎么样？”
<br/>　　“女人，你小心玩火自焚！”
<br/>　　慕容飞盯着佟千雪，恨不得立刻把她吃干抹净，明明是自己老婆，可偏偏就这么有的看没得吃，其中到底有多苦，估计也就慕容飞自己知道了。
<br/>　　“想不想要？”佟千雪贴着慕容飞耳朵问道。
<br/>　　慕容飞知道佟千雪现在是不可能进行房事的，那就只能用别的方法，慕容飞大脑快速转动着看向佟千雪诱人的双唇。
<br/>　　注意到慕容飞的目光，佟千雪就知道慕容飞在想什么，在他脚背上踩了一脚，“你想都不要想，这是不可能的！”
<br/>　　慕容飞的脸立刻就苦了下去。佟千雪微微一笑，一只手如蛇一样伸进慕容飞裤腰……
<br/>　　就在慕容飞沉迷在如此享受中的时候，门铃却是不给面子的响了起来，瞬间就让慕容飞清醒过来。
<br/>　　佟千雪也停下了手上的动作，脸上有些慌乱，说道：“糟了，你妈妈来了。”
<br/>　　“我去！”慕容飞吓得立刻从床上站起来，想要把裤子提起来，可是他此时这个状态，却怎么也没能把裤子穿起来。
<br/>　　“咱妈不知道你回来了？”佟千雪意外地看着慕容飞问道。
<br/>　　“她当然不知道了。”慕容飞苦着一张脸，说道：“我下飞机就直接到这里来了，还没来得及给家里打电话呢。”
<br/>　　“你可真是个‘好儿子’！”
<br/>　　佟千雪白了慕容飞一眼，对他问道：“你到底行不行啊？”
<br/>　　“不行也得行啊。”
<br/>　　慕容飞真的都快哭了，自己老妈怎么就这个时候来了呢？
<br/>　　“去浴室，先帮你把问题解决了。”佟千雪说道：“要是把你憋坏了，老娘下半辈子岂不是只能靠玩具来解决了？”
<br/>　　“我的佟医生啊，你能不能别玩了？”慕容飞哭丧着一张脸看着佟千雪说道。
<br/>　　“你觉得我是在开玩笑吗？”佟千雪一副专业的态度说道：“我以专业的角度告诉你，男人在这个时候如果得不到释放，是会对男人那方面造成影响的，最坏的结果就是不举，这在国外可是有实际例子的，我可不是故意吓唬你而危言耸听。”
<br/>　　龙隐部队也教授过人体的生理结构，但那是为了让队员知道人的致命弱点在什么位置，在战斗中攻击什么位置，能让敌人瞬间致命，至于佟千雪所知道的那些东西，慕容飞并不了解。
<br/>　　“去浴室。”
<br/>　　慕容飞依言走进浴室，佟千雪跟着走进浴室后，立刻把浴室门给关上，继续着刚才没完成的事情。
<br/>　　门铃还在响着，这给了浴室中的慕容飞和佟千雪一种别样的刺激。过了一会儿，门铃停止了，可是家里的电话却是又响了起来，很显然是黄虹担心佟千雪出了意外，开始打电话了。
<br/>　　“你是不是把韭菜当主食了？怎么还没好？”
<br/>　　佟千雪这个时候也是有点急了。
<br/>　　慕容飞自然是知道，这全都是司徒空那套司徒家的祖传秘法的功劳，但是他又不能跟佟千雪明说，只能苦笑着对佟千雪说道：“你还是先去开门，我自己想办法解决。”
<br/>　　佟千雪听着外面电话铃声，还有黄虹拍打门的声音，知道自己如果再不去开门，慕容飞妈妈估计得报警了。站起身在洗脸盆边洗了个手，对慕容飞说道：“那你自己想办法解决，记得把痕迹处理一下，别让咱妈看出什么来。”
<br/>　　等到佟千雪拉开浴室门走出去，坐在马桶上慕容飞抬起自己右手苦笑，作为一个有着如花似玉老婆的男人，到头来居然还要靠“五姑娘”，这是何等的悲催！
