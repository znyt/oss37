        “夜沉哥，我没有！”季思妍瘪着红唇，哭丧着脸。
<br/>
<br/>        傅夜沉只是瞥了季思妍一眼后，目光冷峻地又朝那边的老大爷看去。
<br/>
<br/>        老大爷抬起眼皮子，与傅夜沉的目光在半空中不期而遇，一瞬间，他就像打了霜的茄子，闭了嘴，不再叫嚣了。
<br/>
<br/>        男警员立即过来，毕恭毕敬地向傅夜沉解释事情的经过。
<br/>
<br/>        因为那个路段刚好在监控盲区里，所以，他俩谁说的是真的，谁说的是假的，压根就无法分辨。
<br/>
<br/>        傅夜沉了解了事情的经过后，掏出手机打了通电话。
<br/>
<br/>        不到十分钟，派出所大门口就出现了几个医护人员，直接将老大爷给“架”走了。
<br/>
<br/>        “你的事情，我已经帮你解决了。以后再敢冒充我未婚妻，我绝不轻饶你！”傅夜沉怒瞪了季思妍一眼，冷冷地撂下话后，转身离去。
<br/>
<br/>        季思妍抿了抿唇，默默地跟在了傅夜沉的身后。
<br/>
<br/>        其实，刚刚应该是她走得太急，又没看路才撞到那个老大爷的。
<br/>
<br/>        但是，她好心扶起那个老大爷，还跟他说了“对不起”，那老大爷却趁机讹她的钱，就不对了！
<br/>
<br/>        当时，季思妍也不知道怎么想的，忽然间就想把这事情闹大，让傅昼景出来替她解决这个麻烦，可谁知，傅昼景对她如此绝情。
<br/>
<br/>        她倒是反而阴差阳错地把傅夜沉给叫出来了。
<br/>
<br/>        傅夜沉刚坐上驾驶座，季思妍立即拉开副驾驶座的车门，坐了进去，故作俏皮地笑着说：“夜沉哥，你送我回家，好不好？”
<br/>
<br/>        “下去，自己搭出租车！”傅夜沉剑眉微蹙，语气冷淡。
<br/>
<br/>        季思妍不依不饶，一边拉起安全带系上，一边嬉皮笑脸地回答道：“我就不！反正，今晚我被人讹了。我就讹你好了！既然你出来了，就得对我负责到底！”
<br/>
<br/>        “滚！”傅夜沉深吸了口气，忽然怒吼了一声。
<br/>
<br/>        季思妍被他这一声怒吼给吓了一跳，惊愕得脸上的笑意全无。
<br/>
<br/>        她并不是感情白痴，察言观色，她还是会的。
<br/>
<br/>        很明显，傅夜沉此刻的心情非常不好。
<br/>
<br/>        不过，静下心来，仔细回想一下，他起初进派出所的时候，唤她“漫雪”，莫非……
<br/>
<br/>        “夜沉哥，今晚的事情，我跟你道歉。你不要生我的气啦！好不好？就看在，我爷爷和你爷爷彼此认识的份上，行吗？”季思妍缓了缓心情，眼珠子一转，决定对傅夜沉来软磨硬泡这一招。
<br/>
<br/>        他们傅家和他们季家，确实只是彼此认识的这一层犹如薄膜般的关系。
<br/>
<br/>        傅夜沉确实没必要给她季思妍面子，更何况，她冒充他未婚妻的事情，着实令他心里窝火。
<br/>
<br/>        他是有多么想念“漫雪”，心里的那团火，只要浇一丁点的油，就能燃烧他心底整片寂静的森林。
<br/>
<br/>        季思妍见傅夜沉不吭声，于是装出一副楚楚可怜的样子，脱了高跟鞋，抱膝坐在副驾驶座上蜷缩着身子。
<br/>
<br/>        反正，她就是不下车，而且，眸中含泪，红唇微嘟。
<br/>
<br/>        她装出的可怜模样，傅夜沉一眼也未看，心里眼里，完全视她为空气。
<br/>
<br/>        但他并未再赶她下车，而是直接发动了车子，载着她扬长而去。
<br/>
<br/>        随后，傅夜沉将季思妍只送到她家院门口，她刚下车，他便踩着油门，片刻不停地驱车而去。
<br/>
<br/>        季思妍刚走进家门，哥哥季运恒就拍着手从楼上走下来，称赞道：“不愧是我妹妹，这么快就能让傅夜沉亲自送你回家了！”
<br/>
<br/>        “今晚是场误会而已，人家心里压根就没我。”季思妍甩了鞋，赤脚走进了屋。
<br/>
<br/>        从另一边出来的保姆见状，立即迎上去，蹲身给季思妍递上拖鞋。
<br/>
<br/>        “大小姐回来了，晚上要不要吃点东西？”保姆毕恭毕敬地询问。
<br/>
<br/>        季思妍淡淡地应了声：“不用了，你去睡吧！”
<br/>
<br/>        “是！”保姆颔首应了声，这才退下去。
<br/>
<br/>        季运恒吊儿郎当地将双手插在裤袋中，走到大厅中央的沙发上，慵懒地往上一躺，话中有话地说道：“傅昼景欠了银行巨款，现在应该很缺钱吧！”
<br/>
<br/>        “哥哥你是又在打什么歪主意吗？”季思妍打了个哈欠，准备上楼去休息。
<br/>
<br/>        季运恒接下来的话，让她停下了步伐。
<br/>
<br/>        “你有没有想过，傅昼景明明有个那么有钱的大哥，却仍旧还不上银行的贷款？”季运恒咧嘴阴笑，“六百万，这对他大哥傅夜沉来说，只不过是零花钱而已。”
<br/>
<br/>        “就算是亲兄弟，也要明算账的吧！”季思妍不以为然地应了声，打算继续往楼上走，不想理会自己这个无聊透顶的哥哥。
<br/>
<br/>        季运恒冷不丁地笑了笑：“我的好妹妹，亏你还是跟傅昼景谈过爱，居然这点事情都看不透。”
<br/>
<br/>        “你什么意思？”季思妍身心一怔，忽然间对哥哥季运恒的这个话题有了一丝好奇与纳闷。
<br/>
<br/>        “傅昼景是傅家的私生子，傅老太爷压根就不重视他，而且，他大学毕业后就跟傅家分开了。虽然，他在给自己的家企打工，但是，他却是一个自尊心极强也非常好面子的男人！”季运恒双眸微眯，咧着嘴，意味深长地说道。
<br/>
<br/>        季思妍似乎听懂了哥哥话中的另外一层意思，反问道：“那又如何？”
<br/>
<br/>        “以傅昼景的个性，他一定不会向他大哥开口要钱。所以，他一定会想办法证明自己的实力，努力赚回这笔钱，去偿还银行的债务，可偏偏没有一个好的机遇助他一臂之力。我们何不利用他这点？”季运恒得意洋洋地说道。
<br/>
<br/>        季思妍却嗤之以鼻：“阿景连他大哥的钱都不会要，又岂会要你的钱。我看，还是算了吧！”
<br/>
<br/>        “谁说我要免费给他钱了，我只不过是想到一个妙计，让他傅昼景成为我手里的棋子！”季运恒伸出手来，对着空气抓了抓，随即又瞄了一眼季思妍，打趣地问道，“妍妍，你该不会还爱着傅昼景吧？这种私生子有什么好爱的？我劝你，最好不要爱上任何一个男人。这世上，没有一个好男人。”
