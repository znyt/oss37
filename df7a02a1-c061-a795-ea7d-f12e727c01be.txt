　　“唔？不玩了。”
<br/>　　元始见自家的两兄弟都完成了斩首任务，就自己还没有，顿时觉得无趣，不想再玩。
<br/>　　“不好！”
<br/>　　帝俊与太一心中暗道一声，只觉得一股强烈的危机笼罩其身，就想要躲开。
<br/>　　“你们想要躲到哪里去啊，给我过来。”
<br/>　　元始那淡淡地声音在两人耳边响起，紧接着，元始两只大手分别向着两人毫不犹豫地拍去。
<br/>　　顿时，两人觉得自己寒毛颤栗，想要躲开，但却无论如何也躲不开。两妖顿时被拍了一个正着。
<br/>　　“噗嗤——”
<br/>　　随着两道鲜血喷出，金黄色的鲜血之中还有正在燃烧着的太阳真火火苗，于虚空之中燃烧，画面极为凄美。两道身影坠落虚空，摔倒在地。
<br/>　　怎么处理他们？杀了？还是放了？
<br/>　　直到此时，元始才想起一个极为头疼的问题。放在他眼前的四位大妖，可是妖族中的佼佼者，日后大名鼎鼎的妖皇与圣人，下一量劫的主角。
<br/>　　杀了，明显是不可能，就算元始想杀，鸿钧或者是说天道也不可能同意。更何况元始还不想杀他们。
<br/>　　不杀？那岂不是便宜了他们，我元始又岂会做那种出力不讨好的事？
<br/>　　“咦！”
<br/>　　元始看着眼前的混沌大钟，顿时觉得眼前一亮，他突然想到一个他自认为绝妙的主意。
<br/>　　要命，可以，用宝物来换！
<br/>　　看着眼前四妖眼中那不甘的神色，元始那淡淡的声音再次在他们耳边响起：
<br/>　　“四位，尔等还有何话要说？”
<br/>　　“技不如人罢了，只叹自己实力不足。要杀要剐，吾帝俊悉听尊便。”帝俊与太一不愧为妖皇，到了现在的地步，仍毫不低头道。
<br/>　　“哼！”女娲则是娇哼一声。伏羲的嘴角略带苦涩，却是不言不语。
<br/>　　元始看着四妖，淡淡道：“四位，现如今尔等之性命全在本座一念之间，本座要尔等死，尔等亦不能有丝毫反抗。”
<br/>　　说到这里，他停顿片刻。而四人听到这里，却是心中一紧，眼神中闪过一丝不易察觉的惊慌，但面目上，却是毫不动色。
<br/>　　元始看到此处，脸上却是微微一笑，继续说道：
<br/>　　“但——，本座如今心情不错，感念上苍要好生之德，却是不想再继续杀戮。你们想要活命，可以，就看你们自己的表现了。”
<br/>　　“什么意思？”四人眼神微微闪烁，询问道。
<br/>　　“什么意思？本座的意思很明显，你们的命，本座可以还给你们。”
<br/>　　“什么？”
<br/>　　几人惊讶道，想不到元始会放过他们，这简直令人难以置信，要知道，放了他们，那就意味着他以后可能会多出几个敌人。
<br/>　　“事情没那么简单吧。”伏羲若有所思，慢慢地说道。
<br/>　　“聪明！本座就喜欢和聪明人说话，这样就不必多费口舌了。”元始毫不吝啬地赞叹他一句，接着说道：
<br/>　　“想要活命，可以，你们估估你们的命值多少价值，你们就拿多少价值的东西来换了。”
<br/>　　“什么，元始，你欺妖太甚！”
<br/>　　听了元始的话后，四妖大怒，元始这话的意思十分明显，显然是将他们的命当做东西来换，丝毫没有将他们放在眼里。
<br/>　　若是交易价值少了，那不就意味着他们的命只值那么点东西，若是交易多了，不仅他们会心疼，而且他们的命还是可以用东西来衡量，结果一样。
<br/>　　一个人的生命可以是有价值的，也可以是无价的。四妖认为他们的命无限，不能用价值来衡量。而元始却偏要他们自己用价值来衡量，这不是侮辱他们又是在干什么？
<br/>　　若是这事今后传出去，那他们岂不会是被洪荒众人耻笑。这简直不能忍！
<br/>　　想一想自己以后去什么地方，就会听到路边的人耻笑：
<br/>　　“你看那就是妖皇啊，他的命只值一件什么什么宝物。”
<br/>　　“咦，那不就是什么妖皇吗？”
<br/>　　“还真是，听说当年……”
<br/>　　“哈哈哈！”
<br/>　　“……”
<br/>　　一想到这些，他们就感觉到自己的一生全部毁了，未来的人生一片黑暗啊。
<br/>　　“欺妖太甚？本座可没那份心情，若是你们偏要如此认为，那就随你们的意吧。现在就看你们的选择了，你们是要自己的命，还是要宝物，全在你们一念之间。”
<br/>　　元始那充满不屑、狂傲的声音传来，将他们的自尊心打击的一点不剩。四妖顿时清醒过来，认清现实，是啊，自己如今还有何自尊呢？
<br/>　　是命重要，还是宝物重要？当然是命重要了！没了命，就算宝物还在，那又有什么意义呢？没了宝物，有了命，这意味着自己以后有无限可能，说不定哪一天自己实力大涨，能够亲自报仇呢。
<br/>　　四妖眼神交流一番，顿时达成一致意见，问道：
<br/>　　“你要怎么换？”
<br/>　　“什么本座要怎么换？选择权在你们自己手中，这得看你们的了。”
<br/>　　几人面面相觑，不知该说什么是好。元始面无表情，静静地看着他们选择。
<br/>　　“用我的混沌钟！”过了一会儿，太一大声道。
<br/>　　“不，不行！二弟，用我的河图洛书！”帝俊争道。
<br/>　　“不行，大哥，你的河图洛书不能失去。”太一总感觉自己大哥的河图洛书未来会有大用，不能丢失。而自己的混沌钟虽然强大，但却没有给他河图洛书的那种感觉。
<br/>　　“二弟，你——”
<br/>　　“大哥，不用多说，我们是自家兄弟。”太一拦着自己的大哥，说道。
<br/>　　“我们又该如何？”
<br/>　　女娲与伏羲见此，不知该怎么办，自己两人并没有好的灵宝，手中唯一的红绣球与伏羲琴，也不过是上品先天灵宝，以元始的个性，应该看不上，这该怎么办？
<br/>　　两人顿时看向帝俊与太一。
<br/>　　四妖商量了一番，最终还是决定拿混沌钟交换。
<br/>　　“商量好了，如何？”
<br/>　　“我们以先天至宝混沌钟作为交换之物。”
<br/>　　太一小心地说道，他觉得自己有些不好意思，他们有四妖，却用一件至宝交换，他觉得元始不会同意。
<br/>　　但，事实却出乎了四人的意料，元始直接说了一个字，同意了：
<br/>　　“可以。”
<br/>　　其实，他们并不知道，元始的根本目的就在混沌钟上，至于其他的，帝俊的河图洛书虽为极品先天灵宝，但却是与推演、布阵方面有关，属于辅助型灵宝，元始并不需要。女娲的红绣球与伏羲的伏羲琴，都是上品先天灵宝，攻击力都很一般，元始亦看不上。
<br/>　　所以，有了一个混沌钟，就足够了。至于其他的，随意就好，有了更好，没有也无所谓。他也不好将四妖逼的太急。
