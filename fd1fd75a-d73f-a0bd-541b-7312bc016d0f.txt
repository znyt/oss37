　　一群圣党弟子，浩浩荡荡的，直接撕裂空间，来到了这武王城中。
<br/>　　在这圣党弟子的人群之中，楚风眠便是见到了许多曾经在北邙圣域中见到的熟悉身影。
<br/>　　其中北邙学院的许多核心弟子，都是站在其中，这些核心弟子，都是选择臣服了夜天君。
<br/>　　北邙圣域的核心弟子，可都是有着去争夺这北邙学院下一任院长资格的，如今他们都乖乖臣服了夜天君。
<br/>　　足以看得出夜天君的实力，到底是有着多么恐怖。
<br/>　　在这无数圣党弟子中，就在人群的最中心，有着一位众星捧月一般的青年。
<br/>　　这青年，一身华贵衣服，脸上始终带有着一副淡淡的笑容，看起来便是如同一位贵公子一般。
<br/>　　楚风眠一眼便是认得出来，此人，便是北邙学院的第一人，夜天君。
<br/>　　“混乱神体，真的是混乱神体。”
<br/>　　楚风眠扫视了一眼这夜天君，心中便是真正确定，这青年，真的就是夜天君。
<br/>　　他身上的那种混乱之力，是绝对无法模仿的，能够拥有这种力量，只有夜天君一人。
<br/>　　“恩？”
<br/>　　就在楚风眠看向这夜天君的一刻，夜天君的视线，突然也是看了过来。
<br/>　　这一道视线，打量了楚风眠跟寒月漓一眼，似乎是带有着几分冷笑，尤其是看向寒月漓的时候，眼神中的贪婪，毫不掩饰。
<br/>　　这眼神，便是在挑衅，尤其是看向楚风眠的时候，夜天君的眼神中，明显显现出了几分杀意。
<br/>　　楚风眠可以杀了不少圣党弟子，还将圣党的三位长老都给斩杀，跟着圣党，已经是不死不休的状态。
<br/>　　这夜天君，如今眼神中，已经是满是杀意。
<br/>　　不过这视线相对，下一刻便是都各自收了回来，在武斗大会前，一切的冲突，都没有任何的意义。
<br/>　　“这夜天君的实力，好强！”
<br/>　　虽然只是短短的接触，互相对视一眼，楚风眠便是看得出来，这夜天君，也无愧于是北邙学院的第一天才。
<br/>　　这夜天君表现出来的实力，只怕都已经是不在生死境巅峰之下了，更何况楚风眠还足以看得出来。
<br/>　　这夜天君还隐藏有着不少的实力，论起实力来说，比起地魔门的第一人冷面魔君，也是毫不逊色。
<br/>　　在这一次武斗大会上，几乎是排到前十位，毫无悬念的人选之一。
<br/>　　“楚风眠，寒月漓，果然他们两个，也来参加这次武斗大会了。”
<br/>　　夜天君扫视了楚风眠跟寒月漓一眼，眼神中尽是冷笑。
<br/>　　这还是他第一次跟楚风眠见面，不过在此之前，对楚风眠这个名字他也已经算是熟悉无比了。
<br/>　　当初听到楚风眠的时候，楚风眠还是一个就连神力境都没有踏入的武者，在夜天君看来，就是如同蝼蚁一样的存在，所以也没有多家注意。
<br/>　　当时只是让圣党的人，前去对付楚风眠。
<br/>　　没想到楚风眠倒是化解了一次次的危机，实力更是暴涨。
<br/>　　一直是成长到了这一步，这让夜天君的心中都有些诧异，有些后悔，没有将楚风眠扼杀在萌芽之中。
<br/>　　现在的楚风眠，已经是成长到了他的心腹大患的地步，能够轰杀龙未央，击败陆画，陆雀，楚风眠的实力已经是得到了证明。
<br/>　　“圣主，要不要我们去对付一下那小子？”
<br/>　　站在夜天君身边的一尊武者，也是注意到了夜天君脸色的变化，冷冷的开口道。
<br/>　　“这小子不知天高地厚，敢来跟圣主看上的女人站在一起，还敢来参加武斗大会，据说此人还得到了北邙圣君的传承，如今倒不如主动出击，将他灭杀。”
<br/>　　这武者，也是一尊北邙圣域的核心弟子，他看向楚风眠的眼神中，尽是杀意。
<br/>　　楚风眠得到北邙圣君的传承。
<br/>　　这个消息，如今已经是传遍了整个北邙学院了，北域中其他势力的弟子也许不知道，不过他们这些北邙学院的弟子，却都是知道的一清二楚。
<br/>　　这种奇遇，令他们都是一个个极为嫉妒，恨不得直接杀了楚风眠。
<br/>　　“这里是武王城，不能太招摇了。”
<br/>　　夜天君摇了摇头道。
<br/>　　“反正到了武斗大会的时候，有着无数次的机会，可以杀了他，让他在多活一段时间。”
<br/>　　“那两个人，就是北邙学院新一届的新人。”
<br/>　　“其中那个女的，可是先天神体，玄寒之体，据说度过生死大劫的时候，可是引出了千雷皇劫！”
<br/>　　就在周围的不少武者，也同时注意到了楚风眠跟寒月漓的存在。
<br/>　　其中大部分的视线，都是放在了寒月漓的身上，寒月漓的名声，可是比起楚风眠要大的多。
<br/>　　先天神体，千雷皇劫，这两件事，让如今寒月漓的名字，在北域之中几乎是无人不知，无人不晓。
<br/>　　号称北邙学院千年以来的超级天才，足以是跟夜天君相提并论的人物。
<br/>　　面对着一道道眼神不同的目光，寒月漓倒是毫不在乎，只是对旁边的楚风眠轻声开口道。
<br/>　　“距离武斗大会开始，还有几日时间，强者还没有全部到来，我们也可以找个地方，好好休息休息。”
<br/>　　“嗯。”
<br/>　　楚风眠点了点头。
<br/>　　如今这武王城中，虽然已经是占满了无数武者，但是人的确是还没有到齐。
<br/>　　起码如今北邙百绝榜上的第一位，现在楚风眠还无缘见到。
<br/>　　在众人的议论声之中，楚风眠已经是听到了，这北域百绝榜的第一人，名为风无心。
<br/>　　这风无心，神秘无比，来自北域之中的武道家族，风家。
<br/>　　风家的势力，本来也就是跟司空世家，公羊世家等极为接近的。
<br/>　　虽然有着不小的势力，但是却远远比不过北邙学院，地魔门这两大庞然大物。
<br/>　　只是却突然冒出来了这样一个风无心，这风无心，那是一年前才突然冒出来的人物，二十岁出头的年纪，境界已经是达到了生死境六重，踏入到了生死境高阶的地步。
<br/>　　一身实力，更是强的吓人，据说是从一尊超越生死的碎命之境强者手中，逃出升天，都只是受到了些伤势。
<br/>　　能够是从碎命之境的强者，平安的活下来，仅次一人，这风无心，也是因此踏入到了北域百绝榜的第一位，他的位置，无可撼动。
