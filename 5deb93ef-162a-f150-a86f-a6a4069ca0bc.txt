　　第762章
<br/>　　别的事情玉雀可能不懂，但男人看女人的眼神玉雀再不懂，她就妄称花魁。
<br/>　　一瞬间，她的心脏被恐惧占有。龙胖子笑吟吟看自己的眼神，像一头野狗垂涎天鹅肉的美味，忍不住用舌头舔起口水。
<br/>　　现在自己已经能看见他吐出的舌头，满嘴腥气扑面。
<br/>　　心思百转，思考着自保之策。拼死一搏，苦苦哀求，用自己的色相稳住对方，再思脱身之策……
<br/>　　心中转过一百个念头，却发现没有一个念头有效。美色在这个世道是很有价值的，某种时候也可以充当工具。但这种工具关键时刻，并无有自己想象中那样有价值。
<br/>　　身为满庭芳花魁，玉雀平素结交了不少达官贵人。但她知道自己现在在什么地方，这里是兵营！那些爱惜羽毛的老爷们，绝不会为了一个青楼女子得罪当兵的。
<br/>　　美貌，青楼女子……同样是没有力量的存在。
<br/>　　龙胖子盯着面前千娇百媚的美人，感觉心脏砰砰打鼓。他好色如命，平素也没少逛青楼，这次掌了大权，那些小姐夫人更是被他糟蹋不少。可如玉雀这等美貌的女子，却是头一遭遇到。
<br/>　　眼如水，肤若玉，嫩得一掐一出水，吱呀呀乱叫。
<br/>　　龙胖子心里像黑瞎子吃了蜜一样甜：“小娘子，不用怕，先让胖哥哥吃个口口。”
<br/>　　玉雀以手撑地往后退，现在她的玲珑心肝完全没有用处，只是靠本能求生。看上去，像是被大灰狼逼的走投无路的小白兔。
<br/>　　龙胖子捧腹大笑，庞大的身躯遮挡在玉雀身前。
<br/>　　“小娘子，你怕什么。有什么委屈同胖哥哥说，胖哥哥为你做主。”
<br/>　　二人一个追一个逃，在逼仄的房间内绕圈子。龙胖子东一挡，西一挡，就挡得玉雀无路可逃。
<br/>　　他更是开怀，一面缩小玉雀的活动空间，口中一面说个不停。
<br/>　　“我知道你们这些小娘子都爱小白脸，还是没有经验。银样镴枪头，中看不中用，关键还是胖哥哥这样耐得主的男人，现在你怕，等下你会哭着说不要停。”
<br/>　　玉雀终于被逼到墙角，龙胖子的身躯投下一片阴影。她逃无可逃，退无可退，双目因恐惧而睁大，却无眼泪。
<br/>　　龙胖子顺着她素白脖颈往下去，沉甸甸的软玉引人向往。娇小的身躯缩成一团，仿佛揽开双臂就能拥入怀中。好好把玩，百般爱怜……
<br/>　　他的眼神变得炙热，呼吸滚烫，双臂摊开，呐呐道：“小娘子，胖哥哥来了……”
<br/>　　砰！
<br/>　　房门在这一瞬被踹开，龙胖子一激灵，差点从九霄云端跌到十八层地狱。
<br/>　　“哪个王八羔子，扰了你爷爷的好性子，不想活了么。”
<br/>　　他骂咧咧回过头，表情忽地怔住。倒不是因为看见踹门的程大雷，而是看到一旁的卢俊义。
<br/>　　卢俊义的本事他是明白的，大家能从凉州拼杀出来，没有卢俊义绝对做不到。
<br/>　　他缩头缩脑的立在一旁，心中倒也不是如何害怕。这种事，在军营中根本不算个事，哪个女俘能全须全尾的从兵营中走出。大家都是血气方刚的大老爷们，又不是吃素的兔爷。
<br/>　　莫说是娘们，就是细皮嫩肉的爷们，他龙胖子也不是没有祸害过。
<br/>　　卢俊义恶狠狠的瞪他一眼，自己明明吩咐过不许动这女人，龙胖子竟然不听自己的话。
<br/>　　玉雀好险是回过神来，没有被吓成失心疯。她呆滞的眼神慢慢有了光彩，当视线可以看清楚时，一支大手搁在他面前。
<br/>　　“我带你走。”
<br/>　　几乎是下意识，玉雀将手搭在他掌心中。慢慢站起身，一步一步跟在对方背后，眼睛里只看得见他的背影。
<br/>　　龙胖子挠了挠头：“这货是谁呀？”
<br/>　　程大雷突然回过头来。
<br/>　　卢俊义立刻感觉到一股狂暴的杀机，他几乎是下意识往前迈步，一边走一边伸出手：“程当家，不可。”
<br/>　　程大雷松开玉雀的手，鬼影莫测的出现龙胖子面前，抬腿砸出一记重脚。直接踹在龙胖子的小腹上。
<br/>　　他臃肿肥胖的身躯倒飞出去，砸在屋子的柱子上。屋顶轻轻震荡，飘下一片灰尘，落满程大雷双肩。
<br/>　　龙胖子双手捂着小腹，身体蜷缩在一起，冷汗落了一层又一层。看到这一幕的所有人都明白，即便龙胖子经历这件事不死，下辈子也不能人道了。
<br/>　　卢俊义伸出的手无力的收回来，牙关咬紧，心中好恨。恨得是程大雷的残忍，同样恼恨龙胖子为何要多嘴。
<br/>　　忍一时风平浪静，让这个煞星快点离开不好么。
<br/>　　程大雷踏出屋子，外面的大兵聚在一起，将屋子严严实实的围住。
<br/>　　程大雷、李婉儿、刘发财、玉雀以程大雷为中心站成一团。程大雷目光冷冷扫过，从每个人脸上都读出了愤怒。
<br/>　　有人跑进屋子，将痛得撕心裂肺的龙胖子抬了出来。看到重伤的龙胖子，所有人的怒火在无法克制。
<br/>　　这些人多是随百里胜从凉州一路冲杀出来的兄弟，是你将后背交给我，我将后背交给你的交情。
<br/>　　龙胖子为人不如何，对兄弟却不错。他一身肥肉，不知为同伴挡了多是飞剑铁枪，又有不少人，是他从死人堆里扛出来的。
<br/>　　“卢二哥，就这样让他们走了么！”
<br/>　　“大哥死了，咱们是不是就得受这窝囊气。做这鸟官就要受气，还不如回山里做贼痛快。”
<br/>　　“都住口。”
<br/>　　卢俊义突然喝了一声，目光看向程大雷，程大雷也正看着他。
<br/>　　他双手一抱拳：“程当家，我得给弟兄们一个交待。”
<br/>　　程大雷没有说话，只是看着他。
<br/>　　“我愿与程当家一战，十招之内，程当家能立而不倒，这件事我们认了。”
<br/>　　嚯，好大口气。
<br/>　　刘发财嘴角不仅咧起，心想，这汉子也不怕风大闪了舌头。天下之大，程当家怕过谁，更何况只是十招。
<br/>　　程大雷却没有半点轻视卢俊义的意思，毕竟枪棒无双，同是绝世，马孟起还太年轻，但卢俊义正是炉火纯青的年纪。
<br/>　　程大雷探出手，口中吐出一个字。
<br/>　　“来。”
