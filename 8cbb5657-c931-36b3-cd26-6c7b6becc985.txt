　　突如其来的变故，让所有人都有些措手不及，但很快的，白小飞就镇定了下来。
<br/>　　“怎么回事，发生了什么？”
<br/>　　声音有些疑惑，下意识的看了一眼刘哔的方向，他却没有从后者那里得到什么答案。
<br/>　　鬼知道怎么回事。
<br/>　　“我出去看看。”
<br/>　　咬了咬牙，白小飞将卷帘门拉开一条缝隙，小心翼翼的向外面看去。
<br/>　　此时，外面十分混乱，昨晚他们交战的那栋大楼，顶端已经成为了废墟，一些破碎的墙体和石块散落在广场上，没有尸兄。
<br/>　　可昨天他们上楼的时候，这边的还有密密麻麻的好多尸群。
<br/>　　“哇……”
<br/>　　一道类似于婴儿啼哭的声音响起，强劲的声波，让白小飞耳膜有些发胀。
<br/>　　顺着声音传来的地方看去，他看到了一个庞大的身影，但因为距离有些遥远，他辨认不出是何等生物。
<br/>　　只觉得是个人形。
<br/>　　刚才爆炸的声音，大概也是从那个方向穿过来的。
<br/>　　“我觉得，我们现在，还需要在这里呆上一段时间。”
<br/>　　卷帘门被拉开，刘哔也看清楚了外面的情况，他依稀记得，这个城市里有一个体型十分巨大的尸兄，大概就是现在他们看到的这一只。
<br/>　　只是不知道实力如何，他想找个机会去试探一下。
<br/>　　“在这里吗？”白小飞沉思了一下，最终点下了头：“这样也好，毕竟周围的尸兄不知道什么原因都被清空了，这里相对来说，要安全许多。”
<br/>　　“并且，也可以趁着这段时间，在周围搜索一下小鹿母亲的踪迹。”
<br/>　　颔了颔首，刘哔也没有多说什么，之所以想留下，是因为他想起来，附近还有一个挺有意思的图书馆。
<br/>　　他要去确认一些东西。
<br/>　　既然白小飞和刘哔两人都同意，那众人自然也不会去反对，重新拉下了卷帘门，白小飞将众人都聚集在了一起。
<br/>　　“周围的尸兄已经都不见了，我觉得，应该是被那爆炸声吸引过去了，我觉得这是个好消息。”
<br/>　　停顿了一下，白小飞拿出了地图展开：“趁着这段时间，我们可以在附近搜索一下，说不定就会有什么收获。”
<br/>　　点了点头，刘哔没有反对，这正合他的意思。
<br/>　　即使白小飞不提出，他也要想办法推动白小飞外出的。
<br/>　　“那么这样，小惠，你没有什么能力，就留在便利店，照顾好小鹿，晓佳，你刚刚觉醒，实力还比较弱，就跟在刘大哥身边，这样比较安全，最后小胖就和我一起行动。”
<br/>　　看着地图，白小飞提出了一个相对合理的分配，继而说道：“从这边往西，我和小胖去搜索，往东的话，就交给刘大哥了。”
<br/>　　仔细看了一眼地图，刘哔点下了头，他对这边并不熟悉，不过地图的东半部分，有一个十分明显的图书馆标志。
<br/>　　就是不知道是不是那个图书馆，但他总要过去看看。
<br/>　　“那，我们就开始行动吧。”
<br/>　　安排完毕，白小飞看了一下时间，趁着现在天色还早，早些行动也会多出一些时间。
<br/>　　不仅仅是为了寻找小鹿的母亲，而更重要的一个原因，也是为了熟悉周围的情况，毕竟不出意外的话，他们要在这里生存一段时间。
<br/>　　如果周围的情况不搞清楚，他们也没办法安心的呆下去。
<br/>　　……
<br/>　　便利店东边的一个街道上，一个体型异常庞大的婴儿尸兄，正把守在一栋楼前。
<br/>　　“刘……刘哥，我们要跑这么远吗，再往前，会不会被那个大家伙给发现啊？”
<br/>　　晓佳紧紧拉着刘哔的衣角，面对前方的巨婴，她感受到了一股来自灵魂的战栗。
<br/>　　那种东西，怎么能让人提起对抗的心思，如果被发现，恐怕她们谁都活不下去吧。
<br/>　　“嘘……有人来了。”
<br/>　　似乎是察觉到了什么，刘哔俯在晓佳耳边，低声说了一句，随后拉着她躲了起来。
<br/>　　“队长，那个大家伙会再生，不好对付啊。”
<br/>　　不远处，两个鬼鬼祟祟的身影摸索了过来，其中一个赤裸着上身，烫着九枚戒疤的光头，脖子上还戴着一串巨大的佛珠。
<br/>　　那个佛珠，怎么看都有些熟悉，似乎和之前保安使用的那串，有一些相同之处。
<br/>　　只是两个佛珠之间的质量，根本不能相比罢了，最起码刘哔现在没感觉到有什么危险。
<br/>　　而另外一个，一身黑衣，模样很年轻，头发很长，相对于一个男人来说。
<br/>　　这两人似乎很早就来到了这里，只不过都没有察觉到刘哔和晓佳的存在。
<br/>　　“鬼棍，稍安勿躁，我们的目的是为了救出楼里的民众。”
<br/>　　和尚的姿态十分小心，看了一眼大楼的方向，打了一个手势。
<br/>　　随后，两人同时行动，动作一致的潜行着向大楼内部冲去。
<br/>　　这两个人……
<br/>　　刘哔皱着眉头，这明显是来自某个组织，而且那个和尚脖子上的佛珠，引起了他的兴趣。
<br/>　　“你躲起来，在这等我，我自己去看看。”
<br/>　　没有犹豫太久，刘哔转头对晓佳嘱咐了一句，随后不管她紧张的眼神，径直跟随到了鬼棍两人后面。
<br/>　　这两个人的实力都不算强，他有心隐藏的情况下，两人根本察觉不到什么。
<br/>　　路上很顺利，没有惊动任何尸兄，远处守门的那个巨婴也没有发现，三人很顺利的进入了大楼。
<br/>　　这栋大楼，就是刘哔之前在地图上看见的那个图书馆。
<br/>　　进入了楼里，鬼棍和和尚两人明显小心了许多，甚至，他们之前所有的交流，都是用手势来表达了。
<br/>　　没受过相应的训练，刘哔也不知道他们传达的这些信息是什么意思。
<br/>　　但估计也能猜到。
<br/>　　因为两人很快就分头行动了。
<br/>　　反倒是刘哔，此时不怎么好选择，犹豫了一下，最终他还是向和尚搜查的方向赶去。
<br/>　　那串佛珠，对他的吸引力实在是太大了。
<br/>　　“咦，竟然摸进来一只老鼠，在大乖乖的看守下你也能进来，不简单啊。”
<br/>　　还没太过于深入，一道阴冷的声音响起，随后和尚身后，突兀出现了一个鬼魅般的身影。
