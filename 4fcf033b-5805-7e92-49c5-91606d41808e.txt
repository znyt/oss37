<br/>　　“虽然不知道这战门是什么，但是既然是潜在威胁那理应要慎重对待，你在这里坐会，我去打几个电话。”

<br/>　　朱民松了口气，“有队长帮忙应该没多大问题了。”
<br/>　　“那可不见得。”林峰表情凝重起来。
<br/>　　“什么意思？”

<br/>　　他就恍然大悟。
<br/>　　因为出来了五个军人持着手枪瞄准了他们。

<br/>　　队长从房间内走了出来冷笑的看着朱民和林峰。

<br/>　　林峰冷笑道。

<br/>　　朱民满脸不可置信，万万没有想到结局会是这样。
<br/>　　刷刷刷。
<br/>　　然而下一刻。
<br/>　　啊啊啊。
<br/>　　一片惨叫。
<br/>　　却是这几个警卫员直接被飞镖爆头，这群人不过三阶罢了，根本躲不过林峰飞镖，而且也压根没想到林峰有飞镖。

<br/>　　队长面色大变立即就是要去捡枪。

<br/>　　队长冷笑道。

<br/>　　林峰眯着眼，这个家伙的职位可不算低。不可能不知道。

<br/>　　这队长忽然是瞪着眼然后脖子一软直接死了。
