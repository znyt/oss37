　　萧家府邸内的战斗，以慕容家家主带领的长老忽然袭击萧家议事厅而拉开帷幕，慕容家武者则从府邸大门强攻。
<br/>　　这便衍生了两个战场。
<br/>　　一个在府邸里边，议事厅附近，先天武者们的战斗。
<br/>　　一个在萧家比武场和大门附近，为执事们与两方族人之战。
<br/>　　萧逸一路往府邸里面跑去，还未靠近，远远已经感受到澎湃的真气在四周肆虐。
<br/>　　先天武者的力量比后天武者强得太多了，挥手间便是开山裂石，一招一式都甚为恐怖。
<br/>　　当萧逸来到议事厅附近时，才发现这里的战斗激烈到何等程度。
<br/>　　原本的议事厅，此刻已是一片断壁残垣。
<br/>　　周围的府邸，几乎没有一座是完好的，大多数都被战斗的余波毁成碎石烂瓦。
<br/>　　嗖，忽然，一道凌厉的真气打来。
<br/>　　萧逸脚步一动，堪堪躲开，真气打到了他身旁的地面上。地面立刻被轰得粉碎，扬起漫天尘土。
<br/>　　这还仅仅是余波，先天武者的战斗，可想而知有多激烈。
<br/>　　萧逸灵动的目光看向前方，果然，各个长老都在对战。
<br/>　　三、七八九长老，对上了慕容家的四位长老。
<br/>　　二长老则对上了慕容家家主，慕容山；两人皆是先天九重武者，谁也奈何不了谁。
<br/>　　至于大长老，竟是以一敌二，却丝毫不落下风，甚至将那二人压着打。
<br/>　　“嗯？”萧逸皱了眉头。
<br/>　　其余长老的对手，都是慕容家的人。可唯独大长老对手，是两个生面孔的中年男子。
<br/>　　萧逸虽然无法确定这两人的修为，但从他们的气息强弱上来看，竟与二长老和慕容山相差无几，想必也是先天九重武者。
<br/>　　“怎么回事，慕容家从哪找来的两个先天九重？”萧逸敢肯定，紫云城中绝对没有这两号人物。
<br/>　　难怪之前萧如意说大长老被牵制住了，无法救援。
<br/>　　萧逸转头看了眼三长老萧重，发现他和敌人打得不相上下，没有受伤，顿时松了口气。
<br/>　　至于七、八、九长老这边，这三人是萧家出了名的战力最弱的长老，修为只有先天三、四重左右。
<br/>　　而他们的战斗，竟也是处于胶着。证明慕容家的三个长老也没有强到哪里去，同样只有先天三、四重左右的战力。
<br/>　　“哼，就拿你们来下手了。”萧逸冷笑一声，眼中丝毫杀意涌现。
<br/>　　柿子，专挑软的捏这个道理，萧逸还是懂的。
<br/>　　九长老正与一个慕容家长老打得激烈，忽然，一道火红色人影快速朝他而来。
<br/>　　萧家众长老，一眼就看出这道人影是谁，顿时大急。
<br/>　　“逸儿，这个层次的战斗不是你能参加的，快离开。”
<br/>　　“萧逸，快退出去。”
<br/>　　三长老和大长老着急地喝了一声。
<br/>　　九长老更是猛地加大了攻势，试图压制敌人，嘴中吼道，“萧逸，你来干什么，快走！”
<br/>　　那慕容家长老也认出了萧逸，暗道，“哼，萧家的第一天才吗，若是杀了你，萧家众长老必定方寸大乱。”
<br/>　　“萧家九长老，你加大了攻势牵制我又如何。区区一个后天武者，我单靠气势就能将他震死。”
<br/>　　慕容家长老冷笑一声，一身气势覆盖全身。
<br/>　　然而下一秒，他的笑容却忽然戛然而止。因为，火红色人影的气势竟瞬间暴涨，那火剑上的威势更是让他闻到了危险的意味。
<br/>　　他现在才反应过来，已经迟了。
<br/>　　萧逸一记‘崩山斩’，破了他身上的气势，火剑直接劈在他身上。
<br/>　　“噗。”慕容家长老猛地吐出一口腥血，两眼一黑，直接倒下。
<br/>　　萧逸趁势再补了一剑，结束了他的性命。
<br/>　　单靠冰火气丹，萧逸的攻击就不亚于后天九重；加上升龙的增幅，他早已是先天一重、二重左右的战力。
<br/>　　配合上玄阶中级武技崩山斩，这慕容家长老不过是先天三重，而且又不用真气抵挡，不死才怪。
<br/>　　单靠肉身和气势，慕容槐尚且重伤，更何况这个长老呢。
<br/>　　“卧槽，好猛。”九长老吓了一跳。
<br/>　　“别愣着，去帮七八长老他们。”萧逸说了一声，再次提剑而上，攻向了八长老的敌人。
<br/>　　另一边，慕容山见自家长老被杀，顿时大怒，“萧逸，我要你的命。”
<br/>　　二长老眼疾手快，立刻将他拦下，冷笑道，“慕容山，你的对手是我。哼，处心积虑想对付我萧家，看来，这次败的会是你们。”
<br/>　　另一边，萧逸和八长老联手，对付一个慕容家的长老。
<br/>　　“八长老，找机会牵制住他，我找机会一剑劈了他。”萧逸冷笑道。
<br/>　　那慕容家长老顿时大惊，瞥了眼已经死去的自家长老，不禁面露惧色。
<br/>　　在惊惧之下，这长老渐渐开始心慌。
<br/>　　加上萧逸与八长老联手，他慢慢手忙脚乱起来。
<br/>　　数十个回合后，他甚至出现了破绽。
<br/>　　两人自然不会放过这个破绽。
<br/>　　“噬火百刃。”八长老大喝一声，数十道火刃劈出。
<br/>　　“碧波叠浪掌。”慕容家长老释放出一股巨大的波涛。
<br/>　　不得不说，两方的长老释放玄阶武技，其威势比之执事们强了太多了。
<br/>　　火刃与波涛碰撞，双双消散。
<br/>　　“崩山斩。”萧逸趁机而上。
<br/>　　此时，那慕容家长老刚刚施展过武技，来不及调动体内大量真气再施展一次。只能靠双手，覆盖上少量真气来抵挡。
<br/>　　轰的一声，崩山斩之威，再次显现。慕容家长老的双手直接被劈得血肉模糊，嘴中口吐鲜血，显然受了不轻的伤势。
<br/>　　萧逸与八长老联手攻击，趁他病要他命，数分钟后，这位慕容家长老在萧逸剑下饮恨，成为了一具尸体。
<br/>　　“该死，萧逸不过是个16岁的少年，怎么会这么强。”那与七长老对战的慕容家长老愤愤地骂了一声。
<br/>　　如今，两个慕容家长老已经死去，他除了愤怒外，也渐渐出现了担忧。
<br/>　　另一边，慕容山也是愤怒无比，咬牙切齿，“阿，今日明明可以拿下萧家的，竟因为一个萧逸坏了我们许久的筹划。”
<br/>　　他虽愤怒，但作为家主，他的眼光和考虑显然比其他人更好更周到。
<br/>　　他很清楚，再这样下去，慕容家长老只会一个接一个地死去，最后他们大败而逃。
<br/>　　仅仅是数秒的考虑后，他愤愤地一掌震退二长老，而后不甘地怒吼一声，“慕容家子弟，全部撤退。”
<br/>　　声音响彻整个萧家。
<br/>　　慕容山脸上那不甘的神色，仿佛要滴出血来，但他别无他法。
<br/>　　那与七长老对战的慕容家长老，闻言也一掌震开七长老，快速撤退，他已经看到萧逸和八长老准备围攻他了。
<br/>　　同时，那与大长老鏖战许久的两个中年陌生男子也立刻脱离战斗，快速撤退。
<br/>　　嗖、嗖、嗖、、、五道身影瞬间脱离了战斗。
<br/>　　临走前，慕容山怒声道，“萧逸，你坏我好事，他日必取你性命。”
<br/>　　“追。”三长老怒喝一声，“慕容家的杂碎，一个都别想跑掉。”
<br/>　　萧逸和七长老等人刚想去追。
<br/>　　大长老却阻止道，“别追了，噗...”
<br/>　　大长老一口腥血吐出，脸色煞白，脚步也摇晃不稳。
<br/>　　幸好二长老眼疾手快，一把扶住了他。
<br/>　　“大长老，你受了重伤。”二长老惊道。
<br/>　　“无妨。”大长老摆摆手，道，“去看看族人们的伤亡如何，处理好后，来议事厅...”
<br/>　　大长老看了眼已经成为断壁残垣的议事厅，摇了摇头，道，“处理好后，直接去家族大堂议事。”
<br/>　　“是。”众人应答一声。
<br/>　　萧逸回到府邸大门那边的战场，战斗已经停止了，慕容家的武者早已离去。但在离去前，在萧家执事的追击下，死伤惨重，留下一地尸体。
<br/>　　萧家这边也不乐观，但总体伤亡比慕容家少了大半以上。
<br/>　　今日一战，只能用惨烈去形容。
