　　昆仑山，玉虚宫中，元始还在借助道祖鸿钧的斩三尸之法，推演属于自己神通。
<br/>　　两千年时间过去了，元始还是没有从中领悟出属于自己的神通，虽然他已经有了一些雏形了，但还是没有真正的成形。
<br/>　　他的修为，也没有突破，这让他许成了许多洪荒大神的笑柄。三清之首，道祖首徒，别人都已经突破了，你竟然还没有突破，这算什么事？
<br/>　　当然，也有人觉得他早已突破，只是不知何原因，让人不知道罢了。
<br/>　　对此，元始并没有多想，他刚从领悟之中醒来，还不知道这些事，但就算他知道了，也不会在意。
<br/>　　我修我道，何须在意他人之想法。
<br/>　　“哎！罢了，罢了！没有修炼过，只凭自己推演，终究是不行的，还是先自己修炼一番，再来推演自己的神通。或许那样会容易许多。”
<br/>　　想到此处，元始将注意力集中在斩三尸之上，却是打算亲自修炼一番，感受一下其中的效果。
<br/>　　“斩善、恶、执三念，明悟自我？世间本无善恶，只因沾染了红尘之气，也就有了善恶。无所谓善，无所谓恶，善恶本相对，无绝对的善者，亦无绝对的恶者，正所谓魔亦有善念，佛亦有魔性，佛魔本同源，一念成佛，一念成魔。善恶二尸对于本座来说，却是一点难度也没有。至于执尸——”
<br/>　　这一刻，元始回想起了自己的前世今生。
<br/>　　那一年，他还是一个小道士，过着普普通通的日子。
<br/>　　但，因为一个意外，他捡到了一个圆印，改变了他的一生。
<br/>　　之后，他从圆印之中得到他第一个神通——他心通！
<br/>　　后来，他借助他心通，接触到了修真者，就这样，他一步步走上了修道的道路。
<br/>　　可是，当时天地处于末法，是修士的黄昏！
<br/>　　但，他亦没有放弃，靠着自己去拼、去抢，领悟魔功修炼。
<br/>　　就算被同道中人当成魔头，追杀于他，他亦没有放弃。
<br/>　　一次次的生死逃亡……
<br/>　　一次次的绝杀危机……
<br/>　　一次次的突破……
<br/>　　终于，他到了渡劫成仙的一步。
<br/>　　那一刻，他与雷霆拼杀，与天搏斗，就算到死的那一刻，也没有放弃。
<br/>　　之后，他成了混沌中的因果魔神。
<br/>　　那一刻，他想到了开天之劫，想到盘古，想到了其他魔神，他的心中充满危机感。
<br/>　　他拼命夺舍因果魔神……
<br/>　　意志不灭，灵魂之火！
<br/>　　终于，他成功了。
<br/>　　之后，游混沌，战苍茫，斩魔神，战盘古。
<br/>　　一步步地谋划，一步步地算计，一步步地拼命，一步步地生死。
<br/>　　他，终于成了洪荒的元始。
<br/>　　之后，废因果之道，重修元始大道。
<br/>　　更加努力地修炼，用手中的道，手中的法，拼出如今的元始。
<br/>　　为的是，得证大道不朽，超脱诸般大道，主宰自己命运！
<br/>　　他，目标一直没变，决心一直不改，道心一如当初！
<br/>　　此刻，之前的一幕幕，一点点，全在他的心中流过，他仿佛在此回到了当初，再次经历了一遍人生。某一刻，他醒了过来，他仿佛变了，但又好像没变，他身上仿佛多了一种说不清道不明的气质。
<br/>　　这一刻，他开口说道：
<br/>　　“执念，吾已明了。本座的执念，一直就是大道巅峰，虽然如今还在求道路上，但它一直没变。”
<br/>　　“所以，如今，三尸对于本座，已经不成问题了。那么——”
<br/>　　元始盘腿而坐，体内发出闷哼巨响，仿佛雷霆，又好似天怒一般。他的头顶，是丈许大小的太上无极元始庆云，看似很小，实则非常大，大到好似包括了整片天地。
<br/>　　此刻，他的庆云之上，被分成三部分，一部分漆黑如墨，好似滔天黑水。这是他的恶念，恶念如海，汹涌彭拜，透过庆云，贯穿苍穹，化而为海，此海磅礴，无边无际，狂卷而来。
<br/>　　另一部分洁白如光，好似一团光亮无比的光源，照耀着整片天地，光芒四射，光辉璀璨。又好似一朵洁白纯净的白莲，能够净化人心中的杂念。这是他的善念。
<br/>　　最后一部分，为红色，好似一朵妖艳鲜红的花朵，绽放在庆云之上，红光耀眼，璀璨无比，就好似人生一般烂漫璀璨。这是他的执念，坚信人生当如此。
<br/>　　“吱吟！”
<br/>　　头顶是庆云在翻滚，庆云之上，善、恶、执三念化为黑、白、红三海，海之滨，天地力量凝聚时，猛然出现一把刀，此刀磅礴，此刀无形亦无影，此刀乃大道之刀，乃天地之刀。
<br/>　　力之极致之时，蓦然向着元始身上的庆云，狠狠地斩了下来。这一刀，斩在了虚无，落在元始的道心之上，霎时间，前世今生，经历过的一切，浮现在心头。
<br/>　　人心悱恻，修道之路，注定尸山血海，一路走来，有可为，有可不为，坚持自己的原则，在岁月之中，执自己之道念，早就坚定了一颗道心。
<br/>　　魔心、善心、道心，三颗心代表着他的善、恶、执三念。
<br/>　　“吾元始此生，定当披荆斩棘，开元为道，阻我道者，管他是仙、是佛、是魔、是神、是鬼、是妖、是人、是圣，皆可杀之！”元始神色坚定，心中蓦然大吼。
<br/>　　这一吼，似生命里，所有意志，绽放而出，成就了璀璨！嗡鸣声中，头顶庆云光芒大盛，玄之又玄的气息，扩散开来。
<br/>　　在元始意志达到极致之时，横亘苍穹的斩尸之刀，气势更加磅礴，更加恢宏。
<br/>　　“斩！——”
<br/>　　“斩！”
<br/>　　“斩！”
<br/>　　三声大喝，三刀连斩。只听轰的一声，元始体内发生了巨大的变化，无尽恶念汇聚成一个身穿黑袍，头顶十二品灭世黑莲的道人。
<br/>　　无尽善念汇聚成一个身穿白袍，头顶十二品净世白莲的道人。无尽执念汇聚成一个身穿红袍大衣，头顶混沌钟的道人。
<br/>　　《元始无名天经》、《玉清元始经》、《神元九转》、《玄黄不灭》在这一刻，全部疯狂运转起来。无尽的天地之力，汇聚而来，灌体而入。
<br/>　　霎时间，无穷的力量，扩散到四肢百骸，弥漫五脏六腑，肉体道行在飙升。
<br/>　　无尽的明悟，道心圆满，直冲丹田、三百六十五大神窍洞天、元神空间，元始终于厚积薄发，连斩三尸，修为狂飙。
<br/>　　这一刻，他的道行，到了准圣圆满。
